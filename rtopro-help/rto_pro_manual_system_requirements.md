Minimum System Requirements:

•Any computer that will run Windows 10 or newer, Windows Server 2016 or newer (32bit and 64bit versions of Windows) (the faster the better but there is no speed requirement)

•16GB of RAM recommended. 

•1TB Free Hard Disk Space. (during installation only about 100MB are used but you will need room for all the data that you will be storing, 1TB should hold about 100 years worth of data.)

•A Backup device such as an external USB Hard Drive or USB Thumb Drive or something similar is recommended.

•A UPS or Uninterrupted Power Supply is recommended.

•At least a 17" monitor is recommended due to the 1024x768 resolution required.

Printers:

•A Receipt Printer such as an Star TSP100 or Star TSP700 is recommended for receipts. 

•A Laser or Deskjet type printer is recommended for reports, invoices and contracts
