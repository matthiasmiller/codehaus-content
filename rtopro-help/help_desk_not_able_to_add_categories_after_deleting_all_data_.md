# Not able to add Categories after deleting all data.

|  [](help_desk_problem_home_office_record_counts_not_matching_with_store_.md "Previous Topic") [](help_desk_pricingsetup.md "Parent Chapter") [](error_invalid_property_value_w.md "Next Topic")

After deleting data you may not be able to enter categories. The New category screen will not come up.

Delete the setting "Last Date" in the Init Edit function then restart RTO Pro, this will correct the problem.
