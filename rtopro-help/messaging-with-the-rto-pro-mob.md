The RTO Pro Mobile App is an app for Android and IOS that is for your employees. It can be used for collections, driver routing, and messaging.

[Click here for info about the mobile app for collections](http://www.rtopro.com/mobileosam.aspx)

[Click here for more info about the mobile app for driver routing](http://www.rtopro.com/mobileschedule.aspx)

The RTO Pro Mobile App can also be used for messaging employees.

Requirements:

1\. [RTO Pro SMS Messaging Service](http://www.rtopro.com/smstexting.aspx)

2\. Check the box in Store Setup to use [RTO Pro App messaging](http://www.rtopro.com/accountmessenger.aspx) (customers will get messages by SMS if they don't have the app installed)

3\. Employees install [RTO Pro Mobile](http://www.rtopro.com/mobileosam.aspx) app on their mobile devices.

4\. Make sure any employee you want to message has their email address they used to register the mobile app saved in their employee records in employee setup (Setup menu > Security > add or edit employees)

5\. To send an employee message from the main menu of RTO Pro press the F7 key, this will bring up the customer messaging screen, from there press the F6 key.
