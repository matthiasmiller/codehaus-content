Main Menu Option "5" Setup Option "4"

Use this function to enter a message that you want to be printed on receipts.

There are 2 lines of information that can be entered to be printed on the top of the receipt. The message will be printed exactly as you enter it.

For the bottom of receipts you can have as long of a message as you like. You can have different message for retail receipts and all other receipts. The box on the left is for retail receipts and the box on the right is for all other receipts.
