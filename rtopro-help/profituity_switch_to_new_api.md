# Profituity switch to new API

|  [](welcome_welcome_1.md "Parent Chapter")  
---|---  
  
To switch to the new API for Profituity the store would have to contact Profituity and let them know they want to switch. They will provide them with the new user/pass for the new API.

After switching they will have to uncheck the "Use New API" box and check results for the transactions in the old API daily until all the old trans are cleared. Procedure steps are below.

To switch check the box in #36 below and enter the user name / password / merchant ID they get from Profituity.

Then daily do the following until all trans from the old API are cleared.

1\. Uncheck #36 below

2\. Check for ACH Returns

3\. Check #36 back
