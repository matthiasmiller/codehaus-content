The Inventory Custom Report Viewer allows you to sort and group fields the way you like, you can also include only the fields you would like to have in the report. Below is a screen shot of the report viewer, explanations of the buttons are below the screen shot.

![image9](image9.jpg)

![image10](image10.jpg) These are the "Open", "Save" and "Save As" buttons. The Open button allows you to open a saved report layout. The Save button will save the current layout. The Save As button will allow you to save the current layout and rename it if it was previously saved.

![image1](image1.gif) Clicking on this button will print the report.

![image2](image2.gif) Clicking this button will display a print preview of the report.

![image3](image3.gif) Clicking this button will toggle the "Fit columns to page when printing" option. When it is on, the columns will be shrunk so that all columns fit the width of the page when printed.

![image4](image4.gif) Clicking this button will export the report in Excel Spreadsheet format. The file will be saved to your desktop and displayed in Excel (if you have it installed) after it is exported.

![image5](image5.gif) Clicking this button will export the report in HTML format. The file will be saved to your desktop and displayed in your web browser after it is exported.

![image6](image6.gif) Clicking this button will export the report in Text format. The file will be saved to your desktop and displayed in your default text editor after it is exported.

![image7](image7.gif) Clicking this button will Export the report and attach it to an E-mail using the RTO Pro Sendmail interface. You will be prompted to select a format to export in after clicking the button.

![image9](image9.gif) Clicking this button will bring up the report summary information (Shown below), where you can make changes to the report layout.

![image8](image8.jpg)

![image10](image10.gif) Clicking this button will bring up the form where you can edit the fields that are included in the report, you can also change the order of the fields on the report. Note: You can also change the order of the fields by clicking on the header columns and dragging. You can remove columns from the report by right clicking on the header columns and clicking on "Remove Column".

![image11](image11.gif) Clicking on this button will bring up the Group form where you can change how the report is grouped. Note: You can also change the grouping from the main report screen by clicking on the header columns and dragging them to the grouping box above the columns.

![image12](image12.gif) Clicking this button will bring up the sort form where you can change the way the report is sorted. Note: You can also change sort order by clicking on the column headers.

![image13](image13.gif) Clicking this button will toggle on/off the Group box at the top of the report.

![image14](image14.gif) Clicking this button will toggle on/off the Card View layout of the report.

![image15](image15.gif) You can use these buttons to navigate through the report records.
