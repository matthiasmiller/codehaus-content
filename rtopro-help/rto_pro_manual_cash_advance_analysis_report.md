Revenue Reports Option "9"

This report will break down cash advance contracts by amount of loan. This is used to fill out reports which are required in some states.

To run this report enter the date range you wish to include, select print or display and push F5.

This report also has a Cash Advance Managers report section and that Deferred Presentment Providers Report section. These reports displays totals of loan amounts, total collected and % collected and other information for Cash Advance loans.
