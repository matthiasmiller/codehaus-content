To download updates for RTO Pro, go into RTO Pro and click on Help > Download RTO Pro Update from the menu bar on the Main Menu. This will start a function that will enable you to download and install an update easily. You must have internet access and you must be on-line for this function to work. You also must have an account ID for your store to download an update. Call 352-383-9375 to get your account ID.
