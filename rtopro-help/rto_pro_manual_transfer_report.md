Inventory Menu option "D"

This report shows inventory transferred in and out over a given period of time. The field descriptions for this report are below.

Transfer Dates: Enter the date range for the report.

Print or Display: Choosing display will cause the report to be displayed before printing, you can click the "Print" button from the report viewer if you want to print it after reviewing it.

Push F5 to begin the report.

Note: For Non Serialized inventory, the transfers in section of this report does not show the quantity transferred in. Also it only counts as 1 item in the count, cost and balance totals. There is no way to determine the quantity of non serialized items transferred in from this report.
