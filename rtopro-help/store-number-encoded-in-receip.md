# Store Number Encoded in Receipt number

|  [](welcome_welcome_1.md "Parent Chapter")  
---|---  
  
All receipts have the store number in them, for instance "P010000008" is for store #1. The underlined characters is where the store number is embedded. Since the receipt started with only 2 characters for the store number we created an alphanumeric coding system to identify stores over 99. Below are the codes for each valid store number in RTO Pro (Central Server is the only edition that can go over 99 as a store number). Valid store numbers are 1 to 974. For example the same receipt above for store #974 would be "PZZ0000008"

Note the letter O is not used to prevent it being mistaken for a 0(zero).
