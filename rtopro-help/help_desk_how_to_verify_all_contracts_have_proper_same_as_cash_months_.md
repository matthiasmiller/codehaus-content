To change a single contracts same as cash option use [Contract Maintenance](rto_pro_manual_contract_maintenance_and_inventory_switchout.md) (Point of Sale menu option 3).

To verify that all open contracts have been setup with the correct number of months for same as cash use the function [Check Same as Cash Months on Contracts](rto_pro_manual_store_setup_information.md#check_same_as_cash_months_on_contracts) This function can be found under the Tools Menu bar at the top of [Store Setup Information](rto_pro_manual_store_setup_information.md) (Setup menu option 1).
