
# Matthias Miller - CodeHaus OBD

## Records

### OBD Account Record

Sections and Fields:

* OBD Account section:
  * Name (string; required)
  * Is Active (checkbox)
  * FB Account (drop list of FB Accounts)
  * GA Account (drop list of GA Accounts)
  * HS Hub (drop list of HS Hubs)
  * GHL Location (Silverloom ID) (drop list of GHL Locations)
  * TSA Domain (drop list of TSA Domains)
* Dashboard Users section:
  * Dashboard Users; embedded spreadsheet of:
    * User (drop list of User Profiles; required)
    * Email (email; auto-calculated; read-only)
