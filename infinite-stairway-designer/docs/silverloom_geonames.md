
# Silverloom GeoNames

## Records

### ZipCodeMap Record

Sections and Fields:

* ZipCodeMap section:
  * ID (autonumber; hidden & read-only)
  * Zip Code (text; required)
  * City (text; required)
  * State (drop list of States; required)
  * GeoLat (number; 9 decimals)
  * GeoLng (number; 9 decimals)
