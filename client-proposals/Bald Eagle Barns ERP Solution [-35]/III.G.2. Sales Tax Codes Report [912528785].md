3.7.2. Sales Tax Codes Report

  


Requirements

This is a report of the Sales Tax Codes that are synced from QB and stored in the database.

  


This is accessed from a "Sales Tax Codes" menu option on the the Admin Menu. 

  


Title: Sales Tax Codes

  


Columns: 

  * Sales Tax Code (link to record)
  * Item ??  (link to record)??
  * Description
  * Type 
  * Rate % 
  * Included Rates (multi-row display of Code and Rate % of the included rates if Group) 



  


Grouped by: Active/Inactive status (Active at the top), then by Type (Sales Tax Group then Sales Tax Item)

  


Sorted by: Sales Tax Code

  


Filters: N/A

  


Buttons: N/A

  


Data Access: Admin Only

  


*Done.
