3.6.4. Manage SKUs Report

  


Requirements

This is a report of all SKUs in the system.

  


This would be accessed from a "Manage SKUs" menu option on the Admin Menu. 

  


Title: SKUs

  


Columns: 

  * SKU (code; link to the detail screen)
  * Description
  * Price
  * Category (hidden if grouped by Category)
  * UOM



  


Grouped by: (depending on the filter selection)

  * Category, then Active/Inactive
  * Active/Inactive status (Active at the top; Inactive at the bottom in gray).



  


Sorted by: Alphabetically by SKU code

  


Filters:

  * SKU Category (multi-select list of Categories; default to include all)
  * Group By (required; drop list of Category, Active/Inactive; default to Category) 



  


Buttons:

  * New SKU (opens a new blank SKU record)



  


Other Notes: N/A
