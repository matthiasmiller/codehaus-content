6.3. Time Sheets Report

  


Requirements

This is a report of hours/days worked, overtime hours worked, and PTO time used as tracked on the Weekly Time Sheets.

  


This would be accessed from:

  * View My Time Sheets option on the Main Menu
  * View Time Sheets link on Members' contact records
  * Open Employee Time Sheets on the Admin Menu



  


Title: Weekly Time Sheets for <LastName, FirstName> 

  


Columns: 

  * Work Week (mm/dd/yyyy - mm/dd/yyyy; link to Time Sheet record)
  * Worked Hours (shows hours, to 1 decimal place) 
  * Overtime Hours (shows hours, to 1 decimal place) 
  * PTO Hours Used (shows hours, to 1 decimal place) 
  * Total Hours (shows hours, to 1 decimal place) 
  * Worked Days (shows days, to 2 decimal places)
  * PTO Days Used (shows days, to 2 decimal places)
  * Total Days (shows days, to 2 decimal places)
  * Tasks $ (show dollar amount, to 2 decimal places) 



  


Grouped by: Payroll Period

  


Sorted by: Work Week (most recent at top)

  


 Filters: 

  * Member (required; drop list of all Member contacts; auto-filled to the current user for non-admins)
  * Compensation (drop list of Days, Hours, Both; default to Days or Hours based on the selected Member's current Compensation type; if left blank, shows both Days and Hours)



  


Buttons: N/A 

  


Data Access: Same as the Time Sheet records

  


Other Notes:

  * Note that this report can only be run for one Member at a time:
    * If opened from the "View My Time Sheets" option Main Menu, default to the current user/contact
    * If opened from the contact record, default to the corresponding contact
    * If opened from the "Open Employee Time Sheets" option on the Admin Menu, first prompt for Member
  * The report would show columns for days, hours, or both, depending on how the Compensation filter is set.



  


*Done.
