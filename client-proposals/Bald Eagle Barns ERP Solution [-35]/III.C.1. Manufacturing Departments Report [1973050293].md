3.3.1. Manufacturing Departments Report

  


Requirements

This is a report of the Manufacturing Departments. It can be used to view, edit, and create departments.

  


This would be accessed from the "Configure Manufacturing Departments" option on the the Admin Menu. 

  


Title: Manufacturing Departments

  


Columns:

  * Department Name (link to record) 
  * Sequence #



  


Grouped by: Active/Inactive status (Active at the top; Inactive at the bottom in gray).

  


Sorted by: Sequence #

  


Filters: N/A

  


Buttons: New Department (opens a blank record in a new tab)

  


Data Access: Admin Only

  


Other Notes: N/A 

  


*Done.
