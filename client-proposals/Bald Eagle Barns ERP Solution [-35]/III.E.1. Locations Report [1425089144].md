3.5.1. Locations Report

  


Requirements

This is a report of all Locations.

  


This would be accessed from a "Configure Locations" menu option on the the Admin Menu. 

  


Title: Locations

  


Columns: 

  * Location Name (link to record)
  * Dealer (name)
  * Address
  * City, ST Zip
  * Coordinates



  


Grouped by: Active/Inactive status (Active at the top; Inactive at the bottom in gray).

  


Sorted by: Location Name

  


Filters: N/A

  


Buttons: 

  * New Location (opens a blank record in a new tab)



  


Data Access: Admin Only

  


Other Notes: N/A 

  


*Done.
