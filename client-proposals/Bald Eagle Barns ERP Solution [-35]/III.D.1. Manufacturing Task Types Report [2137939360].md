3.4.1. Manufacturing Task Types Report

  


Requirements

This is a report of Manufacturing Tasks. It can be used to view, edit, and create task categories.

  


This would be accessed from a "Configure Manufacturing Departments" option on the the Admin Menu. 

  


Title: Manufacturing Task Types

  


Columns:

  * Task Name (link to the record)
  * Manufacturing Department
  * Payment Type
  * Payment %
  * Payment $ 
  * Sequence #



  


Grouped by: Active/Inactive status (Active at the top; Inactive at the bottom in gray), then by Manufacturing Department.

  


Sorted by: Sequence

  


Filters: N/A

  


Buttons: New Task Type (opens a blank record in a new tab)

  


Data Access: Admin Only

  


*Done.
