3.6.2.1. Diagram Shapes for Options

  


Requirements

Overview: These shapes would be saved on a separate record type (Diagram Shape) and linked to the option SKU. 

  


Accessed via: 

  * A configuration menu option on the Admin menu: "Configure Diagram Shapes"



  


Sections and Fields: 

  * Name (required; text field)



  


Data Access: 

  * Menu option (and therefore the report) would be hidden for non-Admin users
  * Only Admins could edit the records (via the "Edit Diagram Shapes" permission)



  


Other Notes: Each of these will allow attaching a .png through the Document Archive feature.
