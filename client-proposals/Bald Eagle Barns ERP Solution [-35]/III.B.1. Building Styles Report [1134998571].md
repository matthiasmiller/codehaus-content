3.2.1. Building Styles Report

  


Requirements

This is a report of all Building Styles.

  


This would be accessed from a "Configure Building Styles" menu option on the Admin Menu. 

  


Title: Building Styles

  


Columns: 

  * Style Name (link to record)
  * Style Code



  


Grouped by: Active/Inactive status (Active at the top; Inactive at the bottom in gray).

  


Sorted by: Style Name

  


Filters: N/A

  


Buttons: 

  * New Building Style (opens a blank record in a new tab)



  


Data Access: 

  * View: 
  * Edit: Full Admin Only



  


Other Notes: N/A

  


*Done.
