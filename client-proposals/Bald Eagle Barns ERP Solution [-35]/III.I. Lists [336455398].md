3.9. Lists

  


Requirements

The database will store various BEB-specific simple lists to be referenced by various records and reports in the system. These will be user-managed lists. 

  


These would be pulled from the old system and imported as part of the data migration process. 

  


The current lists are as follows: 

  * Building Type (should be user-configurable)  
    * Metal 
    * Painted 
  * Building Siding Colors 
    * (To be pulled from the existing database) 
  * Building Trim Colors
    * (To be pulled from the existing database) 
  * Building Roof Colors
    * (To be pulled from the existing database)
  * Payment Type 
    * Check
    * Cash
    * Purchase Order
    * Credit Card
  * SKU Categories
    * Building Styles 
    * Options 
    * Materials 
  * 


  


  


Data Access: Editable by Full Admin only
