10.0.1. Expenses

Fields:

  * Date (date; required; default to today)
  * Time (time; required; default to now)
  * District (required)
  * Description (text; required)
  * Type (user-configurable list; required; can add to list)
  * Paid (checkbox)
  * Prepaid by (company list)
  * Amount (required)
  * Receipt (memo with attachments)
