2.1. Mobile Form

The hauler and dealer portal will have an option to search for a building via any combination of:

  * STIN
  * Sales Order #
  * Customer Name



  


When searching by Customer Name, show a list of matching customer names, addresses, phone numbers, STIN, and building description.

  


After the building is selected, prompt for:

  * Contact Information
    * Customer Name
    * Customer Address
    * Customer Phone Number
    * (Can be defaulted from latest invoice)
  * Description (multi-line text form; include an note for the user, "Please include any additional information required for the repair, such as directions, gate codes, etc.")
  * Pictures (one or more file attachments, allow via phone camera)
