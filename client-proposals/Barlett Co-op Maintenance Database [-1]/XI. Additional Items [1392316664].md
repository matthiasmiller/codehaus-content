11\. Additional Items

  


Requirements

Tablet Resolution

Reports and record screens will be optimized to fit the iPad tablets currently used by Bartlett Co-op.

  


Users and Permissions

The software supports user-specific permissions. Key users can be specified to be system administrators, allowing them to add/remove users. The following permissions will also be available:

  * Manage equipment lists
  * Edit questionnaire templates



If desired, additional permissions can be added as part of the development phase.

  


A location will be specified for each user.

  


Modification History

All record screens will have links to view record modification history.
