4.4.2. Contacts

  * Title: Import Contacts
  * Run from: Import Contacts
  * Database Level: Contact
  * Type of import: non-RG
  * Included columns, corresponding to the sequence on the import file: 
    * Customer ID
    * First Name
    * Middle Name
    * Last Name
    * Email
    * Phone Numbers (including Fax)
    * Addresses
  * Special instructions: 
  * How to match records: Customer ID
  * Action for unmatched records: ignore
