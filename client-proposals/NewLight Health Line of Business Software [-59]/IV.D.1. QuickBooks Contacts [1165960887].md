4.4.1. QuickBooks Contacts

  * Title: Import QuickBooks Contacts
  * Run from: Import QuickBooks Contacts
  * Database Level: QuickBooks Contact
  * Type of import: non-RG
  * Included columns, corresponding to the sequence on the import file (and corresponding procedure for columns that don't directly match a field): 
    * Customer ID
    * First Name
    * Middle Name
    * Last Name
    * Email
    * Phone Numbers (including Fax)
    * Addresses
  * Special instructions: 
  * How to match records: Customer ID
  * Action for unmatched records: ignore
