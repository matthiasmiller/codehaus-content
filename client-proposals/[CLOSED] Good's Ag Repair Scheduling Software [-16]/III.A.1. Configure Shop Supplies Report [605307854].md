3.1.1. Configure Shop Supplies Report

Configure Shop Supplies Report (new): 

  * Accessed via: A "Configure Shop Supplies" menu option in the Configuration section of the Advanced menu 
  * Column(s): 
    * Description
  * Grouped by: Active (at the top) / Inactive (at the bottom, in gray) 
  * Sorted by: Description
  * Menu Visibility: All users
  * Other Notes: N/A
