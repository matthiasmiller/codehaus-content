2.2. Contact Record

Overview: The Solution will use the standard AppHosting Contact module, with some customizations.

  


Accessed via: 

  * View Contacts report



  


Sections and Fields: 

  * Standard Sections and Fields:
    * Contact Section:
      * Active Contact (checkbox; defaults to filled)
      * First Name (required)
      * Middle Name (optional)
      * Last Name (required)
      * Summary (optional; plain text field)
      * Organization (optional; drop list)
    * Address (optional)
    * Phone (optional)
    * Email (optional)
    * Notes (memo)
    * Incidents
    * AppHosting Settings Section:
      * User Id (drop list of all User IDs)
      * Is Incident Assignee (checkbox; defaults to unchecked)
      * Is Time Card User (checkbox; defaults to unchecked)
  * Executive Notes (custom section; memo)
