1.6.1.1. Configure Service Locations Report

  


Requirements

This is a report of all Service Locations.

  


This is accessed from Advanced | Configuration | Configure Service Locations. 

  


Title: Service Locations

  


Columns: 

  * Location Name (link to record)
  * Location Category
  * Use Quality Control (Yes/No)



  


Grouped by: Active/Inactive, then by Location Category (Shop at the top)

  


Sorted by: Location Name (alphabetical)

  


Filters: 

  * Location Category (drop list of Categories; defaults to blank = all)
  * Show Inactive Service Locations (checkbox; defaults to unchecked)



  


Buttons: 

  * New Service Location (opens blank new Service Location record)



  


Menu Visibility: All Users

  


Other Notes:

  * N/A
