1.4.1. Job Progress

  


Requirements

This is a user-editable simple list used to track Job Progress. 

  


Example list items may include: 

  * Diagnose
  * Customer Confirmation
  * Order Parts
  * Waiting on Parts
  * Final Repair



  


This list is universal for all users, and all users can add to it via the "+" option on the drop list on the Incident record. 

  


The initial items for this list are to be set up by Good's Ag, with assistance from CCI/CH as needed.
