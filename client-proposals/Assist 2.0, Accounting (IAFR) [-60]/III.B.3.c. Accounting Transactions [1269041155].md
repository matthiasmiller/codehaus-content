3.2.3.3. Accounting Transactions

  


Requirements

Each accounting transaction will include the following custom fields:

  * Event Type (list of Accounting Events)
  * Event Record ID (number; editable and required if Event Type is specified)
  * Event RG ID (number; editable if Event Type is specified)
