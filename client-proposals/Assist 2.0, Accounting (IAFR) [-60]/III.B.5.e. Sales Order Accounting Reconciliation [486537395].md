3.2.5.5. Sales Order Accounting Reconciliation

  


Requirements

This will work the same as Building Accounting Reconciliation.

  


This will be run every night before midnight.

  


It should run the Building Accounting Reconciliation for any associated STINs.

  


It should support moving buildings from one district to another, as described in Building Accounting Reconciliation.
