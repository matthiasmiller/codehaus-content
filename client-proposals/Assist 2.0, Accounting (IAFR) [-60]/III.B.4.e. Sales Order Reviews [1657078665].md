3.2.4.5. Sales Order Reviews

This is an editable report of unapproved accessory accounts.

  


This is accessed from Accounting | Sales Orders | Accessory Accounts. 

  


Title: Accessory Accounts

  


Columns: 

  * Sales Order Created Date
  * Sales Order Sale Date
  * Accessory Description
  * Accessory Amount
  * Accessory Notes
  * Default Account
  * Assigned Account (editable)
  * Approved (editable)



  


Grouped by: 

  * Sales Order ID



  


Sorted by: 

  


Filters: 

  * Date Range (default to previous date)



  


Buttons: 

  * NA



  


Menu Visibility: 

  


Save Type (for editable reports): Instant Save

  


Live Updating (for editable reports): no

  


Other Notes:

  * Include all that are not canceled and are not a quote.
  * Default Account should be a macro that checks for the first matching text in the Financial Allocation's accessory RG. These are rules that Daune/builder will write. If no rows are found, it should use the default income account for the building.
