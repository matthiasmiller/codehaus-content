6.1.1. Contact Name Format Overrides

Contact Name Overrides

The following systems override contact name manually (most only do so via the ContactNameFormat field in Silverloom Settings).

  * XFB
    * Organization Name: <Organization Name entry field> '<[optional] Preferred Name>'
    * Individual Name: <Last Name>, <First Name> <Middle Name> '<[optional] Preferred Name>'
    * ZLP
      * Employee type contact:  <Default Display Name from Settings>
      * Other contacts: <Default Display Name from Settings> (<Address Line 1>)
