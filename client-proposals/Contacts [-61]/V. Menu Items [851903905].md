5\. Menu Items

The standard menu groups and items for the module are the following:

  


Contacts

View Contacts

View Organizations

  


Imports and Exports

Export Organizations

Import Organizations

Export Contacts

Import Contacts

  


Configuration

Configure Contact Types

  


Notes:

  * Organization items are only included if the "Use legacy organizations" system switch is set to "Yes".
  * The Contact export and import items are included by default, but can be excluded via a developer-configurable setting.
