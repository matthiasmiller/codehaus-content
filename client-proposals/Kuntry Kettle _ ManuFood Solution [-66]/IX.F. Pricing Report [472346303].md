9.6. Pricing Report

  


Requirements

This report would include the following columns (designed after the existing Pricing Spreadsheet): 

  * SKU
  * Description
  * Smallest UOM + Primary UOM (i.e. "50 lb bag")
  * Pounds / Primary UOM
  * Supplier
  * FOB Cost / Primary UOM
  * Freight Cost / Primary UOM
  * Combined Cost / Primary UOM
  * Cost / Smallest Unit
