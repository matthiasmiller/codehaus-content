9.7. Orders Report

  


Requirements

This report would show: 

  * Current/in-process orders
  * Completed work for date range



  


It would include the following columns: 

  * Order/Recipe Number
  * Customer/Product Description
  * Container
  * Labels
  * Qty Batches
  * Expected Yield
  * Changeover
  * Allergens
  * Est. Points
  * Order Date
  * Ship Date



  


The completed work would show:

  * Production Date
  * Order Number
  * Customer/Product Description
  * Packaging
  * Qty Batches
  * Yield
  * Changeover
  * Allergens
  * Points
  * Turnaround



  


These can likely be included in a shared report with different views.
