4.4. Silverloom Changes

New Permission:

  * Manage Portal Access (disabled by default for all users, including admins)



  


SPE Member Application record:

  * New field: Joinder Agreement Filed (checkbox; visible when the Year # is "Year 1 of 2")
  * Link: Submit Payment (the same link that is presented in the portal)
  * Link to report of ACH Payments for the member application. (if using a ACH Payment record)



  


Contact record:

  * New field on the Related Contacts RG: Enable Portal Access (boolean, editable for users with the "allow portal access" permission; enables the related contact to log into portals the school or business)
    * Warning on save if there are no linked SPE Member Applications for the contact.
    * When this field is filled, a Portal Access Welcome email will be generated to be sent to the contact.



  


New Report: Portal Activity Log

  * This is the standard activity report filtered down to modifications made by the dedicated portal user.



  


New Email Template: Portal Access Welcome

  


Possible new record: ACH Payment: (likely only justified if BillerIQ provides sizable information in the response.)
