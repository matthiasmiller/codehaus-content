5\. Family Application

Overview:

  * Allow families to log in with the email address listed on the Family record.
  * Allow the editing of the following Family record items: (prefilled in Green)
    * 1st Parent / Guardian First Name
    * 1st Parent / Guardian Last Name
    * 2nd Parent / Guardian First Name
    * 2nd Parent / Guardian Last Name
    * Phone
    * Address
    * City
    * State
    * Zip
    * School District
    * County
  * Allow the creation and editing of the current year's application: (prefilled in Green)
    * School Year 
      * Show years for all past applications as well as the current and next year.
    * Adults
    * Dependents
    * Income
    * Adding / Removing students and editing the following RG fields:
      * Student (dropdown of students who are linked to the family)
      * Grade
      * School
      * Learning Disability
