3\. School Application Portal

Send an email to schools to notify them that applications are open and provide information to get them into their online application.

  


Create a link on the FBEP website for a new school application.

  


Portal functionality:

  * Update contact information
  * Fill out and update school applications
  * View detailed financial information:
    * Distributions to the school
    * Withdrawals
    * Remaining balance
  * If School Managed: Upload family documents



  


  


System alerts:

  * School uploads family documents
  * School submits application
  * School updates application
  * School updates account



  


  


Changes to the School Application record:

  * Addition: link; "view family documents"; visible if the user has the permission to "view S3 documents" and the paperwork setting on the application is "School Managed".
    * This link leads to the S3 bucket linked to the application.
