10.2. Report Title

Basically 2 types of report titles across the accounting reports.

  


<Report Name> <Start Date> through <End Date>

[X] For example: Income Statement January 01, 2020 through December 31, 2020.

[X] This has the following versions based on the start and end dates. Coded in [[PC0108588]]

[X] SD- ED: Income Statement January 1, 2018 through December 31, 2018

[X] [NA] - ED: Income Statement <First transaction date> through December 31, 2018

[X] SD - [NA]: Income Statement January 1, 2018 through <Last transaction date>

[X] [NA] - [NA]: Income Statement <First transaction date> through <Last transaction date>

  


[X] Used in the following reports.

[X] Income Statement

[X] General Ledger

[X] Journal Entries

[X] Transactions

[X] All Bills

  


<Report Name> as of <End Date>

[X] For example: Balance Sheet as of December 31,2020

[X] Used in the following reports.

[X] Balance Sheet

[X] Trial balance

[X] Aging Bills
