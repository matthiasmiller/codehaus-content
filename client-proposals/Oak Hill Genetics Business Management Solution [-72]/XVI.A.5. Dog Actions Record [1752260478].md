16.1.5. Dog Actions Record

Overview: This record tracks all of the actions that are done to dogs. There should be the ability to deactivate items so that historical data isn't lost.

  


Accessed via: 

  


Sections and Fields: 

  * Action Name
  * Medication (optional drop list)
  * Frequency (number field; days)
  * Quantity (number field; units)
  * Age in Days (number field)
  * Date Active (date; defines the date when this should start being required, to avoid requiring a new action to be done to an old dog)
