16.8. Trip Record

Overview: This record is used to track delivery trips, with the ability to include multiple Orders on a delivery trip.Â 

  


The record includes the following sections and fields:

  * Overview:
    * Departure Date (date)
    * Departure Time (single line plain text field)
    * Pickup Date (date)
    * Pickup Time (single line plain text field)
    * Delivery Date (date)
    * Delivery Time (single line plain text field)
    * Vehicle (droplist)
    * Status (droplist)
    * Driver (droplist)
  * Stops (w/ either sequence or Move Up/Down)
    * Order -
    * Location (droplist)
    * Order Type (droplist)
    * Broker Contact (droplist)
