17.3.1. Action Search Report

  


Requirements

Clicking this menu options opens a page that prompts for the following filters. Clicking "Continue" opens a report of the corresponding data. 

  


Filter prompts: 

  * Action (droplist)
  * Medication (droplist)
  * Person (droplist)
  * Dog Serial # (single line plain text field)
  * Action Date From (date)
  * Action Date To (date)
  * Litter Date From (date)
  * Litter Date To (date)
  * Comp. Date From (date)
  * Comp. Date To (date)
  * Completed (droplist)



  


Report columns (possibly editable):

  * Date (date)
  * Dog (droplist)
  * Litter (droplist)
  * Action (single line plain text field)
  * Medication (single line plain text field)
  * Person (droplist)
  * Complete Dt. (single line plain text field)
  * Complete By (single line plain text field)
  * Complete (checkbox)
  * Complete (button)
