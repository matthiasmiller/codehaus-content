17.6. Order Reports

This report shows all orders in the Solution. There probably will be a separate version of this report for each Type of order (details to be specced out in in-depth design). 

  


Columns: 

  * Order #
  * Departure Date
  * Departure Time
  * Ship Date
  * Order Type
  * Buyer
  * Order Status
  * Facility
  * Driver
  * Vehicle
  * PO #
  * Unload order



  


Filters: 

  * Order Type (droplist) 
  * Buyer (droplist)
  * Ship Date From (date)
  * Ship Date To (date)
  * Ship To (droplist)
  * Status (droplist)
  * Order Type (droplist)
  * Facility (droplist)
  * Location (droplist)
  * Order # (single line plain text field)
  * PO # (single line plain text field)
  * Driver (droplist)
  * Vehicle (droplist)



  


Buttons: 

  * View Order 
  * New Animal Order (opens a new blank Animal-type order) 
  * New Feed Order (opens a new blank Feed-type order) 
  * New Transport Order (opens a new blank Transport-type order) 
  * Dispatch
