7.1.1. Contact Types Report

This is the standard AppHosting Contact Types report, which is a report of the Contact Types in the Solution. By default it would only show active items.

  


This would be accessed from a "Configure Contact Types" menu option on the Advanced Menu. 

  


Title: Contact Types

  


Columns: 

  * Type (link to record)
  * Individual (Yes/No)



  


Grouped by: Active/Inactive (active at the top)

  


Sorted by: Type name

  


Filters: 

  * Show inactive contact types (checkbox; defaulting to not filled)



  


Buttons: 

  * New Contact Type
