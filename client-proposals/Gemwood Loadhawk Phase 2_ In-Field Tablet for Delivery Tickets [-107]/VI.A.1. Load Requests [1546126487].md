6.1.1. Load Requests

Overview: This is a custom report of Load Requests.

  


Accessed from: Main Menu | Mobile App | Load Requests

  


Title: Load Requests

  


Columns: 

  * Date
  * Member
  * Description (link to the record)
  * Items ("<Qty> \- <Thickness> <Species> <Grade>" for every row in the itemization embedded spreadsheet, separated by newlines)
  * Filled BFT (sum of BF Qty for all packs linked to the load request)



  


Grouped by: 

  * Open/Closed (if Include Closed is checked) 



  


Sorted by: Date (most recent at the top)

  


Filters: 

  * Include Closed (checkbox)



  


Buttons: 

  * New Load Request
