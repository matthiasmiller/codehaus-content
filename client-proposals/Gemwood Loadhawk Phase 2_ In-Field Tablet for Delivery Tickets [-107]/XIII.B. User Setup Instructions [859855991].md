13.2. User Setup Instructions

Steps to create a member user:

  * Create a user profile (see details below)
    * User ID will be used as the username in the app
    * Enter and confirm password (uncheck Change Password Next Login)
    * Make user a part of the members user group
  * On the member contact record, link the user profile.
  * Send the username and password to the member.
