3.1.1. Grouping 1: Load Requests

Grouping 1: Load Requests

This grouping contains all open load requests for the members.

  * Load requests contain the following information:
    * Species
    * Thickness
    * Grade
    * Required BFT
    * List of entered packs
      * Pack #
      * BFT
        * Button to remove packs
    * Total BFT of entered packs
    * Button to add a pack
      * Displays species + thickness + grade + pack #
      * Prompts for BFT
      * Buttons to Cancel and Add Pack
