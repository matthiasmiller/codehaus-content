1.2.1. QR Code Scanning

  


Requirements

The app uses a QR code reader to scan codes on the log tags. The codes communicate the Tag #, so scanning a code will do one of the following: 

  * Add the Tag # to a new Log record 
  * (Maybe) Find a log
