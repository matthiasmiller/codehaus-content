3.32. Delivery Scheduling - Requested Deliveries

We will provide a report of all Requested Deliveries. It will default to showing all scheduled and pending deliveries, as well as all deliveries completed or canceled in the past 30 days. It would show:

  


  * Date/Time
  * Item Number
  * Item Description
  * Name
  * Address
  * Phone Number
  * Delivery Status (Scheduled/Delivered/Pending/Canceled)



  


It would include a prompt to adjust the number of days.
