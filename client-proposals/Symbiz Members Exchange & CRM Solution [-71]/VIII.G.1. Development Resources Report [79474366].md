8.7.1. Development Resources Report

  


Requirements

This is a report of all Development Resource records, with various filtering options. Note that all columns with text include text wrapping as needed, to accommodate printing in a vertical format

  


This is accessed from: Members Menu | Development Resources | Development Resources (bypasses the filter screen to open the report directly)

  


Title: Development Resources

  


Columns: 

  * Cover Image (conditionally visible based on the corresponding filter; if visible, displays the Cover Image thumbnail from the record) 
  * Title (link to record)
  * Creator 
  * # Reviews (displays the number of Reviews for the Resource; link to open the Development Resource Reviews Report, filtered to the Resource) 
  * Average Rating 
  * Primary Category 
  * Additional Categories (displays each distinct Category included on the All Categories embedded spreadsheet except for the Primary Category, comma-separated, in the sequence that they appear on the embedded spreadsheet)
  * Status 
  * Submitted By (displays the Member name without the Contact ID)



  


Grouped by: Selection on the "Group by" filter:

  * If "Recent": "Past 2 weeks" group at the top, "All Older" group below; looking at & sorted by Submitted Date
  * If "Primary Category": Alphabetically by Category 
  * If "Media Type": Alphabetically by Media Type
  * If blank: No grouping at all; simply the full list sorted by the "Sort by" selection



  


Sorted by: Selection on the "Sort by" filter:

  * If "Submitted Date": most recent date at the top
  * If "Creator": Alphabetically by Creator, then alphabetically by Title
  * If "Primary Category": Alphabetically by Category, then alphabetically by Title
  * If "Title": Alphabetically by Title



  


Filters: 

  * Media Type (drop list of Media Type items; default is blank = all)
  * Status (drop list of all Development Resource Statuses; default is blank = all) 
  * Categories (multi-select drop list of all Development Resource Categories; the report displays results for all Resources that have had any of the selected Category chosen at least once) 
  * Primary Category Only (checkbox; defaults to unchecked; if checked, the report displays results only for Resources that have the selected Category(ies) as the Primary Category)
  * Title (plain text field; matches on partial searches; defaults to blank)
  * Creator (drop list of all Creators; default is blank = all) 
  * Submitted Date Start (date; defaults to blank = all)
  * Submitted Date End (date; defaults to blank = all)
  * Approved Date Start (date; defaults to blank = all; looks at the Status Date on the record, if Status = Approved)
  * Approved Date End (date; defaults to blank = all; looks at the Status Date on the record, if Status = Approved)
  * Group by (drop list of Recent / Category / Media Type / blank = all; defaults to "Recent") 
  * Sort by (required; drop list of Submitted Date / Status / Creator / Primary Category / Title; defaults to "Submitted Date")
  * Include Cover Images (checkbox; defaults to cleared; filling it includes the Cover Image column on the report)



  


Buttons: 

  * New Resource (opens a new blank Development Resource record page)
  * Print Resource List (generates the Resource List Printout PDF, based on the selected filters; see note below for generating an Excel export)



  


Menu Visibility: All users

  


Other Notes:

  * If a user wishes to generate an Excel file export of this report, they can select the "Open in new tab" option in the top left corner of the left pane, which will open only that pane in a new tab. Then the user can click the Advanced drop-down arrow in the top right corner of the screen and select "Excel".
