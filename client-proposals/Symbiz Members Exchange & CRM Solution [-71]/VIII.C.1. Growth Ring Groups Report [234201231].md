8.3.1. Growth Ring Groups Report

  


Requirements

Overview: This is a two-pane report of Growth Ring Group records, with various filtering options, and the corresponding members for a selected Group.

  


Accessed from:

  * Members Menu | Members & Groups | All Growth Ring Groups (bypasses the filter screen to open the report directly)
  * Admin | Group Management | Find Group



  


Title: Growth Ring Groups

  


Left Pane (displays Growth Ring Groups, based on the selected filter criteria):

Columns: 

  * Group Description (displays the Description; link to open record)
  * Group Type
  * Facilitator 
  * Facilitator State/Province
  * Last Facilitator Change 
  * Next Facilitator Change (displays in orange text if the date is within 30 days in the future; displays in red text if the date is in the past)
  * # of Members
  * Region
  * Regional Rep
  * Last Rep Visit 
  * Next Rep Visit (displays in orange text if the date is within 30 days in the future; displays in red text if the date is in the past)



  


Grouped by: Active/Inactive

  


Sorted by: Group Description

  


Right Pane (displays members of the most recently-selected Group row in the left pane)

Column(s): 

  * Member (displays the name; link to open the corresponding Contact record) 



  


Filters (all apply directly to the left pane): 

  * Group Description (drop list of Group Descriptions; filters down as you type; defaults to blank = all)
  * Facilitator (drop list of all Facilitators; defaults to blank = all)
  * Facilitator State/Province (drop list of all States/Provinces; defaults to blank = all) 
  * Region (drop list of all Regions; defaults to blank = all) 
  * Regional Rep (drop list of all Regional Reps; defaults to blank = all)
  * Info Meeting (drop list of Event IDs for all non-canceled Info Meeting-type Events; defaults to blank = all) 
  * Group Type (drop list of Group Type items; defaults to blank = all)
  * Active/Inactive (visible only for Regional Reps & Super Admin; drop list of Active / Inactive / All; defaults to Active)



  


Buttons: 

  * N/A



  


Menu Visibility: 

  * On the Members Menu: All users
  * On the Admin Menu: Regional Reps and Super Admins (by virtue of the Admin Menu being restricted)

  * N/A
