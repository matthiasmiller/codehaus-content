8.1.8. Configure Development Resource Media Types Report

  


Requirements

This is an editable report of all Development Resource Media Types.

  


This is accessed from: Advanced | Configuration | Configure Development Resource Media Types 

  


Title: Development Resource Media Types

  


Columns: 

  * Sort Order 
  * Media Type (link to record)
  * Include Downloads Section (displays Yes / No)



  


Grouped by: Active/Inactive

  


Sorted by: Sort Order (lowest number at the top) 

  


Filters: 

  * N/A



  


Buttons: 

  * New Media Type (opens blank new Development Resource Media Type record)



  


Menu Visibility: 

  * Super Admin and Regional Reps only (by virtue of the Advanced menu being visible only to Super Admins and Regional Reps)



  


Other Notes: 

  * N/A
