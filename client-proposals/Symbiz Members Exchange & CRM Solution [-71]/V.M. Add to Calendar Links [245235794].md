5.13. Add to Calendar Links

  


Requirements

The Solution includes "Add to Calendar" links, used to add certain meetings / events to a member's digital calendar. This is initially included only in user notifications, but eventually maybe could be included elsewhere as well.  

  


  * Add to Google Calendar:
    * Opens a new google calendar item at [https://www.google.com/calendar/](https://www.google.com/calendar/), with the following details: 
      * Title: "Growth Ring Meeting"
      * Start date & time 
      * End date & time (calculated from the Meeting Duration)
      * "Meeting Details" link in invite body/notes 
      * Location 



  


  * Add to Outlook Calendar:
    * Opens a new Outlook calendar item at [https://outlook.office.com/calendar/](https://outlook.office.com/calendar/)
      * Title: "Growth Ring Meeting"
      * Start date & time 
      * End date & time (calculated from the Meeting Duration)
      * "Meeting Details" link in invite body/notes 
      * Location
