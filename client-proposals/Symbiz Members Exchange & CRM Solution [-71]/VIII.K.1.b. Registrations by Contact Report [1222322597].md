8.11.1.2. Registrations by Contact Report

Overview: This is a filtered version of the the Event Registrations Report that displays Registrations (of all Event Types) for one Contact.

  


Accessed from:

  * "View My Registrations Report" link in the Event Registration section of the Contact record



  


The following filters are set to something other than their default:

  * Registrant = corresponding Contact



  


Other Notes:

  * See the spec for the Event Registrations Report for additional details (title format, visible columns & buttons, etc.)
