8.3.2. Open Growth Ring Groups for Member Report

  


Requirements

Overview: This is a simple report of Growth Ring Groups for a selected Member.

  


Accessed from the My Growth Ring menu option, if the logged-in user is part of more than one Growth Ring Group. 

  


Title: Open Growth Rings for Member

  


Column(s): 

  * Growth Ring Group Description (link to open record) 



  


Grouped by: N/A 

  


Sorted by: Growth Ring Group Description

  


Filter(s): 

  * Member (required; drop list of all Member-type Contacts with an active membership) 



  


Buttons: 

  * N/A



  


Menu Visibility: Each user can see their own

  


Other Notes:

  * N/A
