8.15.1. Open Regions for Rep Report

  


Requirements

This is a simple report of Regions for a selected Regional Rep.

  


This is accessed from Regional Rep | My Region Management | My Region if the logged in user is a Rep for more than one Region.

  


Title: Open Regions for Rep

  


Column(s): 

  * Region Name (link to open record) 



  


Grouped by: N/A 

  


Sorted by: Region Name (alphabetically)

  


Filter(s): 

  * Regional Rep (required; drop list of all Regional Reps) 



  


Buttons: 

  * N/A



  


Menu Visibility: Each user can see their own

  


Other Notes:

  * N/A
