8.3.1.1. Growth Ring Groups by Info Meeting Report

  


Requirements

Overview: This is the Growth Ring Groups report, filtered to a selected Info Meeting.

  


Accessed from: 

  * "View <X> Linked Group(s)" link in the "Launch Meeting (Internal)" section of the Event record
  * Admin | Growth Ring Group Management | Find Group (the Admin user can select an Info Meeting from the "Info Meeting" filter that opens from this menu option) 



  


Title: Growth Ring Groups

  * Note that a possible future change could be to add the Event Name or ID to the report title. 



  


The following filters are set to something other than their default:

  * "Info Meeting" = corresponding Info Meeting-type Event record
