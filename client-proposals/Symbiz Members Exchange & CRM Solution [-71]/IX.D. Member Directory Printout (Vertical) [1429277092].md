9.4. Member Directory Printout (Vertical)

  


Requirements

Purpose/Overview: This is a customized PDF printout of the Member Directory Report. This can be printed as a reference, for example for a member who does not have internet access. Note that this vertical format generates the PDF without the Bio column.

  


Printed From: 

  * "Print Vertical" button on the Member Directory Report(s) (available to all users)
  * Admin | Admin Printing | Print Member Directory (Vertical)



  


File Format/Name: PDF as generated from the report

  


Fields to be Filled: This displays the columns and rows of the report based on the selected filters, but always without the Bio and Username columns.

  


Template: N/A

  


Handling Page Breaks: Break as needed between rows

  


Other Notes:

  * The printout includes the Symbiz logo at the top of every page (can be small).
  * The printout will have alternating background colors for the rows (light gray and white).
