9.5. Member Directory Printout (Horizontal)

Purpose/Overview: This is a customized PDF printout of the Member Directory Report. This can be printed as a reference, for example for a member who does not have internet access. Note that this horizontal format generates the PDF with the Bio column.

  


Printed From: 

  * "Print Horizontal" button on the Member Directory Report(s) (available to all users)
  * Admin | Admin Printing | Print Member Directory (Horizontal)



  


File Format/Name: PDF as generated from the report

  


Fields to be Filled: This displays the columns and rows of the report based on the selected filters, with the Bio column and without the Username column.

  


Template: N/A

  


Handling Page Breaks: Break as needed between rows

  


Other Notes:

  * The printout includes the Symbiz logo at the top of every page (can be small).
  * The printout will have alternating background colors for the rows (light gray and white).
