8.5.2. Monthly Mailing List Report

  


Requirements

This is a report of current offline Members without email addresses. This report will be used to generate an Excel file to print address labels for mailing the Symbuzz Extra (Monthly).

  


This is accessed from Admin | Symbuzz Extra | Monthly Mailing List. 

  


Title: Monthly Mailing List

  


Columns: 

  * Name (link to the record)
  * Address 1
  * Address 2
  * City 
  * State 
  * Zip 
  * Country



  


Grouped by: N/A

  


Sorted by: Alphabetically by name

  


Filters: 

  * N/A



  


Buttons: 

  * N/A



  


Menu Visibility: Regional Reps and Super Admins (by virtue of the Admin Menu being restricted)

  


Other Notes: 

  * An Excel file version of this report is included with the Print Symbuzz Extra (Monthly) email notification.
  * To manually generate an Excel file, a user can open the Advanced drop-down menu for the report (via the down arrow button in the top right corner of the report), and click the "Excel" option.
