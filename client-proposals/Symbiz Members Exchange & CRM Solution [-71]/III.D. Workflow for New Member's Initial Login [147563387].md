3.4. Workflow for New Member's Initial Login

The following is a basic description of how a new Online Member can log into the Members Exchange for the first time:

  


  * If the Member is an Online Member, the Members Exchange Welcome Email is sent as soon as the "Activate Membership" button is clicked on the Member's Contact Record.

  * When the member receives the Members Exchange Welcome Email, they should follow the provided instructions to access the Members Exchange and log in for the first time.
