8.9.2. Knowledge Tree Quick Entry Report

  


Requirements

This is an editable report of Knowledge Tree Entries for a selected Member (and Development Resource, if desired). The user can use this report to batch enter new Knowledge Tree Entries.

  


This is accessed from: 

  * Members Menu | My Knowledge Tree | Knowledge Tree Quick Entries
  * The Add Knowledge Tree Entries link on the Development Resource Review record 



  


New Records Details:

  * Source Record Type: N/A 
  * Source Record Subset: N/A
  * Source Repeating List: 1-20 (the number of new entries that show up on the report) 
  * New Record Fields and Values:
    * Member (read-only; filled from the corresponding Filter - see below)  
    * Development Resource (optional; drop list of all Development Resources; filled according to the corresponding Filter - see below)
    * Knowledge Tree Entry Text (required; multi-line plain text field)
    * Page Number (optional; plain text field)
    * Knowledge Tree Branch (optional; drop list of Branches for the selected Member)
  * New Record Match On (unique identifying fields):
    * Member 
    * Knowledge Tree Entry Text



  


Title: Knowledge Tree Quick Entries

  


Columns: See the items and descriptions under "New Record Fields and Values" above.

  


Grouped by: N/A

  


Sorted by: N/A

  


Filters: 

  * Member (required; drop list of all Members that the user has access to; if opened from the menu option, defaults to the current User; if opened from a Review record, defaults to the name of the Member on the Review; otherwise defaults to the current user)
  * Resource (drop list of all Resources; if opened from the menu option, defaults to blank = all; if opened from a Review record, defaults to the Resource for the Review)



  


Buttons: 

  * Finish (clicking this button creates and saves a new record for each row with data, and takes the user back to the Knowledge Tree Entries Report for the selected Member)



  


Menu Visibility: All users

  


Save Type: "Finish" Button

  


Other Notes:

  * N/A
