8.1.6. Configure Growth Ring Group Types Report

  


Requirements

This is a report of all Growth Ring Group Types.

  


This is accessed from Advanced | Configuration | Configure Growth Ring Group Types. 

  


Title: Growth Ring Group Types

  


Columns: 

  * Group Type (link to record)



  


Grouped by: Active / Inactive

  


Sorted by: Group Type (alphabetical)

  


Filters: 

  * N/A



  


Buttons: 

  * New Group Type (opens blank new Growth Ring Group Type record)



  


Menu Visibility: 

  * Super Admin and Regional Reps only (by virtue of the Advanced menu being visible only to Super Admins and Regional Reps)



  


Other Notes:

  * N/A
