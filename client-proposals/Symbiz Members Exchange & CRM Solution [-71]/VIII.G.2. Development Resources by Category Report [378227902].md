8.7.2. Development Resources by Category Report

  


Requirements

This is a report of Development Resources of a selected Category. 

  


This is accessed from: Members Menu | Development Resources | Resources by Category. 

  


Clicking the above menu option opens a prompt page with the filter(s) listed below, and a Continue button. The user selects the desired filter options and clicks Continue to open the report. 

  


Title: Development Resources by Category - <Selected Category> 

  


Columns: 

  * Title (link to record)
  * Creator 
  * # of Category Selections  
  * Submitted By (displays the Member name without the Contact ID)
  * Status 
  * Average Overall Rating 



  


Grouped by: 

  * Primary Category (includes all Resources that have the selected Category as the Primary Category)
  * Non-Primary Category (includes all Resources that have had the selected Category chosen at least once)



  


Sorted by: # of Category Selections 

  


Filters: 

  * Category (required; drop list of all Development Resource Categories; defaults to blank) 



  


Buttons: 

  * Print Resource List (generates the Resource List Printout PDF, based on the selected filters; see note below for generating an Excel export)



  


Menu Visibility: All users

  


Other Notes: 

  * Note that only one Category can be selected at a time.
