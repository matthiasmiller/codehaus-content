9.3. Resource List Printout

Purpose/Overview: This is a PDF printout of the left pane of the Development Resources report, showing only basic details such as the Title, Author / Creator, and Overview. This can be printed as a reference, for example for a member who does not have internet access. It can also be printed to be included in the binders that are handed out at the Annual Summit.

  


Printed From: 

  * The Development Resource Reports (available to all users via the "Print Resource List" buttons)
  * Admin | Manage Development Resources | Generate Resource List Printout (opens a prompt screen with the report filters, to allow the user to choose whether to print with or without the Cover Image column) 



  


File Format/Name: PDF as generated from the report

  


Fields to be Filled: This displays the columns and rows of the report based on the selected filters.

  


Template: N/A

  


Handling Page Breaks: Break as needed between rows

  


Other Notes:

  * The printout includes a small Symbiz logo at the top of every page.
  * The printout will print in a vertical layout so that it can be included in notebooks.
  * This will only print the left pane of the Development Resources Report.
