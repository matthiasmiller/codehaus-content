8.1.4. Configure Church Affiliations Report

  


Requirements

This is a report of all Church Affiliations.

  


This is accessed from: Advanced | Configuration | Configure Church Affiliations 

  


Title: Church Affiliations

  


Columns: 

  * Affiliation Name (link to record)



  


Grouped by: Active/Inactive

  


Sorted by: Affiliation Name (alphabetical)

  


Filters: 

  * N/A



  


Buttons: 

  * New Church Affiliation (open blank new Church Affiliation record)



  


Menu Visibility: 

  * Super Admin and Regional Reps only (by virtue of the Advanced menu being visible only to Super Admins and Regional Reps)



  


Other Notes: 

  * N/A
