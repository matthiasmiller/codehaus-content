8.1.1. Configure Regions Report

  


Requirements

This is a report of all regions.

  


This would be accessed from: Advanced | Configuration | Configure Regions

  


Title: Regions

  


Columns: 

  * Region (link to record)
  * Regional Rep



  


Grouped by: Active / Inactive

  


Sorted by: Region (alphabetical)

  


Filters: 

  * Region (drop list of Regions)
  * Regional Rep (drop list of Regional Reps)



  


Buttons: 

  * New Region (opens a blank new Region Record)



  


Menu Visibility: 

  * Super Admin and Regional Reps only (by virtue of the Advanced menu being visible only to Super Admins and Regional Reps)



  


Other Notes:

  * N/A
