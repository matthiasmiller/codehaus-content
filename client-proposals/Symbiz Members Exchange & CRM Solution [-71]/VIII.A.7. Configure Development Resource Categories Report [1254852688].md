8.1.7. Configure Development Resource Categories Report

  


Requirements

This is a report of Development Resource Categories. 

  


This is accessed from: Advanced | Configuration | Configure Development Resource Categories

  


Title: Development Resource Categories

  


Columns: 

  * Category Name (link to open record)



  


Grouped by: Active/Inactive

  


Sorted by: Category Name (alphabetical)

  


Filters: 

  * N/A



  


Buttons: 

  * New Category (opens a blank new Development Resource Category Record)



  


Menu Visibility: 

  * Super Admin and Regional Reps only (by virtue of the Advanced menu being visible only to Super Admins and Regional Reps)



  


Other Notes:

  * N/A
