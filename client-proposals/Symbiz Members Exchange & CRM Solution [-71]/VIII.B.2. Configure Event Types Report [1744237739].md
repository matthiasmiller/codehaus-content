8.2.2. Configure Event Types Report

  


Requirements

This is a report of all Event Types.

  


This is accessed from Advanced | Configuration | View Event Types. 

  


Title: Event Types

  


Columns: 

  * Event Type (link to record)
  * Default Event Visibility
  * Requires Food Notes if Scheduled (displays as Yes / No) 
  * Always Requires Registration Deadline (displays as Yes / No)
  * Displays Region & Rep (displays as Yes / No)



  


Grouped by: Active / Inactive

  


Sorted by: Event Type Name (alphabetical)

  


Filters: 

  * N/A



  


Buttons: 

  * N/A (new Event Types cannot be configured in the Solution) 



  


Menu Visibility: 

  * Super Admin and Regional Reps only (by virtue of the Advanced menu being visible only to Super Admins and Regional Reps)



  


Other Notes:

  * This report is called "View Event Types" rather than "Configure..." because the Event Types are fully read-only.
