8.1.5. Configure Industries Report

  


Requirements

This is a report of all Industry records.

  


This is accessed from Advanced | Configuration | Configure Industries

  


Title: Industries

  


Columns: 

  * Industry Name (link to record)



  


Grouped by: Active / Inactive

  


Sorted by: Industry Name (alphabetical)

  


Filters: 

  * N/A



  


Buttons:

  * New Industry (opens a blank new Industry Record)



  


Menu Visibility: 

  * Super Admin and Regional Reps only (by virtue of the Advanced menu being visible only to Super Admins and Regional Reps)



  


Other Notes:

  * N/A
