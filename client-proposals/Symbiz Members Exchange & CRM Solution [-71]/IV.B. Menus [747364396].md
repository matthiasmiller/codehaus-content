4.2. Menus

  


Requirements

The Symbiz Members Exchange will have the following menu tabs: 

  


  * Members Menu menu (visible for all users; contains the most commonly used menu options): 
    * Customized menu options (others specified in the corresponding sections of the proposal):
      * My Details section: 
        * My Symbiz Profile (opens the Contact record for the current user)
        * My Current Goals (if the current user has only 1 Goals record with a Status of In Progress or Overdue, this opens that record directly; if the current user has more than 1 Goal record with a Status of In Progress or Overdue, including because of being part of multiple Groups, this opens the Open Growth Ring Goals for Member report - see corresponding section of the proposal) 
        * My Knowledge Tree Branches (opens the Configure Knowledge Tree Branches report, filtered to the current user)
      * My Next Meeting Details 
        * My Next Growth Ring Meeting (opens the Growth Ring Meeting record for the current user for the next upcoming meeting; if a member of more than one Group, this opens the My Next Growth Ring Meeting report - see corresponding section of the proposal)
        * Print Book/Resource Impact Worksheet (opens the PDF saved in the "Book/Resource Impact Worksheet" field on the Silverloom Settings page)
        * My Next Meeting Goals
        * Print Meeting Goals Worksheet (opens the "Print Meeting Goals Worksheet" prompt screen - see corresponding spec)
        * My Next Resource Review 
        * My Next Knowledge Tree Entries
      * My Growth Ring Group Details section:
        * My Growth Ring Group (opens the Growth Ring Group record for the current user; if a member of more than one Group, this opens the Open Growth Rings for Member report - see corresponding section of the proposal)
      * My Knowledge Tree section:
      * Development Resources section:



  


  * Group Facilitation menu: (visible for Facilitators, Regional Reps, and Super Admins; contains menu options for managing downlines, groups, regions, and other upline responsibilities):
    * Customized menu options (others specified in the corresponding sections of the proposal):
      * Downline Management section (visible for Facilitators, Regional Reps, and Super Admins):
        * New Review for Downline (opens a New Review Prompt Page)
      * My Group Management section (visible for Facilitators and Super Admins):
        * Manage Growth Ring Meetings (opens the Growth Ring Meetings report, filtered to the Group for which the user is Facilitator; this allows the Facilitator easy access to view and manage the Group's meetings)
        * Goals for Group Members (opens the Growth Ring Goals report, filtered to the Group for which the user is Facilitator; this allows the Facilitator easy access to view and edit the Group's goals)
        * Print Book/Resource Impact Worksheet (opens the PDF saved in the "Book/Resource Impact Worksheet" field on the Silverloom Settings page)
        * Print Life Goals Worksheet
        * Print Meeting Goals Worksheet (blank) (opens a blank version of the Meeting Goals Worksheet PDF)
      * My Region Management section (visible for Regional Reps and Super Admins):
        * My Region (opens the Region record for the current user; if a Rep of more than one Region, this opens the Open Regions for Rep report; disabled / grayed out if user ≠ Regional Rep)
      * Ad-Hoc Printing section with the following option(s) (visible for Facilitators, Regional Reps, and Super Admins): 
        * Group Facilitation | Ad-Hoc Printing | Print Growth Ring Health Report (blank) (link; opens the PDF from the "Growth Ring Health Report" memo on the Silverloom Settings page, so that it can be printed for an offline Secretary to write out their posts)



  


  * Facilitator menu (visible for Facilitators and Super Admins; contains menu options for managing groups, and other Facilitator-related responsibilities):
    * Customized menu options (others specified in the corresponding sections of the proposal):
      * My Group Management section:
        * Manage Growth Ring Meetings (opens the Growth Ring Meetings report, filtered to the Group for which the user is Facilitator; this allows the Facilitator easy access to view and manage the Group's meetings)
        * Goals for Group Members (opens the Growth Ring Goals report, filtered to the Group for which the user is Facilitator; this allows the Facilitator easy access to view and edit the Group's goals)



  


  * Regional Rep menu (visible for Regional Reps and Super Admins; contains menu options for managing regions and other Regional Rep-related responsibilities):
    * Customized menu options (others specified in the corresponding sections of the proposal):
      * My Region Management section:
        * My Region (opens the Region record for the current user; if a Rep of more than one Region, this opens the Open Regions for Rep report - see corresponding section of the proposal)



  


  * Admin menu (visible for Super Admin and Regional Reps; contains menu options for various admin-level reports, membership management / CRM, and other features).
    * Customized menu options (others specified in the corresponding sections of the proposal):
      * Lead Management section:
        * New Lead (opens a new blank Lead-type Contact record)
      * Membership Management section:
        * Admin | Membership Management | New Member (opens a new Contact record with Type = Member) 
      * Admin Printing section:
        *       * Growth Ring Group Management section: 
        * New Group (opens a new blank Growth Ring Group record) 
        * Find Group (opens a prompt page with prompts corresponding to the filters on the Growth Ring Groups report, allowing the user to enter search criteria for a Group; clicking Continue opens the Growth Ring Groups report, filtered to the correspondingTODO_ criteria) 
        * All Growth Ring Meetings (opens the "Growth Ring Meetings" report filtered to show all Meetings in the last 6 months)
        * Print Book/Resource Impact Worksheet (opens the PDF saved in the "Book/Resource Impact Worksheet" field on the Silverloom Settings page)
        * Print Life Goals Worksheet
        * Print Meeting Goals Worksheet (blank) (opens a blank version of the Meeting Goals Worksheet PDF)



  


  * Advanced menu (visible for Super Admin and Regional Reps; contains menu options for various settings, configurations, etc.)
