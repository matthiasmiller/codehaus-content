7.9.2. Event - Planning Details (Internal) Section

  * "Planning Details (Internal)" section: (visible for all Event Types; visible for Regional Reps and Super Admins): 



  


  * Execution Status (optional; drop list of Execution Status list items; note: initially used for internal reference only and not tied to any features / automations, but could be changed in the future) 
  * Food Notes (plain text; defaults to blank; optional unless the "Requires Food Notes if Scheduled" checkbox is checked on the selected Event Type record, in which case it is required if "Event Status" = "Scheduled" and "Event Format" ≠ "Virtual")
  * Food Contact (optional; drop list of all active Contacts) 
  * View Contact (link; visible if "Food Contact" is not blank; opens the selected Contact record)
  * Planning Notes (memo; can be used for attaching documents and links)
