8.2.7. All Members Mailing List Report

  


Requirements

Overview: This is a report of all current Members (all Member-type Contacts with active memberships). This report will be used to generate an Excel file to print address labels for mailings.

  


Accessed from: Admin | Admin Printing | All Members Mailing List (bypasses the filter screen to open the report directly)

  


Title: All Members Mailing List

  


Columns: 

  * Name (link to the record) 
  * Address 1
  * Address 2
  * City
  * State
  * Zip
  * Country



  


Grouped by: N/A

  


Sorted by: Alphabetically by name

  


Filters: 

  * N/A



  


Buttons: 

  * N/A



  


Menu Visibility: Regional Reps and Super Admins (by virtue of the Admin Menu being restricted)

  


Other Notes:

  * To manually generate an Excel file, a user can open the Advanced drop-down menu for the report (via the down arrow button in the top right corner of the report), and click the "Excel" option.
