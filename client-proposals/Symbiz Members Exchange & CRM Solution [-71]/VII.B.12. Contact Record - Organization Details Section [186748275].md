7.2.12. Contact Record - Organization Details Section

  


Requirements

  * Organization Details custom section (visible if Contact Type = Organization):
    * Description (optional; multi-line plain text field) 
    * Employees (optional; plain text field; used to track the number of employees in the organization) 
    * Annual Revenue $ (visible for Regional Reps and Super Admin only; optional; plain text field)
    * View Members (link; opens the Member Directory report with the Organization defaulted)
