5.10.5.1. Notification: Approve New Resources

Name: Approve New Resources

Overview: This is a monthly notification that alerts Regional Reps and Super Admins of new Development Resources that have been submitted and are needing approval.

Record Type: Development Resource with Status = "Submitted"

Date: 

  * Email version: Monthly on the first day of the month if there is at least one Resource with Status = "Submitted".
  * In-app notification version: The first day that any user opens the Solution in a given month if there is at least one Resource with Status = "Submitted". 



Trigger / Scheduled Task Details: 

  * Email version: Scheduled task that runs on the 1st of each month at 5:00 am ET.
  * In-app notification version: When any user opens the Solution for the first time in a given month. 



Scheduled Task: Approve New Resources

Format: Member's choice

Send To: Super Admins and Regional Reps who have the notification enabled (Facilitators & Standard Members do not have the option for this notification)

Subject / Message:

Subject: Approve New Resources

Message: One or more Development Resources have been submitted and need to be approved. 

  


See the report: <link to the Development Resource Report, grouped by Recent and sorted by Submitted Date, and with the "Status" filter set to "Submitted">
