8.4.1. All Growth Ring Meetings Report

  


Requirements

Overview: This is a report of Growth Ring Meeting records.

  


Accessed from Admin | Growth Ring Group Management | All Growth Ring Meetings (bypasses the filter screen to open the report directly). 

  


Title: Growth Ring Meetings

  


Columns: 

  * Growth Ring Group (displays the Description) 
  * Facilitator 
  * View Meeting (link to open the Meeting record; displays as "View Meeting")
  * Meeting Date (link to record)
  * Meeting Time  
  * Time Zone (for US time zones, displays a simplified name; for other time zones, displays the full name) 
  * Meeting Status 
  * Host
  * Meeting Location
  * Development Resource Title
  * View Goals for Meeting (link; displays as "View Goals"; opens the Growth Ring Goals report for the corresponding Meeting)
  * Regional Rep



  


Grouped by: 

  * If the "Grouped by Month" checkbox is checked: Display the section headings in the format of <Month> <Year> and show the newest Month at the top.
  * If the "Grouped by Month" checkbox is not checked: No grouping. 



  


Sorted by: Meeting Date (greatest date at the top) 

  


Filters: 

  * Growth Ring Group (drop list of all Growth Ring Group Descriptions that the current user has access to; default is blank = all)
  * Facilitator (drop list of all Facilitators; visible for all Facilitators and uplines; default is blank = all)
  * Regional Rep (drop list of all Regional Reps; visible for all Facilitators and uplines; default is blank = all)
  * Meeting Date Start (date field; default is 6 months before the current date; blank = all) 
  * Meeting Date End (date field; default is blank = all)
  * Group By Month (checkbox; defaults to checked) 
  * Display Only Future Meetings (checkbox; default is cleared)



  


Buttons: 

  * Schedule New Meeting (visible for Facilitators and uplines; opens a blank new Growth Ring Meeting record with Group defaulted) 



  


Menu Visibility: 

  * Regional Reps and Super Admins (by virtue of the Admin Menu being restricted)



  


Other Notes:

  * Show the following message if the report is empty:



"No Growth Ring Meetings exist for this Group. Use the Schedule Next Meeting option on the Growth Ring record to create a meeting."
