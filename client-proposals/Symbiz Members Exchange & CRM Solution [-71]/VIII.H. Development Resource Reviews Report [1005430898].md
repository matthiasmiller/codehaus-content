8.8. Development Resource Reviews Report

  


Requirements

This is a report of Development Resource Reviews. 

  


This is accessed from: Members Menu | Development Resources | Resource Reviews (bypasses the filter screen to open the report directly)

  


Title: Development Resource Reviews

  


Columns: 

  * Reviewer's Name
  * Review Date (link to record)
  * Resource Title
  * Creator
  * Dev. Resource Primary Category



  


Grouped by: Resource Title (alphabetically)

  


Sorted by: Review Date (most recent at the top) 

  


Filters: 

  * Reviewing Meeting (drop list of Growth Ring Meetings that the current user has access to (i.e. Meetings that the current user attended); default is blank = all; if a Meeting is selected, the report shows results for the Members on the Attendance list of the corresponding Meeting)
  * Reviewer's Name (visible if Reviewing Group filter = blank; drop list of all reviewers; default is blank = all)
  * Creator (drop list of all Creators; default is blank = all)
  * Resource Title (drop list of all Resource Titles; filters / searches as the user types; default is blank = all)
  * Dev. Resource Primary Category (drop list of all Categories; default is blank = all)
  * Review Date Start (date field; default is blank = all)
  * Review Date End (date field; default is blank = all)



  


Buttons: 

  * N/A



  


Menu Visibility: All users

  


Other Notes: 

  * N/A
