6.7. Development Resource Creators List

  


Requirements

This is a user-editable simple list of Creators (i.e. authors, presenters, etc.) of Development Resources. 

  


This list is universal for all users, and all users can add to it via the "+" option on the drop list on the Development Resource record. 

  


The initial items for this list are to be set up by Symbiz, with assistance from CCI/CH as needed.
