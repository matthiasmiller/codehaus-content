7.7.1. New Review Prompt Page

  


Requirements

New Review Prompt Page: 

  


  * Opened from: 
    * "Review This Resource Now" link on Development Resource records (for Facilitators & uplines) 
    * Group Facilitation | Downline Management | New Review for Downline 



  


  * Contains the following: 
    * Prompt for Reviewer's Name (defaulting to blank) 
    * Prompt for Resource (defaulting to the Resource on which the link was clicked) 
    * "Continue" button



  


  * Behaviors: 
    * Clicking "Continue" opens a Review record: 
      * If a Reviewer's Name was selected who already has reviewed the Resource, opens that Member's existing Review; 
      * Otherwise opens a new Review with Reviewer's Name and Resource defaulted to the prompt selections.
