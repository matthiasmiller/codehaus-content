8.11.1.3. Info Meetings by Contact Report

Overview: This is a filtered version of the the Event Registrations Report that displays Info Meeting Registrations for one Contact.

  


Accessed from:

  * "<#> Info Meetings" link in the Lead Details section of the Contact record



  


The following filters are set to something other than their default:

  * Event Type = "Info Meeting"Â 
  * Registrant = corresponding Contact
