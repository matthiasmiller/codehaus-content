26.6.1.1. Core Values

The Core Values section of the Life Goal Worksheet would include the following fields:

  * Read-only text field containing the following paragraph:



Core values are deeply ingrained principles that guide our actions and serve as cultural cornerstones. Make a list with answers to the following questions: What non-work environment are you best engaged in? What jobs or roles do you do best in? What help or advice do people most often come to you for? This list should reveal what is close to your heart, what areas we are gifted in, and what brings a sense of meaning for us. The final test for each item on your list is to ask yourself: If this Core Value is violated, will it drain or anger me? Narrow it down to the best words and provide a description for each word.

  * Read-only bolded text field containing the following sentence:



Core values are beliefs which guide our behavior

  * Core Values Word (single-line text field)
  * Description (single-line text field)
  * Add Row (button)
