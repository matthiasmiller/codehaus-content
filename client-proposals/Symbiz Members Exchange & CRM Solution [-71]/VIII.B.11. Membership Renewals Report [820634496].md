8.2.11. Membership Renewals Report

  


Requirements

Overview: This is a report of all Member-type Contacts with active memberships needing renewal (all Member-type Contacts with the Active Membership checkbox checked for whom the latest row of the Membership History embedded spreadsheet is unpaid and is not the only row, and the "Membership Year Start / Renewal" date for that latest row is on or before the first day of the upcoming month (to catch due and overdue memberships).

  


Accessed from:

  * Admin | Membership Management | Membership Renewals
  * The Annual Renewal Notices Due user notification



  


Title: Membership Renewals 

  


Columns: 

  * Member Name (link to open record)
  * Paid Membership Start (displays the current Paid Membership Start date)
  * Renewal Date (displays the Membership Year Start / Renewal date for the latest row) 



  


Grouped by: 

  * Overdue (grouping includes all memberships that are past their Renewal Date) 
  * Due Today (grouping includes all memberships that are due on the current day; used if viewing the report on the first day of a month) 
  * Due Next Month (grouping includes all memberships due for renewal in the upcoming month)



  


Sorted by: Member Name (alphabetically)
