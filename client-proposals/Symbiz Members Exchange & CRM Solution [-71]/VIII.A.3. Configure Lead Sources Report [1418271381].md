8.1.3. Configure Lead Sources Report

  


Requirements

This is a report of all Lead Sources.

  


This is accessed from: Advanced | Configuration | Configure Lead Sources 

  


Title: Lead Sources

  


Columns: 

  * Lead Source Name (link to record)



  


Grouped by: Active/Inactive

  


Sorted by: Lead Source Name (alphabetical)

  


Filters: 

  * N/A



  


Buttons: 

  * New Lead Source (open blank new Lead Source record)



  


Menu Visibility: 

  * Super Admin and Regional Reps only (by virtue of the Advanced menu being visible only to Super Admins and Regional Reps)



  


Other Notes: 

  * N/A
