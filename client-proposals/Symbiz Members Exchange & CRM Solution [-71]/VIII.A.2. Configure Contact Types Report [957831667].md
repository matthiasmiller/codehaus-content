8.1.2. Configure Contact Types Report

  


Requirements

This is the standard AppHosting Contact Types report, which is a report of the Contact Types in the Solution. By default it would only show active items but includes a filter to show inactive items.

  


This would be accessed from: Advanced | Configuration | Configure Contact Types  

  


Title: Contact Types

  


Columns: 

  * Type (link to record)
  * Individual (Yes/No)
  * Organization (Yes/No)



  


Grouped by: Active/Inactive (active at the top)

  


Sorted by: Type name

  


Filters: 

  * Show inactive contact types (checkbox; defaulting to not filled)



  


Buttons: 

  * New Contact Type (opens a blank new Contact Type Record)



  


Data Access:

  * Super Admin and Regional Reps only (by virtue of the Advanced menu being visible only to Super Admins and Regional Reps)



  


Other Notes:

  * N/A
