5.8. Restricted Data Policies

  


Requirements

Restricted Data Policies can be accessed and configured via Advanced | Admin | Configure Restricted Data Policies. This is an additional tool that can be used to restrict or allow access to certain features in the Solution. 

  


Once a policy is configured, it can be selected on a User Group record (not to be confused with the Growth Ring Group) to apply it to all users of that group. 

  


The following are configured in the Symbiz Members Exchange & CRM (but note that the Wiki Categories are easily changed in the software, so the below tables can easily become out of date): 

  


Batch Processes: 

Record| Category| Permission  
---|---|---  
Wiki Page| <Default>| Can Edit & Delete  
Wiki Page| Growth Ring Management| Can Edit & Delete

Wiki Page| Members & Facilitators| Can Edit & Delete  
Wiki Page| Regional Reps & Super Admins| Can Edit & Delete  
Wiki Page| Super Admins Only| Can Edit & Delete
