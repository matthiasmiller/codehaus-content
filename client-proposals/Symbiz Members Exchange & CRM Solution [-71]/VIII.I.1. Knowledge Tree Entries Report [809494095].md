8.9.1. Knowledge Tree Entries Report

  


Requirements

This is an editable report of Knowledge Tree Entries for a selected Member.

  


This is accessed from Members Menu | My Knowledge Tree | MyKnowledge Tree Entries (bypasses the filter screen to open the report directly)

  


Title: Knowledge Tree Entries for <Selected Member's Name>

  


Columns: 

  * Member
  * Entry ID 
  * View Entry (link to open record; displays as "View / Edit Entry") 
  * Resource
  * Knowledge Tree Branch (editable; same as the corresponding field on the Knowledge Tree Entry Record) 
  * Knowledge Tree Entry Text (wrapping text)



  


Grouped by: Knowledge Tree Branch (alphabetically); at the top, have a grouping for "No Branch"

  


Sorted by: Entry ID

  


Filters: 

  * Member (required; drop list of all Members that the user has access to; defaults to the current user)
  * Resource (drop list of Resources for which the selected Member has at least one linked Knowledge Tree Entry; defaults to blank = all)
  * Knowledge Tree Branch (drop list of all Branches for the selected Member; default is blank = all)



  


Buttons: 

  * New Entry (opens a new Entry record for the selected Member)



  


Menu Visibility: All users

  


Other Notes: N/A
