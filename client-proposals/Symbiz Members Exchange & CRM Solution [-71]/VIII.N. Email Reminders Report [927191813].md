8.14. Email Reminders Report

  


Requirements

This is a report of Email Reminders in the Solution.

  


This is accessed from Admin | Email Reminders | Email Reminders. 

  


Title: Email Reminders 

  


Columns:

Grouped by: Active / Inactive 

  


Sorted by: Name (alphabetically)

Buttons: 

  * New Record (opens a blank new Email Reminder record) 



  


Menu Visibility: Super Admins only (hidden for Regional Reps, who also have access to the Admin menu)
