9.1. Growth Ring Worksheet Printout

Purpose/Overview: This is a PDF printout that contains the fields on the Development Resource Review record and Growth Ring Goals record. This is printed as an empty document and filled out by hand. For a member who does not have internet access, a facilitator can then enter the information into the Solution for the member.

  


Printed From (all of these open the PDF in the browser, with the option to download or print):

  * Members Menu | My Next Meeting Details | Print Growth Ring Worksheet
  * Facilitator | My Group Management | Print Growth Ring Worksheet
  * Admin | Growth Ring Group Management | Print Growth Ring Worksheet



  


File Format/Name: PDF; as uploaded by Symbiz

  


Fields: N/A (at this point no fields will be filled from the software)

  


Template: N/A (file provided/managed by Symbiz)

  


Handling Page Breaks: N/A (it should all fit on one page)

  


Other Notes:

  * If multiple copies are desired, the user can select the number of copies to print when printing.
