8.2.10. Memberships by Status Report

  


Requirements

Overview: This is a two-pane report of membership statuses and Member-type Contacts with filtering options.

  


Accessed from: Admin | Membership Management | View Memberships by Status

  


Title: Memberships by Status

  


Left Pane (displays the list of membership statuses):

Columns: 

  * Membership Status
  * Number of Members (displays the number of Members with that status)



  


Grouped by: N/A

  


Sorted by: standard sort order for the list

  


Right Pane (displays Member-type Contacts with the selected statuses):

Columns:

  * Member Name (link to Contact record)



  


Grouped by: Status (standard sort order for the list)

  


Sorted by: Members (alphabetically)

  


Filters: 

  * N/A



  


Buttons:

  * N/A



  


Menu Visibility: Regional Reps and Super Admins (by virtue of the Admin Menu being restricted)

  


Other Notes: N/A
