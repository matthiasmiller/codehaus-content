11\. Prompt Screens

  


Requirements

Overview: The Solution uses the following prompt screens when certain links or buttons are clicked:

  


1\. "Print Meeting Goals" prompt screen:

Overview: This prompt screen appears when a "Print Meeting Goals Worksheet" link is clicked in various places in the software.

  


Fields:

  * Member (required; drop list of the current user and all their downlines; defaults to the current user)
  * Include Last Meeting's Goals (checkbox; defaults to unchecked)
  * Continue (button; opens the "Meeting Goals Worksheet" printout with the fields filled for the selected Member)
