5.10.3.3. Notification: Symbuzz Extra (Weekly)

Name: Symbuzz Extra (Weekly) 

Overview: This is a user notification that shares Symbuzz Extra posts from a selected date range (defaulting to the default date range specified on Symbuzz Extra Posts Report -- 7 days). If Format = Email/Fax, the whole newsletter is included in the email body; if Format = In-App Alert, it will open the PDF in the user's browser. Note that the PDF could be attached to the email in the future if desired. This notification is automatically sent out every Tuesday morning at 7:00am Eastern Time. 

Record Type: Growth Ring Meeting 

Date: Every Tuesday at 7:00am ET

Trigger / Scheduled Task Details:

  * In-App Alert: Report alert that runs every Tuesday at 7:00 am ET.
  * Email: Scheduled task that runs every Tuesday at 7:00 am ET.



Scheduled Task: "Symbuzz Extra (Weekly)"

Format: Member's Choice

Send To: All Members who have the notification enabled

Subject / Message:

Subject: Symbuzz Extra - <current date in mm/dd/yyyy format>

Message: 

Good morning, <Member name>! 

  


The current edition of Symbuzz Extra is here!

  


Happy reading!

  


If Format = Email/Fax: <includes the full Symbuzz Extra for the current week, formatted like the Symbuzz Extra Printout> 

  


Symbiz Network

<Symbiz logo>

  


If Format = In-App Alert: <opens the Symbuzz Extra Printout PDF in the browser>
