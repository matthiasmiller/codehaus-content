9.2.1. Detail Screen: Trucks

Create a new Truck lookup record with:

  


  * Name (use Name + Record Number of record name)
  * Active (checkbox)
  * District (List of districts)
  * Primary Driver (List)
  * Secondary Driver (List)
  * Maintenance (document storage; defer for now until we get S3)
  * Add fields for each of the following, with a date and a reminder note. Add a warning on save if any of these are blank:
    * IRP Renewals
    * Annual Permits
    * Inspection Reminders
    * Consortium Renewals
    * Registration and Insurance
  * Notes
