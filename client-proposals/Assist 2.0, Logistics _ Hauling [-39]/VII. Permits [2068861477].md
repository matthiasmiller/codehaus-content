7\. Permits

  


Requirements

On the Hauler contact section, add:

  * Permit (RG)
    * Description (text field)
    * Permit (memo for attachments)
    * Active (checkbox; default to True)
    * Expiration Date (read-only and blank if not Active)



  


Build a report of all permits that:

  * Group by Active/Inactive
  * Sort by Expiration Date, then by hauler
  * Columns
    * Hauler
    * Expiration Date
    * Description
