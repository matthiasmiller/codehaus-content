4.1. Assembly Methods

This is a user-editable simple list used to track assembly methods. 

  


Example list items may include: 

  * Viking 504 A
  * Viking 504 B
  * etc



  


This list is universal for all users, and all users can add to it via the "+" option on the drop list on the Item record. 

  


The initial items for this list are to be set up during migration.
