3.5. System Switches

The following System Switch(es) should be set as following (listed in alphabetical order): 

  


Standard: 

  


Custom: 

  


  


  * Label: __(should be sentence case)  
    * Code: __
    * Field Type: __
    * Description: __
    * Value / set to: __
    * Other Notes: 
      * This is a less-common switch??
      * Editable or hard-coded?? 
      * Note as N/A if none



  


**Example**: 

  * Label: Allow multiple organizations
    * Code: "Config_AllowMultipleOrganizations"
    * Field Type: Drop list of "Yes" / "No" 
    * Description: "Specifies the minimum ID number for contacts." 
    * Value / set to: "Yes" 
    * Other Notes: 
      * This is a less-common switch
      * Editable
