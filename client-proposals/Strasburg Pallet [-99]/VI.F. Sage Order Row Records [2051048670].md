6.6. Sage Order Row Records

Overview: This is an intermediate landing spot for rows from a Sage sales order.

  


Accessed via: NA

  


Sections and Fields: 

  * General section:
    * Silverloom ID (autonumber; hidden & read-only)
    * Row ID (number; 0 decimals)
    * SO Number (number; 0 decimals)
    * Order Date (date)
    * Due Date (date)
    * Order Status (text)
    * Item Code (text)
    * Item Code Desc (text)
    * Customer Name (text)
    * Quantity Ordered (number; 6 decimals)
    * Unit Price (number; 6 decimals)
    * Quantity Sold (number; 6 decimals)
    * Times Sold (number; 6 decimals)
    * SO Header Upd. Date (date)
    * SO Header Upd. Time (Hrs) (number; 6 decimals)
    * Inv. Item Upd. Date (date)
    * Inv. Item Upd. Time (Hrs) (number; 6 decimals)



  


Modification History: This contains the following standard features for modification tracking: 

  * Created: (name, date, and time stamp) 
  * Last Modified: (name, date, and time stamp)
  * Modification History (link to open the report for the record)
