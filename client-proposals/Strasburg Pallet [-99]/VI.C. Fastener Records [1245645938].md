6.3. Fastener Records

Overview: This is used to track fasteners. This is not currently used for cut lists.

  


Accessed via: Advanced | Configuration | Configure Fasteners

  


Sections and Fields: 

  


  * General section:
    * Stored Name (string; hidden & read-only)
    * Name (text)
    * Type (drop list of FastenerTypes; required)
    * Active (checkbox)



  


Modification History: This contains the following standard features for modification tracking: 

  * Created: (name, date, and time stamp) 
  * Last Modified: (name, date, and time stamp)
  * Modification History (link to open the report for the record)
