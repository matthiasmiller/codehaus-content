6.11.2.2. Shop Schedule Report

This is the Master Schedule Report, filtered to show only jobs that are to be done at the shop.

  


This is accessed from: Good's Ag Repair | Scheduling | Shop Schedule 

  


The following filter(s) are to be set:

  * Location Category: "Shop"



  


Other Notes: 

  * Note that there might be gaps in the schedule since this is showing only Shop jobs (and there might be non-Shop jobs that are scheduled in between or before the jobs displayed on this version of the report).
