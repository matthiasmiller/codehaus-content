2.5. Open Work Orders Report

This is a report of all open Incidents.

  


This is accessed from Good's Ag | Scheduling | Open Work Orders

  


Title: My Incidents (incorrect title)

  


Columns: 

  * Origination Date
  * Assignee
  * Title (link to record)
  * Type
  * Contact
  * Priority
  * Due Date
  * Status
  * Workflow
  * Stage
  * ID



  


Grouped by: Incident Type

  


Sorted by: Type

  


Filters: 

  * Search (plain text field)
  * Assignee (drop list of Employees)
  * Type (drop list of Incident Types; defaults to Work Order)
  * Tag(s) (standard Tags filer option)
  * Workflow (drop list of all Workflows)
  * Originated On or After (date picker)
  * Unread Only (checkbox; defaults to unchecked)
  * Open / Closed (drop list of Open, Closed, All; defaults to Open)
  * Group By (drop list of Status, Priority, Assignee, Type, Workflow, Due Date, Modified Date, Closed Date; defaults to Type)



  


Buttons: 

  * N/A



  


Menu Visibility: 

  * All Users



  


Other Notes:

  * N/A
