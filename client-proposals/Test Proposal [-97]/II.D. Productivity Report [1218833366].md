2.4. Productivity Report

This is a report of all closed Incidents that have been Billed.

  


This is accessed from Good's Ag | Incidents | Productivity Report. 

  


Title: Productivity Report

  


Columns: 

  * Title (link to record)
  * Closing Date
  * Estimated Hours
  * Actual Hours
  * Primary Mechanic Hours
  * ID



  


Grouped by: Employee (aplhabetically)

  


Sorted by: Closed Date (nearest at the top)

  


Filters: 

  * Start Date (date picker; blank = all)
  * End Date (date picker; blank = all)
  * Primary Mechanic (drop list of all Contacts; blank = all)



  


Buttons: 

  * N/A



  


Menu Visibility: 

  * All Users
