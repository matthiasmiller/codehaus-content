3.1. Time Card

Time Card Screen

  


For Section:Â 

  * Employee
  * Week Starting (date picker; defaults to current week starting date)
  * View All Time Cards (button; opens new tab with all time cards)



  


Hours Section

  * List of days of the week
  * Hours worked each day (automatically updated every time the page is refreshed)



  


Summary Section

  * Worked (total hours worked that week; automatic)
  * Paid Time Off (automatic)
  * Effective (combination of Worked and Paid Time Off; automatic)
  * Overtime (automatic)
  * Mileage $ (automatic)



  


Time Worked Section

  * Start / Stop (button; switches between names when pressed)
  * Switch (button; creates new row in the report)
  * Clocked In Category (droplist of all jobs tracked)
  * Report with the following columns:
    * Day (droplist of weekdays; defaults to current day)
    * Start (editable number field; defaults to current time when that row was started)
    * End (editable number field; defaults to current time when that row was ended)
    * Duration (automatic and read-only)
    * Break (automatic and read-only)
    * ID (editable field; defaults blank)
    * View (link to corresponding ID)
    * Notes (single line plain text field)
    * Reference (single line plain text field)



  


Notes Section

  * Formatted memo field



  


Paid Time Off Section

  * Report with the following columns:
    * Hrs
    * Mins
    * Day
    * Reason
    * Comment



  


Mileage Section

  * Day
  * Date
  * Miles
  * Rate (Cents)
  * Total ($)
  * Notes



  


Footer Section

  * Modification History
    * Link to open the modification history report
