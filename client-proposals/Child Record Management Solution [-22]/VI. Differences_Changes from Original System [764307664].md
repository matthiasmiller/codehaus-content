6\. Differences/Changes from Original System

  * The prior system used calendars for entry. However, we will NOT be using calendars in the new system.
  * Passport has been renamed to Welcome Baby
  * ECMH Specialist has been renamed to C2K Specialist
  * The programs for ITQA and SRQIP go away
  * The Appointments fields are no longer needed.
  * We need fields but do not need reporting on health and development.
  * Last Seen Dates (vision and hearing) are no longer needed.
  * Home Visitor records are no longer needed. Convert the fields to text.
  * All fields on the Evaluations tab will be removed.
  * We will not be tracking L/R for Tympanometry.
  * The Pending outcome will be removed for screening referrals.
