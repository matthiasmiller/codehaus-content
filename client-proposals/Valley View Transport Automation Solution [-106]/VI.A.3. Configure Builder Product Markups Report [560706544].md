6.1.3. Configure Builder Product Markups Report

  


Requirements

Overview: This is a custom report of Builder Product Markups.

  


Accessed from: Advanced | Configuration | Configure Builder Product Markups

  


Title: Builder Product Markups

  


Columns: 

  * Name (link to record)
  * Markup %



  


Grouped by: Active / Inactive (active at the top)

  


Sorted by: Name (alphabetically)

  


Filters: 

  * N/A



  


Buttons: 

  * New Builder Product Markup (opens blank new record)



  


Menu Visibility: 

  * All users



  


Other Notes:

  * N/A
