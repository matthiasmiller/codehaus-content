6.5. QR Codes Report

  


Requirements

Overview: This is a custom report of QR Codes.

  


Accessed from: Main Menu | Orders | View QR Codes

  


Title: QR Codes

  


Columns: 

  * Internal ID (link to record)
  * Order #
  * Item
  * Qty
  * Notes
  * Warehouse Zone
  * Warehouse Block



  


Grouped by: Active / Inactive

  


Sorted by: Internal ID (numerically)

  


Filters: 

  * Internal ID (drop list of all Internal IDs)
  * Order # (drop list of all Order #'s)
  * Item (drop list of all Items)
  * Warehouse Zone (drop list of all Warehouse Zones)



  


Buttons: 

  * New QR Code (opens blank new record)
  * Print Selected Code (when clicked, prints the QR Code for the selected row)
