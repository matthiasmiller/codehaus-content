6.4. Orders Report

  


Requirements

Overview: This is a custom report of Orders.

  


Accessed from: Main Menu | Orders | View Orders

  


Title: Orders

  


Columns: 

  * Order # (link to record)
  * Builder
  * Customer
  * PO Number
  * Rate %
  * Combined Rate %



  


Grouped by: Status

  


Sorted by: Order Date (newest at the top)

  


Filters: 

  * Order # (drop list of all Order #'s)
  * Builder (drop list of all Builders)
  * Customer (drop list of all Customers)
  * PO Number (drop list of all PO Numbers)



  


Buttons: 

  * New Order (opens blank new record)
