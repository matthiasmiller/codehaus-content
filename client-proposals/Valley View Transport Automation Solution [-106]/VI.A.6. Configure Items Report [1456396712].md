6.1.6. Configure Items Report

  


Requirements

Overview: This is a custom report of Items.

  


Accessed from: Advanced | Configuration | Configure Items

  


Title: Items

  


Columns: 

  * Name (link to record)
  * Default Cubes



  


Grouped by: Active / Inactive (active at the top)

  


Sorted by: Name (alphabetically)

  


Filters: 

  * N/A



  


Buttons: 

  * New Item (opens blank new record)



  


Menu Visibility: 

  * All users



  


Other Notes:

  * N/A
