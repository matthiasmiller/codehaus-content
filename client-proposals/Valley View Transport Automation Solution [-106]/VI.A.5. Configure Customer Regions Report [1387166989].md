6.1.5. Configure Customer Regions Report

  


Requirements

Overview: This is a custom report of Customer Regions.

  


Accessed from: Advanced | Configuration | Configure Customer Regions

  


Title: Customer Regions

  


Columns: 

  * Name (link to record)
  * Route #
  * Description
  * Base Rate %
  * Minimum Charge



  


Grouped by: Active / Inactive (active at the top)

  


Sorted by: Name (alphabetically)

  


Filters: 

  * N/A



  


Buttons: 

  * New Customer Region (opens blank new record)



  


Menu Visibility: 

  * All users



  


Other Notes:

  * N/A
