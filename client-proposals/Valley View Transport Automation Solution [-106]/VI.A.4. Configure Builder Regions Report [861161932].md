6.1.4. Configure Builder Regions Report

  


Requirements

Overview: This is a custom report of Builder Regions.

  


Accessed from: Advanced | Configuration | Configure Builder Regions

  


Title: Builder Regions

  


Columns: 

  * Name (link to record)
  * Standard Markup %



  


Grouped by: Active / Inactive (active at the top)

  


Sorted by: Name (alphabetically)

  


Filters: 

  * N/A



  


Buttons: 

  * New Builder Region (opens blank new record)



  


Menu Visibility: 

  * All users



  


Other Notes:

  * N/A
