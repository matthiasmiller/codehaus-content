6.1.1. Configure Contact Types Report

  


Requirements

Overview: This is a standard Silverloom report of the Contact Types in the Solution. Note that any customizations are noted as such in the spec.

  


Accessed from: Advanced | Configuration | Configure Contact Types 

  


Title: Contact Types

  


Columns: 

  * Type (link to record)



  


Grouped by: Active / Inactive (active at the top)

  


Sorted by: Type name

  


Filters: 

  * Show inactive contact types (checkbox; defaulting to not checked; standard filter)



  


Buttons: 

  * New Contact Type (opens a new Contact Type record) 



  


Menu Visibility: All users

  


Other Notes:

  * N/A
