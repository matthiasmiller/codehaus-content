18.0.2.5. Auto-Create Member Applications

This is a report of new member applications, based upon member applications for the previous year that have Renew checked.

  


Accessed from: Faith Builders SPE | Member Applications | Auto-Create Member Applications

  


Title: Auto-Create Member Applications (Joinder Agreement)

  


Columns:

  * Member
  * Application Type
  * Year #
  * Total Amount
  * Accepted by SPE
  * Payment Due Date
  * SPE Application
  * Prior Year App Internal ID
  * Prior Year Application



  


Sorted by:

  * Member



  


Buttons:

  * Create New Member Applications (creates a member application for each row in the report)
