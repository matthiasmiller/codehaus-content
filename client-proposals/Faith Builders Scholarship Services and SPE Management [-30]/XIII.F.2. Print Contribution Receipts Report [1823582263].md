13.6.2. Print Contribution Receipts Report

  


Requirements

Overview: This is a custom report of Contributions that have been reviewed but have not had their receipt and / or envelope printed.

  


Accessed from: Faith Builders Scholarship Services | EITC Accounting | Print Contribution Receipts  

  


Title: Print Contribution Receipts

  


Preface: N/A

  


Columns: 

  * Business (link to Contribution)
  * Amount
  * Postmarked Date
  * Receipt Last Printed (displays the date that the "Print Receipts (for selected rows)" button was clicked for the selected row)
  * Envelope Last Printed (displays the date that the "Print Envelopes (for selected rows)" button was clicked for the selected row)



  


Grouped by: N/A

  


Sorted by: Contribution ID

  


Filters: 

  * N/A



  


Buttons: 

  * Print Receipts (for selected rows) (when clicked, generates the PDF in a child screen and sets the "Receipt Last Printed" date to the current date)
  * Print Envelopes (for selected rows) (when clicked, generates the PDF in a child screen and sets the "Envelope Last Printed" date to the current date)



  


Menu Visibility: 

  * All Users



  


Save Type: N/A

  


Live Updating: N/A

  


Other Notes: N/A
