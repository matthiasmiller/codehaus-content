10.2. Family Applications Needing Documentation Report Alert

Overview/Purpose: This is a report alert that displays when there is one or more Family Application records with the "Requires Documentation" date 15 or more days in the past. 

  


Details: 

  * Title: Family Applications Needing Documentation 
  * Displays the first time any user logs in on or after the "Requires Documentation" date on a Family Application record is 15 days in the past.  
  * Action: Open the "Family Applications Report", with the "Year" filter set to blank and the "Status" filter set to "Requires Documentation" 
  * User(s) to notify: All users 
  * Dismiss type: Auto-dismiss when all items are resolved
