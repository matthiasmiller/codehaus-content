18.0.2.3. School Designations

  


Requirements

This is a report of SPE Member Applications. This is an internal report, not to be shared with the schools because it includes member names that should remain anonymous.

  


Accessed from: Faith Builders SPE | Member Applications | School Designations

  


Title: School Financial Designations

  


Columns:

  * School
  * Member (visible if Show Member Detail is checked)
  * Anonymous (visible if Show Member Detail is checked)
  * Address (visible if Show Member Detail is checked)
  * City (visible if Show Member Detail is checked)
  * State (visible if Show Member Detail is checked)
  * Zip (visible if Show Member Detail is checked)
  * County (visible if Show Member Detail is checked)
  * Application Status (visible if Show Member Detail is checked)
  * Payment Status (visible if Show Member Detail is checked)
  * Deposit Date (visible if Show Member Detail is checked)
  * School Net Amount (total type = sum; visible if Show Member Detail is checked)
  * School Gross Amount (total type = sum)



  


Filters:

  * Member Application Type (blank = all) (drop-list of Completed by Member, Prefilled by SPE; default to Completed by Member)
  * SPE (blank = all) (drop-list of SPEs)
  * School (blank = all) (drop-list of active school contacts marked Eligible to Receive SPE Donations)
  * SPE Application (blank = all) (drop-list of SPE Applications)
  * Tax Year (blank = all) (default to current tax year)
  * Member Application Status (blank = all) (multi-select drop-list; default to Admitted)
  * Show Member Detail (defaults to checked)



  


Grouped by:

  * School
  * SPE Application if not blank; otherwise Admittance Status.



  


Sorted by:

  * Member



  


Other Notes:

  * The School Net Amount is 95% of the School Gross Amount.



  


Development Specification

The 95% should be a hidden, unchangeable system switch.
