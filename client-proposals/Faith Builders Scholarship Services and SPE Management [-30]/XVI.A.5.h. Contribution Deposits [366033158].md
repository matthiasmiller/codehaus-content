16.0.5.8. Contribution Deposits

Overview: This is a report of Contribution Deposits.

  


Accessed from: FBSS | EITC Accounting | Contribution Deposits

  


Title: Contribution Deposits

  


Columns: 

  * Check Date
  * Check Number
  * Business Name
  * Amount (totaled at the bottom of the report)
  * Contribution Type



  


Grouped by: Check Status

  


Sorted by: N/A

  


Filters: 

  * Check Status (multi-select drop list of all Check Statuses, defaults to "Scheduled for Deposit" being checked)



  


Buttons: 

  * Schedule Selected Checks for Deposit (prompts the user for a date, and marks all selected "Held" rows as "Scheduled for Deposit" on the given date)
  * Deposit Selected Checks (prompts the user for a date, and marks all selected "Held" rows as "Deposited" on the given date)



  


Menu Visibility: 

  * All Users



  


Other Notes:

  * N/A
