13.9.1. Family Applications Report

This is a report of Family Application record.

  


Accessed from: FaithBuilders Scholarship Services | Family | Family Applications

  


Title: Family Application

  


Columns: 

  * Family (link to record) 
  * School Year
  * Income 
  * Requires Documentation Date (displays the "Requires Documentation" date for the record; dates 15 days old or older display in red font) 



  


Grouped by:

  * If "Year" filter is blank: School Year (latest at the top) 
  * Otherwise: Status (Requires Documentation / Fully Documented, based on the "Requires Documentation" checkbox) 



  


Sorted by: 

  * If "Status" filter = blank or "Fully Documented": Family 
  * If "Status" filter = "Requires Documentation": Requires Documentation Date (oldest at the top) 



  


Filters: 

  * Year (number field; defaults to the current Default School Year from AppHosting.zone Settings) 
  * Status (drop list of blank = all / "Requires Documentation" / "Fully Documented"; defaults to blank)



  


Buttons: 

  * New Record
