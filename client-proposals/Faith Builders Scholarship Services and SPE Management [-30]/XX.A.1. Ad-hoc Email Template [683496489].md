20.1.1. Ad-hoc Email Template

Overview: The Ad-hoc Email Template gives the client the ability to create their own emails.

  


Accessed from: Advanced Menu | Configuration | Configure Communications

  


Name: Ad-hoc Email

  


Department: (drop list of Departments; defaults to blank)

  


Level: (drop list of FB Email Levels; defaults to blank)

  


Subject: (expression field; visible if a Level is selected)

  


Send To: Everyone in the "Ad-Hoc Recipients report (see corresponding section of the proposal)

  


Body: (expression field; visible if a Level is selected)

  


Attachments: (button; opens a child screen that allows the user to add an attachment; defaults to blank)
