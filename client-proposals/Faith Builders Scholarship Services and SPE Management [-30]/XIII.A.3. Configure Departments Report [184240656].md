13.1.3. Configure Departments Report

Overview: This is a report of Department records.

  


Accessed from: Advanced | Configuration | Configure Departments. 

  


Title: Configure Departments

  


Columns: 

  * Department Name (link to open record) 
  * Email From Name 
  * Email From Address



  


Grouped by: Active / Inactive 

  


Sorted by: Department Name (alphabetical) 

  


Menu Visibility: All users
