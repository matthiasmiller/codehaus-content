18.0.3.7. SPE Application Funding Summary

This is a totals only report of accepted SPE member applications

  


Accessed from: Faith Builders SPE | Financial Reports | SPE Application Funding Summary

  


Title: SPE Application Funding Summary

  


Columns:

  * Scholarship Organization
  * Total Accepted by SPE (total type = sum)
  * Total Admitted (total type = sum)
  * Total Received (total type = sum)
  * Total Disbursed (total type = sum)
  * Ready to Disburse (total type = sum)
  * Not Yet Received (total type = sum)
  * SO Disbursement Due Date
  * Receipt Submitted Due Date
  * State Acknowledgment Letter Date



  


Grouped by:

  * SPE Application
  * Scholarship Organization



  


Sorted by:

  * Scholarship Organization



  


Filters:

  * SPE (required; drop-list of SPEs)
  * Tax Year (required; defaults to the current tax year)
  * Include Funded SPE Applications (checkbox)
