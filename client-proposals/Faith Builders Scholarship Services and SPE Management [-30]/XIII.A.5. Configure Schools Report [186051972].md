13.1.5. Configure Schools Report

  


Requirements

Overview: This is a report showing all schools, with options to edit existing ones or to create new ones.

  


Accessed from: Advanced | Configuration | Schools

  


Title: Schools

  


Columns:

  * School
  * Primary Contact Name
  * Primary Contact Email
  * County
  * Default Scholarship Organization
  * Financial Need
  * Targeting Level



  


Filters:

  * County (blank = all) (drop-list of PA counties)
  * Default Scholarship Organization (blank = all) (drop-list of active scholarship organization type contacts
  * Financial Need (blank = all) (drop-list of High, Medium, Low)
  * Targeting Level (blank = all) (drop-list of High, Medium, Low)
  * Only show active (default to checked)



  


Grouped by: Active/Inactive

  


Sorted by: School

  


Buttons:

  * New School
