18.0.1.3. Auto-Create SPE Applications

This is a report of new SPE applications, based upon SPE applications for the previous year.

  


Accessed from: Faith Builders SPE | SPE's | Auto-Create SPE Applications

  


Title:

  * Auto-Create SPE Applications



  


Columns:

  * SPE
  * Year #
  * Application Date
  * Application Year
  * Tax Year
  * Application Amount
  * Approval Date
  * Approval Amount



  


Buttons:

  * Create New SPE Applications (creates an SPE Application for each row in the report)
