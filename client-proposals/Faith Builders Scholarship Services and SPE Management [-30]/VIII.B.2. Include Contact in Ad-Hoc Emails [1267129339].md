8.2.2. Include Contact in Ad-Hoc Emails

The following System Switch should be set as following: 

  


  * Label: Include contact in ad-hoc emails  
    * Code: IncludeContactInAdHocEmails
    * Description: Returns True if the Contact should be included in Ad-Hoc emails. This System Switch is evaluated on a Contact record and is used as the Contact-level subset when generating Ad-Hoc email records.
    * Value / set to: False
    * Other Notes: 
      * This is an expression system switch
