6.1.1. Contact Types Report

This is a standard Silverloom report of the Contact Types in the Solution. By default it would only show active items.

  


Accessed from: Advanced | Configuration | Configure Contact Types 

  


Title: Contact Types

  


Columns: 

  * Type (link to record; standard column)
  * Individual (Yes/No; standard column)
  * Organization (Yes/No; standard column)



  


Grouped by: Standard: Active/Inactive (active at the top)

  


Sorted by: Type name

  


Filters: 

  * Show inactive contact types (checkbox; defaulting to not checked; standard filter)



  


Buttons: 

  * New Contact Type (standard) 



  


Menu Visibility: 

  * Employees & Support



  


Other Notes:

  * N/A
