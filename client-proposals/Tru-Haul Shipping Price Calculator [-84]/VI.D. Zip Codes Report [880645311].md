6.4. Zip Codes Report

  


Requirements

This is a report of all Zip Codes.

  


This is accessed from Tru-Haul | Zones | Zip Codes. 

  


Title: Zip Codes

  


Columns: 

  * Zip (link to record)
  * Zone
  * City
  * State
  * Latitude
  * Longitude



  


Grouped by: Active / Inactive

  


Sorted by: Zip (numerical)

  


Filters: 

  * Route (drop list of all Routes; filters down as you type; defaults to blank = all)
  * Zone (drop list of all Zones; filters down as you type; defaults to blank = all)
  * Zip (drop list of all Zip Codes; filters based on the Zone selection; filters down as you type; defaults to blank = all)
  * State (drop list of State abbreviations; filters down as you type; defaults to blank = all)



  


Buttons: 

  * New Zip Code (opens blank new Zip Code record)



  


Menu Visibility: 

  * Employees & Support



  


Other Notes: 

  * N/A
