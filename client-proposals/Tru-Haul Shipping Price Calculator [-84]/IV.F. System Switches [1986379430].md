4.6. System Switches

  


Requirements

The following System Switch(es) should be set as following (listed in alphabetical order): 

  


  * Name: Show executive notes 
    * Code: ConfigContacts_ShowExecutiveNotes
    * Description: Specifies if the Executive Notes section and memo should be visible.
    * Set to: "No" / not checked



  


  * Name: Show Configuration menu in Advanced section 
    * Code: ShowAdvancedConfigMenu
    * Description: If enabled, Configuration will appear in the Advanced menu.
    * Set to: "Yes"
