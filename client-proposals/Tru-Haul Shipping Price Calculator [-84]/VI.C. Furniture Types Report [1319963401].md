6.3. Furniture Types Report

  


Requirements

This is a report of all Furniture Types.

  


This is accessed from Tru-Haul | Pricing | Furniture Types. 

  


Title: Furniture Types

  


Columns: 

  * Name (link to record)
  * Sort Order
  * Pricing Ratio



  


Grouped by: Active / Inactive

  


Sorted by: 

  * First by Sort Order (smallest number at the top, blanks on bottom) 
  * Second by Name



  


Filters: 

  * N/A



  


Buttons: 

  * New Furniture Type (opens blank new Furniture Type record)



  


Menu Visibility: 

  * Employees & Support



  


Other Notes: 

  * N/A
