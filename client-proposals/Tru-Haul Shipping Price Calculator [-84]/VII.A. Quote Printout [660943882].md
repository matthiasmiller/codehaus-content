7.1. Quote Printout

  


Requirements

Purpose/Overview: This is a custom printout that is used to provide details about a quote generated by the Tru-Haul calculator. This printout uses the Tru-Haul logo at the top of the page.

  


Printed From: 

  * "Print this Quote" link on the Calculator record 
  * Included as an attachment in the Quote Email (sent from the Calculator record) 



  


File Format: PDF

  


File Name: "Tru-Haul Delivery Quote (<current date in the m-d-yyyy format>)"

  


Fields to be Filled: 

  * <Address, City, State Zip> (from Silverloom Settings; displayed directly below the Tru-Haul logo)
  * <Email> | <Phone> (from Silverloom Settings; displayed directly below the "<Address, City, State Zip>" line)
  * <Business Hours> (from Silverloom Settings; displayed directly below the "<Email> | <Phone>" line)



  


  * Quote Provided For: <Dealer name> 
  * Quote Date: <current date in the m/d/yyyy format>
  * Delivery Destination: <City>, <ST> <Zip> 
  * Route Notes: <Route Notes> (hidden if blank) 



  


  * <Furniture table, same as the Furniture embedded spreadsheet on the Calculator; rows visible if they have a "Value" set on the Calculator:
    * Furniture Type 
    * Value 
    * Curbside Dropoff 
    * White Glove Delivery >
  * Curbside Dropoff*: $<total> (asterisk visibility same as on the Calculator)
  * White Glove Delivery: $<total> 
  * <*The curbside dropoff total includes the white glove delivery price for furniture not eligible for curbside dropoff."> (message visibility same as on the Calculator) 



  


Handling Page Breaks: N/A (at this point it should all fit on one page) 

  


Other Notes: 

  * Includes the default "Powered by Silverloom" footnote at the bottom of the page.
