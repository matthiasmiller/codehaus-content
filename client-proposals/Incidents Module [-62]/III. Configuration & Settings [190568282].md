      * ZBC
      * ZTK
      * ZCH
      * ZGA
      * AppHosting Time Cards Incidents Wiki



  


  * Config_DefaultIncidentType
    * Specifies the default incident type.
    * Label: Default incident type
    * Added in [[PC0119295]]
    * Used in ZGA system



  


  * Config_IncidentEnableStageHints
    * Enable stage hints
    * Label: Enable stage hints
    * Added in [[PC0119295]]
    * Used in ZGA system



  


  * Config_IncidentSupportContacts
    * Enable contact/incident link.
    * Label: Incident Support Contacts
    * Added in [[PC0098451]]
    * Used in ZTK system

    * Label: Incidents screen show full hierarchy
    * Added in [[PC0144277]]
    * Used in ZGA system



  


  * MaxIncidentIdNum
    * Specifies the maximum incident ID number
    * Label: Maximum incident ID number
    * Added in [[PC0100575]]



  


  * ConfigTimeCard_showIncidentClockIn
    * Show click in/out on incidents
    * Label: Show click in/out on incidents
    * Addedin in [[PC0129220]]



  


  * ConfigTimeCard_ShowClockedInCategory
    * Show clocked in category on time cards
    * Label: Show clocked in category on time cards
    * Added in [[PC0129220]]



  


  


  * ShowIncidentConfigMenu
    * Show incident menu items in configuration
    * Label: Show incident config menu
    * Added in [[PC0119891]]
    * Used in ZBC system.



  


  * ShowIncidentUnreadMarks
    * Incident report unread mark column is visible if the system switch is true.
    * Label: Show Incident Unread Marks
    * Added in [[PC0072864]]
    * Used in following systems:
      * ZBC in Std Incidents (helper).r20



  


  * Config_IncidentShowOrganization
    * Show organization list
    * Label: Show organization
    * Added in [[PC0143969]]
    * Used in ZTK system



  


  


  * Config_ShowTimeTrackingInPipeline
    * Show time tracking on pipeline report
    * Label: Show time tracking on pipeline report
    * Added in [[PC0132846]]
    * Used in ZTK System



  


  * ConfigTimeCard_EnableCategories
    * Enable time card categories
    * Label: Enable Time Card Categories
    * Added in [[PC0049684]]
    * Used in ZCH system, AppHosting Time Cards.txt
