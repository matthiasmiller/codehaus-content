2.4. Course Play

The course would play from start to finish. It would resume from the last place the caller left off.

  


The audio could be controled using the following guide:

  


1Previous Module| 2Decrease Volume| 2Next Module  
---|---|---  
4Rewind 30 seconds| 5Pause / Resume| 6Fast forward 30 seconds  
7Slow Down| 8Increase Volume| 9Speed Up  
*| 0Help Desk| #Account / Course Directory  
  
  


When the course is completed, it says:

You have completed "Risk Free Delegation".

If the directory only has one course, it says, "Goodbye" and hangs up.
