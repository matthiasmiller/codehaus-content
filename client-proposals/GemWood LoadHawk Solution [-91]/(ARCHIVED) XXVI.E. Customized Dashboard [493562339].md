26.5. Customized Dashboard

Sean Miller 04/30/2025: Moved to [[[IN11443](https://zch.apphosting.zone/Detail/Edit/2?RecordType=Incidents&NumberID=-11445&)]] - ZGW - Customized Dashboard

  


  


An optional dashboard can be provided to show a graphic interface of key metrics and calls to action. Preliminary / example design is included below for various features that could be included on this dashboard. If desired, this feature could be handled as a separate or future item. 

  


Possible Charts / Graphs:

  * Monthly Sales: 
    * Vertical bar graph or line graph
    * Displays total Delivery Tickets and/or total Buyer Payment $ by month



  


  * Weekly Sales: 



  


  * Sales by Buyer: 
    * Pie graph 
    * Shows a "slice" for each Buyer, based on Ticket Value $
    * Different color for each Buyer
    * Could display results month-to-date, year-to-date, or for some other date range



  


  * Sales by Member: 
    * Same idea as "Sales by Buyer", but for Members



  


  * Top 10 Buyers: 
    * Vertical or horizontal bar graph 
    * Shows the top 10 Buyers, based on total Ticket Value $ per Buyer 
    * Each bar represents the total Ticket Value $
    * Different color for each Buyer
    * Could display results month-to-date, year-to-date, or for some other date range



  


  * Top 10 Members:
    * Same idea as "Top 10 Buyers", but for Members



  


  * Sales by Species: 
    * Vertical or horizontal bar graph
    * Displays total board footage and/or $ value per Species (based on Member Grades)
    * Different color for each Buyer
    * Could display results month-to-date, year-to-date, or for some other date range



  


  * Sales by Grade: 
    * Same idea as "Sales by Species", but for Lumber Grades



  


  


Possible Calls to Action:

  * These could be based on the notifications specced / included in Phase 1, displayed on the dashboard in a "Calls to Action" section, opening corresponding reports or records.
