8.5.6. Invoices Awaiting Approval Report

  


Requirements

Overview: This is the All Delivery Tickets Report, filtered to display items that are awaiting approval of the Buyer Invoice by the Salesperson.

  


Accessed from:

  * Main Menu | Buyers | Invoices Awaiting Approval (opens the report directly, bypassing the filter screen)
  * "Invoices Awaiting Approval" notification / report alert



  


The following filter(s) are set differently from the main report:

  * Status: only "Awaiting Salesperson Approval" selected
