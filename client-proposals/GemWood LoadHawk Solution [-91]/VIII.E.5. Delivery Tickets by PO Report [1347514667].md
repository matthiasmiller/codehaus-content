8.5.5. Delivery Tickets by PO Report

  


Requirements

Overview: This is the All Delivery Tickets Report, filtered to display items for one Purchase Order.

  


Accessed from:

  * Bottom report on the Purchase Order record
  * Displayed as the right pane of the All Purchase Orders Report



  


The following filter(s) are set differently from the main report:

  * "Buyer's PO #": Defaults to the record from which the report is opened
  * "Status": Defaults to all items
  * "Group By": Defaults to "GemWood PO Status"
