6.2. GemWood Divisions List

  


Requirements

This is a custom non-editable simple list used to track divisions for categorizing GemWood-related Contacts, and corresponding Internal Fee Payouts. It includes the following items, listed in alphabetical order: 

  


  * Office & Administrative 
  * Quality Control
  * Salesperson
  * Training & R&D



  


This list is universal for all users.

  


Note that this sequence is static, so it should always be displayed in this sequence throughout the Solution.

  


This list will be set up by CCI as part of the main development. If changes need to be made to this list, it will require some coding work.
