25.6.1. Member Equity Report

  


Requirements

Overview: This is a custom report of equity accumulated by Members, based on Delivery Ticket records. Note that this does not track draws that the Members may make on their accumulated equity.

  


Accessed from: Main Menu | Financials | Accumulated Equity

  


Title: Accumulated Equity

  


Columns: 

  * Ticket # (link to open the record)
  * Ticket Date
  * Total Buyer Payment $ (total row shows sum) 
  * Equity %
  * Equity $ (total row displays the sum)



  


Grouped by: Member (alphabetically)

  


Sorted by: Ticket # (alphabetically)

  


Filters: 

  * Member (drop list of Member-type Contacts that have a non-zero value set in "Member's Equity %"; defaults to blank = all)
  * Ticket Date Start (date; looks at the Ticket Date; defaults to blank = all)
  * End (date; looks at the Ticket Date; defaults to blank = all)



  


Buttons: 

  * N/A



  


Menu Visibility: 

  * All Users 



  


Other Notes:

  * N/A
