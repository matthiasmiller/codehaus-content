8.7.3. Buyer Payments Lacking Grade Reports Report

  


Requirements

Overview: This is a custom report of Buyer Payment records that are linked to Delivery Tickets with Status = "Awaiting Buyer Payment" or "Awaiting Grade Report / Settlement" that are lacking Buyer Grade Report information (attachments in the "Buyer Grade Report Files" embedded spreadsheet and/or "Buyer Grade $" amount). 

  


Accessed from: Main Menu | Payments | Buyer Pmts Lacking Grade Reports

  * Note that if the "Buyer Payments Lacking Grade Reports" report alert notification (optional add-on) is added in the future, this report would also be opened from there.



  


Title: "Buyer Payments Lacking Grade Reports" 

  


Columns: 

  * Payment Description (link to open the record)
  * Payment Date 
  * Payment Total $
  * Delivery Ticket (link to open the record)
  * Attached Files (displays the number of attached files in the "Buyer Grade Report Files" embedded spreadsheet on the linked Delivery Ticket)
  * Buyer Grade $ 



  


Grouped by: Payment Description (alphabetical) 

  


Sorted by:

  * First by: Payment Description (alphabetical)
  * Second by: Delivery Ticket (alphabetical)
