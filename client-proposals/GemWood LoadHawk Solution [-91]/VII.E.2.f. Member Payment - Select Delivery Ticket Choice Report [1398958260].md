7.5.2.6. Member Payment - Select Delivery Ticket Choice Report

  * Select Delivery Ticket Choice Report (open as a "child screen"; includes the following): 
    * Displays the following: 
      * All non-canceled Delivery Tickets in the Solution and a row for "(none)"; this is the "Delivery Ticket Choice Report" report;  
      * "Refresh" button; 
      * "OK" button (links the item to the selected Payment record, or unlink the item if "(none)" is selected) 
      * "Cancel" button
    * Workflow: 
      * User opens the Choice Report. 
      * User finds the desired Ticket (can use "Ctrl + F" to search on the report). 
      * User selects the desired Ticket and clicks Continue to link. 
      * If the user wishes to unlink the Member Payment record from a Ticket, the user selects the blank line at the top of the Choice Report and clicks Continue to unlink.
