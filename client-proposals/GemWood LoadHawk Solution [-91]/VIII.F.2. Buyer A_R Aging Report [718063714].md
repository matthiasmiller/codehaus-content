8.6.2. Buyer A/R Aging Report

  


Requirements

Overview: This is the "Buyer Invoices" report, filtered to display all unpaid Buyer Invoices (i.e. all Delivery Tickets that are not marked as "Closed" or "Canceled").

  


Accessed from: Main Menu | Buyers | Buyer A/R Aging

  


The following filter(s) are set differently from the main report:

  * A/R Aging: checkbox checked
