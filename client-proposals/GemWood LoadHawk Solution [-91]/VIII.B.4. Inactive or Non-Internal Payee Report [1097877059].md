8.2.4. Inactive or Non-Internal Payee Report

  


Requirements

Overview: This is a report of Contact records that are included on the "Internal Fee Payouts Configuration" embedded spreadsheet on the Silverloom Settings page, but are inactive and/or have a "Contact Type" other than "Internal".

  


Accessed from: The "Inactive or Non-Internal Payee" report alert (see corresponding spec; opens the report directly, bypassing the filter screen)

  


Title: "Inactive or Non-Internal Payee"

  


Columns: 

  * Payee (link to record) 
  * Reason (displays the following: 
    * if Contact is inactive and not "Internal": "Payee is not active and not internal type in Silverloom Settings."; 
    * if Contact is only inactive: "Payee is not active and not internal type in Silverloom Settings."; 
    * if Contact is only not "Internal": "Payee is not internal type in Silverloom Settings.") 



  


Grouped by: N/A

  


Sorted by: "Payee" (alphabetical) 

  


Filters: 

  * Last Viewed Alert ID
  * Last Dismissed Alert ID



  


Buttons: 

  * N/A



  


Menu Visibility: 

  * N/A (not included on the menu) 



  


Other Notes:

  * N/A
