5.5. System Switches

  


Requirements

The Solution contains the following custom System Switches:

  


  * Label: "Ticket GemWood PO required date" 
    * Code: "TktGWPORequiredDate"
    * Field Type: Date 
    * Description: "Specifies the date from which the GemWood PO ID is required for Delivery Tickets."
    * Value / set to: 10/04/2025  
    * Other Notes: 
      * Editable 
      * Note that this is to be set to the date from which a GemWood Purchase Order is required for Delivery Tickets, likely the deployment date for the GemWood PO feature.
