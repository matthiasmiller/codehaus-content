6.3. Lumber Thicknesses List

  


Requirements

This is a custom unsorted, editable simple list used to track lumber thicknesses. 

  


This list is universal for all users, and all users can add to it via the "+" option on the corresponding drop list(s) in the Solution. 

  


The initial items for this list are to be set up by CCI as part of the development ("shipped"), and additional items may be added by GemWood users, with assistance from CCI as needed.

  


The initial items are as follows: 

  * 4/4
  * 5/4
  * 6/4
  * 8/4
  * 10/4
  * 12/4 
  * N/A
