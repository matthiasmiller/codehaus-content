8.5.4. Delivery Tickets by Buyer Report

  


Requirements

Overview: This is the All Delivery Tickets Report, filtered to display items for one Buyer.

  


Accessed from:

  * "Delivery Tickets by Buyer" link on the Contact record



  


The following filter(s) are set differently from the main report:

  * "Buyer": Set to the Buyer-type Contact from which the report is opened
