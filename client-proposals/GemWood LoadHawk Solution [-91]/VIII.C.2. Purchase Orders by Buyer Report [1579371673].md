8.3.2. Purchase Orders by Buyer Report

  


Requirements

Overview: This is the All Purchase Orders Report, filtered to display items for a single Buyer.

  


Accessed from:

  * "Purchase Orders by Buyer" link on the Buyer-type Contact record



  


The following filter(s) are set differently from the main report:

  * "Buyer": Defaults to the record from which the report is opened
