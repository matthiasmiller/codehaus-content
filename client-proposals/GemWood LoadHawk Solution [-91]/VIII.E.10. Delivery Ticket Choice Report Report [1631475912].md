8.5.10. Delivery Ticket Choice Report Report

  


Requirements

Overview: This is a custom report of all non-"Canceled" Delivery Ticket records in the Solution, used for the Select Delivery Ticket Choice Report on the Member Payment record.

  


Accessed from:

  * The Select Delivery Ticket Choice Report child screen on the Member Payment record



  


Title: "Delivery Ticket Choice Report"

  


Columns: 

  * Member
  * Ticket # (link to open record)
  * Ticket Date
  * Buyer
  * Ticket Value $
  * Status (displays the "Delivery Ticket Status")



  


Grouped by: N/A  

  


Sorted by: Ticket # (alphabetically)
