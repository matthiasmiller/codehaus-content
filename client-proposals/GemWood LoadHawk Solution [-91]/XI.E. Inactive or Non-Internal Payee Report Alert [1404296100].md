11.5. Inactive or Non-Internal Payee Report Alert

Overview/Purpose: This is an internal report alert to notify users that one or more "Payees" on the "Internal Fee Payouts Configuration" embedded spreadsheet in Silverloom Settings is no longer an active or "Internal"-type Contact.

  


Details: 

  * Title: "Inactive or Non-Internal Payee Report Alert"
  * Displays within 20 minutes of: One or both of the following being true for a Contact record for a Payee on the "Internal Fee Payouts Configuration" embedded spreadsheet in Silverloom Settings:  
    * is made inactive or 
    * its "Contact Type" is set to something other than "Internal"
  * Action: Opens the "Inactive or Non-Internal Payee" report - see the corresponding spec 
  * User(s) to notify: 
    * Users with the "Manage Financials" Permission
  * Dismiss type: Auto-dismiss when all items are resolved
