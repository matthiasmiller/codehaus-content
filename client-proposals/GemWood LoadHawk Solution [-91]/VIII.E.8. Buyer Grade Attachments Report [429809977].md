8.5.8. Buyer Grade Attachments Report

  


Requirements

Overview: This is a simple report of Buyer Grade Attachments files for a selected Delivery Ticket record.

  


Accessed from:

  * "View <X> Files" link on the Buyer Payment record

Title: Buyer Grade Attachments for Delivery Ticket <Ticket #>

  * Attachment (link; displays the file name from the embedded spreadsheet)
