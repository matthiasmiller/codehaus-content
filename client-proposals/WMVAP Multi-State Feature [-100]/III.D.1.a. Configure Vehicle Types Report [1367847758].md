3.4.1.1. Configure Vehicle Types Report

  


Requirements

*Documented. Tim Reitz 08/19/2025

  


Make the following changes (full spec included here, with changes noted with blue and strike-through): 

  


Overview: This is a custom report of Vehicle Types.

  


Accessed from: Advanced | Configuration | Configure Vehicle Types

  


Title: Vehicle Types

  


Columns: 

  * Type (link to record)
  * Requires VIN
  * Requires License Plate
  * Uses Classes
  * Included Classes
  * Additional Classes ($)
  * Uses & Requires Liability
  * Liability Premium ($)
  * Uses <Alternate text for Collision - Long label>
  * <Alternate text for Collision - Short label> Entry Fee (%) (visible if Solution uses <Collision> Entry Fee)
  * <Alternate text for Collision - Short label> Premium ($) (visible if Solution uses <Collision> Flat Rate Premium)
  * <Alternate text for Collision - Short label> Premium (%) (visible if Solution uses <Collision> Percentage Premium)
  * Uses Cargo
  * Cargo Entry Fee (%) (visible if Solution uses Cargo Entry Fee)
  * Cargo Premium (%) (visible if Solution uses Cargo Percentage Premium)
  * Cargo Premium ($) (visible if Solution uses Cargo Flat Rate Premium)
  * Subtypes (comma-separated, wrapped text)



  


Grouped by: Active / Inactive

  


Sorted by: Type

  


Filters: 

  * Show inactive vehicle types (checkbox; defaults to unchecked)



  


Buttons: 

  * New Vehicle Type (opens a new record)



  


Menu Visibility: 

  * Visible to all users, but accessible for Admins only (grayed-out for non-Admins)



  


Other Notes:

  * N/A
