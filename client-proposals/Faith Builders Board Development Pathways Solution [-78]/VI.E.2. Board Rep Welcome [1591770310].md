6.5.2. Board Rep Welcome

This is a confirmation communication that is sent to the Board Representative after signing up for a new Pathway Subscription. 

  


It contains things such as: 

  * Welcome / introduction to the Pathway
  * Note that a welcome has been sent to all participants and a reminder to make sure all of them give the Board Rep confirmation
  * Note that a separate email has been sent with the receipt for the subscription purchase.
