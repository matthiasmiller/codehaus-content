3.3.1.4. Configure Incident Types Report

This is a report of Incident Types.

  


This is accessed from Advanced | Configuration | Configure Incident Types.

  


Title: Incident Types

  


Columns: 

  * Name
  * Default Workflow



  


Grouped by: N/A

  


Sorted by: Name (alphabetical)

  


Filters: 

  * N/A



  


Buttons: 

  * New Incident Type (opens new blank Incident Type record)



  


Menu Visibility: Admin only (menu option is grayed out for other users) 

  


Other Notes: N/A
