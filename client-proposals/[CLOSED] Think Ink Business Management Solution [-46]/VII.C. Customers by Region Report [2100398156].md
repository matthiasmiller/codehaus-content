7.3. Customers by Region Report

  


Requirements

This is a simple report of all the Customers in one selected Region.

  


This is accessed from:

  * Region Record (bottom report on the record detail screen)
  * Manage Regions Report (right pane)



  


Title: Customers by Region

  


Columns: 

  * Customer (name of customer; link to record)
  * Billing Address (standard 2-line format; displays the first Billing-type address on the Customer's Contact record)



  


Grouped by: N/A

  


Sorted by: Customer (alphabetical)

  


Filters: 

  * Region (required prompt upon opening the report; drop list of all Regions)



  


Buttons: 

  * N/A



  


Menu Visibility: N/A (not accessible by itself on the menus)

  


Other Notes:

  * N/A
