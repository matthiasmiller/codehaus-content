This is a report of Classic Accounting Sales Items records.

  


This would be accessed from a "Classic Accounting Sales Items" menu option in the "Classic Accounting" section on the Advanced menu.

  


Title: Classic Accounting Sales Items

  


Columns: 

  * ID
  * Type Code
  * Location
  * Name
  * Number
  * Mod Date
  * Notes
  * Order Minimum
  * Order Maximum
  * Price
  * Price Type
  * Purchase Description
  * Vendor Part Number
  * Weight
  * Vendor Name
  * Purchase Account
  * Sales Account
  * Alert Note on Purchases
  * Alert Note on Sales



  


Grouped by: N/A

  


Sorted by: 

  * ID



  


Filters: 

  * ID (blank = all)



  


Menu Visibility: Admin Only

  


Other Notes: N/A
