22.3.3. Classic Accounting Sync Results Report

  


Requirements

This is a report of Classic Accounting Sync Result records. 

  


This would be accessed from a "Classic Accounting Sync Results" menu option in the "Classic Accounting" section on the Advanced menu.

  


Title: Classic Accounting Results

  


Columns: 

  * ID (link to record)
  * Date/Time
  * Username 
  * Success 
  * Error Log (word wrap)



  


Grouped by: N/A

  


Sorted by: Reverse sort by ID

  


Filters: 

  * Show Successful Syncs (checkbox; default to cleared) 
  * Start Date (default to 30 days before today; blank = all) 
  * End Date (default to blank = all) 



  


Buttons: N/A

  


Menu Visibility: Admin Only

  


Other Notes: N/A
