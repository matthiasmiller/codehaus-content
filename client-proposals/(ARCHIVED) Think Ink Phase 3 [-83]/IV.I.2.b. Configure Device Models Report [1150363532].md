4.9.2.2. Configure Device Models Report

  


Requirements

This is a report of all Device Models in the Solution (all Sales Items with the Device Model category). While the Device Models can be managed from the main Manage Sales Items report, this is a separate menu option and custom report to make it easier for users. 

  


This is accessed from Advanced | Configuration | Configure Device Models.

  


Title: Configure Device Models

  


Columns: TODO_IDD: Tim Reitz 09/12/2023: review & update

  * Brand
  * Model 
  * Description (link to the record)
  * Print Type (Mono, Color)



  


Grouped by: Active/Inactive (active at the top; inactive at bottom, grayed out)

  


Sorted by: Name

  


Filters: N/A

  


Buttons: 

  * New Device Model (opens a new blank Sales Item record with Category defaulted to Device Model) 



  


Menu Visibility: All users can view and edit

  


Other Notes: N/A
