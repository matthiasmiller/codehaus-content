4.5.1. Brand

  


Requirements

This is a user-editable simple list used to track Brands for Device Models. 

  


Example list items may include: 

  * Brother 
  * Canon
  * HP
  * Kyocera
  * etc.



  


This list is universal for all users, and all users can add to it via the "+" option on the "Brand" drop list on the Device Model (Sales Item) record. 

  


The initial items for this list are to be set up by Think Ink, with assistance from CCI/CH as needed.
