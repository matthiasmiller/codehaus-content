4.9.3.3. Devices Needing Review Report

  


Requirements

This is a report of Devices that have the "Needs Review" checkbox checked.

  


This would be accessed from: 

  * Home | Devices | Devices Needing Review
  * Devices Needing Review alert



  


Title: Devices Needing Review

  


Columns: 

  * Device Description (link to record)
  * Customer
  * Agreement (link to record)
  * Needs Review Reason(s) (comma-separated; wrapping text)
  * Needs Review Notes (wrapping text)
  * Black Pages Used (displays the most recent Black Pages Used; show in red if the row is not invoiced and is 50% more or 50% less than the Average Black Usage) 
  * Average Black Usage (from the Device record)
  * Color Pages Used (displays the most recent Color Pages Used; show in red if the row is not invoiced and is 50% more or 50% less than the Average Color Usage) 
  * Average Color Usage (from the Device record)
  * Prepay End Date (show in red if in the past)



  


Grouped by: First by Customer, then by Agreement

  


Sorted by: Device Description

  


Filters: 

  * Sales Rep (drop list of all Employee Contacts; defaults to blank = all)
  * Customer (drop list of all Customer contacts; filters down as you type; defaults to blank = all)
  * Agreement (drop list of Agreement Names for the selected Customer; filters down as you type; defaults to blank = all)
  * Needs Review Reason(s) (multi-select drop list of Reasons; defaults to blank = all)



  


Buttons: 

  * N/A



  


Menu Visibility: All Users

  


Other Notes:

  * A User would open a Device from this report to review and edit the Device and will need to clear the Needs Review checkbox when everything is resolved.
