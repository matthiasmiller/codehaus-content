4.9.1.2. Configure Device Types Report

  


Requirements

This is a report of all Device Types.

  


This is accessed from Advanced | Configuration | Configure Device Types. 

  


Title: Device Types

  


Columns: 

  * Type (link to record)
  * Always Managed (Yes/No)



  


Grouped by: Active / Inactive

  


Sorted by: Type (alphabetical)

  


Filters: 

  * Show Inactive Types (checkbox; defaults to checked)



  


Buttons: 

  * New Type (opens new blank Device Type record)



  


Menu Visibility: 

  * Grayed-out for non-admins



  


Other Notes:

  * N/A
