22.3.2. Classic Accounting Addresses Report

  


Requirements

This is a report of Classic Accounting Address records.

  


This would be accessed from a "Classic Accounting Addresses" menu option in the "Classic Accounting" section on the Advanced menu.

  


Title: Classic Accounting Addresses

  


Columns: 

  * ID (link to record)
  * Org Name
  * Address Type
  * Street 1
  * Street 2
  * City 
  * State
  * Zip 



  


Grouped by: 

  


Sorted by: 

  * ID



  


Filters: 

  * ID (blank = all)



  


Menu Visibility: Admin Only

  


Other Notes: N/A
