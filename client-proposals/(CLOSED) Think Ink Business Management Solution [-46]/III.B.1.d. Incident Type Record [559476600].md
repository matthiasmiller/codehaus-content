3.2.1.4. Incident Type Record

The Solution will track Incident Types.

  


Sections and Fields:

  * Type Name (plain text field)
  * Default Workflow (droplist)
  * Billable by default (checkbox; defaults to unfilled)
  * Require closing reason (checkbox; defaults to unfilled)



  


Data Access:

  * Editable by Admins
  * Viewable by all other users
