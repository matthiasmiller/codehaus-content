9.4. New: GemWood Purchase Orders Report

Overview: This is a custom two-pane report of all GemWood Purchase Order records in the Solution (left pane) and the linked Delivery Ticket records (right pane).

  


Accessed from: Main Menu | Members | GemWood Purchase Orders (opens the report directly, bypassing the filter screen)

  


Left Pane: This is the "All Purchase Orders" report, with the following filter(s) are set differently from the main defaults:

  * "PO Type": set to "GemWood"



 

Right Pane: This is the "Delivery Tickets by PO" report (see corresponding spec), filtered to the most recently-selected GemWood Purchase Order in the left pane.
