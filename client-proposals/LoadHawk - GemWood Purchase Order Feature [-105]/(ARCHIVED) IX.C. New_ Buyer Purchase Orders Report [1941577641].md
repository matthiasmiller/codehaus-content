9.3. New: Buyer Purchase Orders Report

Overview: This is a custom two-pane report of all Buyer Purchase Order records in the Solution (left pane) and the linked GemWood Purchase Order records (right pane).

  


Accessed from: Main Menu | Buyers | Buyer Purchase Orders (opens the report directly, bypassing the filter screen)

Note that this is replacing the existing report accessed from this menu option. 

  


Left Pane: This is the "All Purchase Orders" report, with the following filter(s) are set differently from the main default:

  * "PO Type": set to "Buyer"



 

Right Pane: This is the "GemWood POs by Buyer PO" report (see corresponding spec), filtered to the most recently-selected Buyer Purchase Order in the left pane.
