12\. Shared Inbox

  


Requirements

Optional:Â The Shared Inbox is an optional add-on and priced separately.

  


This will allow all emails to be viewed within the SPE system.

  


Each email will show:

  * From Name
  * From Email
  * From Contact*
  * To -- embedded spreadsheet of:
    * Name
    * Email
    * Contact*
  * CC -- identical to To
  * Date/Time (stored in GMT and shown in local time)
  * Subject
  * Body



  


The Contact fields would be used to link to a specific contact. These would be defaulted to the first contact matching the email. However, they could be updated as needed.

  


All fields except the Contact fields would be read-only.

  


It will include a Search Emails that would prompt for:

  * Contact (blank = all; list of contacts)
  * Email (blank = all; text)
  * Start Date (blank = all; text)
  * End Date (blank = all; text)
  * Search (blank = all; text)



  


The columns would include:

  * Date
  * From
  * To/CC (may be truncated)
  * Subject



  


It would be reverse sorted by date.

  


The Contact and Member Application screens would show View X Emails links.
