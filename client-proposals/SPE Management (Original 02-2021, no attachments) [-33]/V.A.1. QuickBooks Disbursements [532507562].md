5.1.1. QuickBooks Disbursements

  


Requirements

This report would show all SPE disbursements. It will be used to reconcile with QuickBooks.

  


It should filter by:

  * SPE (required)
  * Payee (list of Scholarship Organizations; blank = all)
  * Check Date (start/end; blank = all; default start to January 1 of the current year)



  


It should show the following columns:

  * Check Date
  * Check Number
  * Payee
  * State Application Number
  * Amount
  * SPE Application (linking to SPE application)



  


It should be sorted by date.

  


It should include a total amount.
