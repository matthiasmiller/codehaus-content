11.1. Export from SO (FBSS)

  


Requirements

The FBSS system will have a new "Export Schools for SPE" option.

  


It will include the following columns:

  * Scholarship Organization
  * School ID
  * School Name
  * School County
  * Contact Name
  * Contact Email



  


The School ID will be the FBSS database's internal ID for the school.

  


Development Specification

The scholarship organization should be controlled by a system switch.

  


Target: 2 hours
