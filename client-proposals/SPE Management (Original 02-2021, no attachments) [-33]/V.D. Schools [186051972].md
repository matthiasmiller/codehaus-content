5.4. Schools

  


Requirements

This will be a report showing all schools, with options to edit existing ones or to create new ones.

  


It should filter by:

  * County
  * Default Scholarship Organization (blank = all)
  * Financial Need
  * Targeting Level
  * Only show active (default to checked)



  


Group by Active/Inactive schools.

  


It should show:

  * School
  * Primary Contact Name
  * Primary Contact Email
  * County
  * Default Scholarship Organization
  * Financial Need
  * Targeting Level
