7.3. Basic Communications Template

  


Requirements

All emails would use a basic format:

  


(Communication body)

  


Lucy Miller

FBEP Bookkeeper/SPE Coordinator

 

Faith Builders Educational Programs, Inc.

28527 Guys Mills Road

Guys Mills, PA 16327

(814) 789-4518 ext. 245

Fax: (814) 789-3396

  


The greet should use the contact's preferred name if specified. Otherwise, it should use the first name. If it's a business contact, it should use the primary contact First Name.

  


The signer's name would be determined by the communication's From Name and From Title.
