7.3. Invoices

  


Requirements

Orders will be automatically exported to QuickBooks invoices. The invoice line items will be grouped together on a per-drawer basis. The invoices can then be printed from QuickBooks.

  


Any orders that need to be printed from the Solution can be printed using the Order printout.
