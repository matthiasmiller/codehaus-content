8\. Add "New Claim" link on Vehicle Detail

  


Requirements

Option 1: 

On the Vehicle screen, add a "New Claim" link to create a new claim for that vehicle (defaulting the Client's name and the Vehicle information. 

  


T&M Estimate: Approximately $100

  


Option 2: 

Add a "View x Claim(s)" link that would open a report showing claims. This could either be on the Client's page (showing all claims for the client) or on the Vehicle page (showing all claims for the vehicle). 

This report would include a "New Claim" button. 

Note that this would require some additional design to determine what details should be shown in the report. 

  


T&M Estimate: $100-200
