2.1.3. Townships

TOS should store a list of townships in Pennsylvania along with corresponding information.

  


The Township will specify:

  * Township Name
  * County (list of counties; required)



  


It will have a section for Design Data, which includes:

  * Design Data; choice of:
    * County
    * Township
  * The following fields are read-only if using County data. Otherwise, they are editable:
    * Snow Load (number)
    * Wind Speed (number)
    * Frost Depth (number)
