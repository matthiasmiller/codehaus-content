7.2.11. Contact Record - Silverloom Contact Settings Section

*Done. 

  


  * Silverloom Contact Settings section (standard section; visible on Contacts that are individuals; automatically visible if the "User ID" field is visible; note that this section is utilized for contact-specific configurations):
    * User ID (standard field; visible if "Is Organization" is not checked; with the following details / behaviors: 
      * drop list of active User IDs that are not already linked to a Contact record; 
      * visible to all users; 
      * editable for users with the "Edit User Groups and Profiles" Permission; 
        * custom note: note this editability means that only Master Administrators can set up users in Silverloom; in the future, this could be changed to allow Group Admins & Reseller Admins to set up users for their Groups / Reseller organizations)
