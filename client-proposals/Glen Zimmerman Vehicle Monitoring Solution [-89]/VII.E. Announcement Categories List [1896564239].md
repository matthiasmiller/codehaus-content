7.5. Announcement Categories List

  


Requirements

*Done. 

  


Overview: This is a custom user-editable simple list used to track types of direct Announcements to Accounts. 

  


Initial list items include: 

  * End User Agreement 
  * Other Legal Notification 
  * General Information 
  * New Feature Notice
  * Best Practice Reminder 
  * Encouragement Reminder 
  * Holiday Communication 
  * Service Outage Notice
  * Payment/Subscription Notice
  * Other 



  


This list is universal for all users, and all users can add to it via the Lists interface. 

  


The initial items for this list are to be set up by CodeCrafters at deployment. Changes can be made by <Service Name>, with assistance from CodeCrafters as needed.
