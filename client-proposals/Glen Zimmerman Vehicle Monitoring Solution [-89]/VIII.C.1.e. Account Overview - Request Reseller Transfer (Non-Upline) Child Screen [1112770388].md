7.3.1.5. Account Overview - Request Reseller Transfer (Non-Upline) Child Screen

  


Requirements

*Done.

  


  * Request Reseller Transfer (Non-Upline) Child Screen (includes the following, when the "Request Reseller Transfer (Non-Upline)" button in clicked - see corresponding spec):



  


  * Close (button; closes the child screen to cancel the transfer request) 
  * New Reseller (required; drop list prompt of "Display Name" items for all active Account Resellers; defaults to the Contact for the current user (if individual Reseller) or to the linked Reseller Contact (if Reseller Rep for a Reseller organization)) 
  * Reason / Comments (required; plain text prompt; defaults to blank) 
  * Submit Transfer Request (button; when clicked with all required prompts filled, the "Request Reseller Transfer (Non-Upline)" user-initiated automatic process runs - see corresponding spec)
