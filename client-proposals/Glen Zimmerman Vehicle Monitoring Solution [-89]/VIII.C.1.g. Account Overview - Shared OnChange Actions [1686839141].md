7.3.1.7. Account Overview - Shared OnChange Actions

*Done. 

  


The following Shared OnChange Action(s) are included for this section: 

  


  * "Set Prior Reseller Approval" Shared OnChange Action:
    * Initiated:
      * When the "Submit Transfer Request" button is clicked on the "Transfer Reseller Prompt Screen" by the "Prior Reseller" (individual) or a "Reseller Rep" for the "Prior Reseller" Contact;
      * When the "Prior Reseller Approval" checkbox is checked on the "Account reseller History" non-embedded spreadsheet;
    * Actions: The following field(s) are affected: 
      * "Prior Reseller Approval User": sets to the "User ID" for the user who checked the "Prior Reseller Approval" checkbox;
      * "Prior Reseller Approval Date": sets to the current date, in the "MM/DD/YYYY" format
