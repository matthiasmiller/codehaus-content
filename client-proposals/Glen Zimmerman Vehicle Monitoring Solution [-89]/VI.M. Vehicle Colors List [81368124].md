6.13. Vehicle Colors List

*Done. 

  


Overview: This is a custom user-editable simple list used to track colors of vehicles on Vehicle records. 

  


Initial list items include: 

  * Beige
  * Black
  * Blue
  * Brown
  * Dark Gray
  * Gold
  * Green
  * Light Blue 
  * Light Gray
  * Navy Blue
  * Orange
  * Purple
  * Red
  * Silver
  * White
  * Yellow



  


This list is universal for all users, and all users can add to it via the "+" option on the drop list on the Vehicle record. 

  


The initial items for this list are to be set up at deployment by CodeCrafters. Items can be added to the list by users, with assistance from CodeCrafters as needed.
