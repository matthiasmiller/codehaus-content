9.2.2.1. All Group Admins Report

  


Requirements

*Done.

  


Overview: This is the "Standard All Contacts" report, filtered to include only Group Admins (including Master Administrators). 

  


Accessed from: 

  * Administrators | Contact Management | All Group Admins (opens the report directly, bypassing the filter screen) 



  


The following filter(s) are set differently than on the main report:

  * "Contact Role" = Set to "Master Administrator" and Group Admin".
