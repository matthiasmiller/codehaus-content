7.2. Contact Roles List

  


Requirements

*Done. 

  


  


Overview: This is a custom non-editable simple list used to track roles within the Silverloom software, and for Traccar User sync records. 

  


It includes the following items: 

  * Master Administrator (for report filters, this includes all Contacts with the "Is Master Administrator" checkbox checked)
  * Group Admin (for report filters, this includes all Contacts with the "Is Group Admin" checkbox checked)
  * Account Reseller (for report filters, this includes all Contacts with the "Is Account Reseller" checkbox checked)
  * Reseller Admin (for report filters, this includes all Contacts with the "Is Reseller Admin" checkbox checked)  
  * Reseller Rep (for report filters, this includes all Contacts with the "Is Reseller Rep" checkbox checked)
  * Primary Account Manager (for report filters, this includes all Contacts that have the "Is Primary Account Manager" checkbox checked) 
  * Account Manager (for report filters, this includes all Contacts that have the "Is Account Manager" checkbox checked) 
  * Driver (for report filters, this includes all Contacts that have the "Is Driver" checkbox checked) 



  


This list is universal for all users.

  


Note that this sequence is static, so it should always be displayed in this sequence throughout the Solution.

  


This list will be set up by CodeCrafters as part of the main development. If changes need to be made to this list, it will require some coding work.
