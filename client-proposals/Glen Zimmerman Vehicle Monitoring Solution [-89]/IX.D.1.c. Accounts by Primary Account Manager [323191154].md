9.4.1.3. Accounts by Primary Account Manager

*Done. 

  


Overview: This is the "Main Accounts Report", filtered to a single Primary Account Manager.

  


Accessed from:

  * "For <#> Account(s)" link on the Primary Account Manager's Contact record



  


The following filter(s) are set differently than on the main report:

  * "Primary Account Manager" = Set to non-blank, based on the place from which the report is opened.
