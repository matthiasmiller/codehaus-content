10.2.8. "New User Request" MailTo Email

Overview: This is an email that can be sent to the  Account Reseller for an Account, as a request for adding a new User to the Account. It is sent via the "MailTo" feature, opening a draft in the user's default email platform (app), with the following items defaulted:

  


From: <the email address associated with the user's default email app> 

  


To: <the primary Email address of the "Account Reseller" for the Account from which the link is clicked> (user can set other "To:" addresses in the email draft if desired)
