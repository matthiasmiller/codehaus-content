8.2.3. Recently Changed Church Affiliations Report

  


Requirements

Overview: This is a custom report of Contact records that have an entry for "Previous Church Affiliation" and/or "Previous Other Church Affiliation".  

  


Accessed from: 

Providers | Accounts | Recently Changed Church Affiliations (opens the report directly, bypassing the filter screen) 

  * "Recently Changed Church Affiliations" report alert notification 



  


Title: Recently Changed Church Affiliations

  


Preface: N/A

  


Columns: 

  * Contact (link to open the record) 
  * Prev. Church Affiliation 
  * Prev. Other Church Affiliation 
  * New Church Affiliation (displays the current "Church Affiliation") 
  * New Other Church Affiliation (displays the current "Other Church Affiliation", if applicable)
  * Change Date (__
  * Changed By (__

Row-Specific Buttons: 

  * N/A



  


Grouped by: Account Reseller (alphabetically) 

  


Sorted by: Contact (alphabetically) 

  


Filters: 

  * Account Reseller (optional; drop list of "Short Display Names" for all Account Resellers; defaults to the current user) 
  * Change Date Start (optional; date; defaults to 30 days before the current date) 
  * End (optional; date; defaults to blank = all) 



  


Buttons: 

  * N/A



  


Menu Visibility: Visible to all users who can view the "Providers" menu 

  


Other Notes:

  * This is an "auto-push" report, meaning that if only 1 item is included on the report, the Solution opens that item directly, bypassing the report screen.
