9.4.1.4. Accounts by Account Manager

*Done. 

  


Overview: This is the "Main Accounts Report", filtered to a single Account Manager.

  


Accessed from:

  * Home | Accounts | My Accounts (opens the report directly, bypassing the filter screen, setting "Account Manager" filter to the Contact for the current user) 
  * "For <#> Account(s)" link on the Account Manager's Contact record



  


The following filter(s) are set differently than on the main report:

  * "Account Manager" = Set to non-blank, based on the place from which the report is opened.
