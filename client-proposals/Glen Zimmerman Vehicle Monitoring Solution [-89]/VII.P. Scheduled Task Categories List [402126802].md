6.14. Task Categories List

*Done. 

  


Overview: This is a standard user-editable simple list used to track categories for Scheduled Tasks. 

  


Initial list items include: 

  * Scheduled Email Notifications 
  * Subscription Automation



  


This list is universal for all users, and users can add to it via the "+" option on the drop list on the Scheduled Task record, or via the Lists interface. 

  


The initial items for this list are to be set up at deployment by <Service Name> and/or CodeCrafters.
