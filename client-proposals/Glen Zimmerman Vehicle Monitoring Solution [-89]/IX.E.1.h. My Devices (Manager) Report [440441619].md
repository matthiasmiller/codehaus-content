Overview: This is the "Main Devices Report", filtered to include Devices for which the current logged-in user is an Account Manager.

  


Accessed from:

  * My <Service Name> | Devices | My Devices (Manager) (opens the report directly, bypassing the filter screen)



 

The following filter(s) are set differently than their defaults:

  * "Account Manager": Set to the Contact for the current logged-in user
