6.1. Contact Phone Types List

  


Requirements

*Done. 

  


Overview: This is a standard user-editable simple list used to track types of phone numbers for Contact records. 

  


Initial (read-only / "Silverloom-supplied") list items include: 

  * Fax
  * Home
  * Mobile
  * Work



  


Additional items can be added by users with the ability to add to lists.
