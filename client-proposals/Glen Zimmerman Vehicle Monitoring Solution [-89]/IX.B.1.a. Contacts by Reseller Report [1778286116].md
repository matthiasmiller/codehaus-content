9.2.1.1. Contacts by Reseller Report

  


Requirements

*Done. 

  


Overview: This is the "<Service Name> Main Contacts Report", filtered to a single Reseller.

  


Accessed from:

  * Providers | Reseller | My Clients (opens the report directly, bypassing the filter screen)



  


The following filter(s) are set differently than on the main report:

  * "Account Reseller": Set to 
    * the Contact for the current logged-in user (if the current user is an "Account Reseller", "Group Admin", or "Master Administrator"); 
    * the organization Reseller Contact to which the current logged-in user is linked (if the current user is a "Reseller Rep")
