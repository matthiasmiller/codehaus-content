9.5. Family

Accessed via: FaithBuilders Scholarship Services | Family | Families

  


Sections and Fields:Â 

Family:

  * ID (auto-set and read-only; <1st Guardian Last Name>, <1st Guardian First Name> #<unique ID>)
  * Parent / Guardian Names (auto-set and read-only)



  


1st Parent / Guardian:

  * First Name
  * Last Name



  


2nd Parent / Guardian:

  * First Name
  * Last Name



  


Contact Information:

  * Phone
  * Email
  * Address
  * City
  * State (unlabeled; drop-list of state abbreviations; defaults to PA)
  * Zip (unlabeled)
  * County (drop-list of PA Counties)



  


Notes

  * Memo



  


Reports:

  * Current Family Application (creates and opens a new application for the current family)
  * Create New Student (creates and opens a new student for the current family)



  


Other Notes:

  * Families have a primary contact. The secondary contact is typically used to distinguish between to families sharing the same primary name.
  * The Students by Family report is displayed as a bottom report, filtered for the current family.
