13.4. PDF Merges: Overview

None of the merges have strict layout requirements.

  


The labels are printed on Avery #5160, which is 30 labels per page 2 5/8 x 1 inch. We will not actually print labels. We will simply provide an Excel export that can be merged with the labels.

  


All of the envelopes are printed on #10 envelopes. In most cases, this will be on a pre-printed letterhead envelope, in which case no return address will be printed.
