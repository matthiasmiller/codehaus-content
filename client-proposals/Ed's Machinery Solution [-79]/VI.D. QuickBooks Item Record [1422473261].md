6.4. QuickBooks Item Record

This is a custom record type that the Solution will use to track the following information from the QuickBooks Items:

  


  * AppHosting ID (number; automatic)
  * List ID (text; required)
  * Item Name (text; required)
  * Item Type (list of QuickBooks Item items)
  * Active (checkbox)
  * Sales Description (text)
  * Purchase Description (text)
