6.5. QuickBooks Bill Record

This is a custom record type that the Solution will use to track the following information from the QuickBooks Bills:

  


  * Reference Number (number; required)
  * Transaction Date (date)
  * Vendor (list of QuickBooks Vendors)
  * Embedded Spreadsheet of:
    * Item #
    * Item Qty
    * Item Description
    * Item Cost
