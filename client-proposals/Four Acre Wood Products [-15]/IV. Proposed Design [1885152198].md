4\. Proposed Design

Product Category

This is the overarching category, such as:

  * Base Corner Cabinets
  * Base Cabinets
  * Full Height Door Base
  * Sink Base
  * Range Base
  * Wall Cabinets
  * Special Wall Cabinets
  * Oven Cabinets
  * Utility / Oven Cabinets
  * Vanity Base



  


Fields:

  * Name



  


Product Line (what's the best term??)

This is tied back to a product category, for example "Blind Corner Base".

  


Fields:

  * Name
  * Product Category



  


Product Item

This is directly tied back to a product line, and thus indirectly back to a product category. See [https://docs.google.com/spreadsheets/d/13A2W8djXDtQ_5Gd7SwQiaSg6brcSpmWh5T18q7n57Ng/edit#gid=0](https://docs.google.com/spreadsheets/d/13A2W8djXDtQ_5Gd7SwQiaSg6brcSpmWh5T18q7n57Ng/edit#gid=0)

  


Fields

  * Name
  * SKU
  * Type
    * Standalone
    * Add-on
  * Available for:
    * Product Category
    * Product Line
    * Product Item
  * Available Sides:
    * N/A
    * Front
    * Back
    * Left
    * Right
    * Top
    * Bottom
  * # of Dimensions (0, 1, 2, or 3)
  * Size (inches)
    * Width (1D+)
    * Height (2D+)
    * Depth (3D+)
  * Standalone dimensions
    * Provide default dimensions
    * Indicate whether the dimensions can be overridden
  * Add-on dimensions
    * Indicate which dimension each one ties back to.
  * Pricing Method
    * % of Parent Product (for add-ons)
    * Fixed Price
  * Doors (repeating group?)
    * Poaition (Front / Back / Left / Right)
    * Include by Default?
    * Desc (Left / Right / etc)
    * Width
    * Height
  * Drawers (repeating group?)
    * Desc (Left / Right / Etc)
    * Width
    * Height



  


Job/Quote

  * Job Name
  * Customer etc
  * Date
  * Cabinets
  * Add Cabinet



  


Cabinet

  * Job/Quote
  * Room
  * Wood
  * Finish
  * Distressing
