6.1.1. Configure Wiki Categories Report

*Done. 

  


Overview: This is a standard report of Wiki Categories.

  


Accessed from: Admin | Configuration | Configure Wiki Categories

  


Title: Configure Wiki Categories

  


Columns: 

  * Display Name (link to record; displays the category's name preceded by the hierarchy; standard column)



  


Grouped by: N/A

  


Sorted by: Display Name (alphabetically)

  


Filters: N/A

  


Buttons: 

  * New Category (opens a blank new Wiki Category record; standard) 



  


Menu Visibility: 

  * Visible to and accessible by all users
