6.2.5. Congregations by Agent

*Done. 

  


Overview: This is a custom report of the number of Clients per Congregation, grouped by Agent.

  


Accessed from: Admin | Admin Reports | Congregations by Agent 

  


Title: Congregations by Agent

  


Preface: N/A

  


Columns: 

  * Congregations
  * Clients (number of clients in the congregation; total row at the top displays sum)



  


Grouped by: Agent (alphabetically; displaying the Agent's name + Agent Code)

  


Sorted by: Congregation (alphabetically)

  


Filters: 

  * Agent (Blank = All) (drop list of all active "In-State Agent"-type Contacts; defaults to blank = all)
  * Include inactive clients (checkbox; defaults to no checked)



  


Menu Visibility: Admin

  


Other notes:

  * N/A
