6.1.2. Configure Contact Types Report

*Done. 

  


Overview: This is the standard Silverloom report to view and configure Contact Types in the Solution. Note that any customizations are noted as such in the spec. 

  


Accessed from: Advanced | Configuration | Configure Contact Types 

  


Title: Contact Types

  


Columns: 

  * Type (link to record)
  * Individual (Yes/No)
  * Organization (Yes/No)
  * Client? (custom column; displays "Yes" / "No")
  * N/C Vehicles (custom column; displays the "Maximum Number of No-Charge Vehicles")



  


Grouped by: Active/Inactive (active at the top)

  


Sorted by: Type name

  


Filters: 

  * Show inactive contact types (checkbox; defaulting to not checked; standard filter)



  


Buttons: 

  * New Contact Type (opens a new Contact Type record) 



  


Menu Visibility: 

  * Visible to all users, but accessible for Admins only (grayed-out for non-Admins) 



  


Other Notes: 

  * N/A
