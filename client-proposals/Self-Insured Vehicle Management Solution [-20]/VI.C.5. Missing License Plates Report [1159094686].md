6.3.5. Missing License Plates Report

*Done.

  


Overview: This is a custom report of active Vehicles with with "License Plate" blank but with a Vehicle Type that requires a license plate.

  


Accessed from: 

  * Home | Vehicles | Missing License Plates (opens a prompt screen with the below filter(s) that are visible to no-Admin users) 
  * Admin | Exception Reports | Missing License Plates (opens a prompt screen with the below filters) 



  


Title: Missing License Plates

  


Preface: N/A 

  


Columns: 

  * Name
  * Address
  * City
  * State
  * Zip
  * Phone
  * VIN / Serial # (link to to the Vehicle record)
  * Year
  * Make
  * Model



  


Grouped by: Agent (displayed as "<Agent Name> \- <Agent Code>")

  


Sorted by: Name (alphabetically)

  


Row-Specific Buttons: N/A

  


Filters: 

  * Current Agent (blank = all) (visible only for Admin users; drop list of active "In-State Agent"-type Contacts with Funds; for Admins, defaults to blank = all; for non-Admins, defaults to the current user's Contact) 
  * Number of Days Since Entry (number; 0 decimals; defaults to "30"; looks at the "Date Entered"; all Vehicles entered more than the specified number of days prior are included) 



  


Buttons: 

  * N/A



  


Other Notes:

  * Note that non-Admin users cannot view this report for other Agents' Clients / Vehicles.
