6.4.4. Liability & <Collision> Expenses Report

*Done. 

  


Overview: This is a custom report of Liability and <Collision> expenses, based on Claim records.

  


Accessed from: Admin | Admin Reports | Liability & <Alternate text for Collision - Short label> Expenses 

  


Title: Liability & <Alternate text for Collision - Short label> Expenses 

  


Preface: N/A

  


Columns: 

  * Agent (no heading label; link to open the Agent's Contact record)
  * # Liability Claims (total row displays sum) 
  * Total Liability $ (total row displays sum)
  * Property Damage $ (Liability) (total row displays sum) 
  * Towing / Storage/ Other $ (Liability) (total row displays sum) 
  * # Pain & Suffering (Liability) (total row displays sum) 
  * Pain and Suffering $ (Liability) (total row displays sum) 
  * # Medical Liability (Liability) (total row displays sum) 
  * Medical $ (Liability) (total row displays sum) 
  * # <Alternate text for Collision - Short label> Claims (total row displays sum) 
  * <Alternate text for Collision - Short label> $ (total row displays sum) 
  * Total # of Claims (total row displays sum) 
  * Total $ (total row displays sum) 



  


Grouped by: Fund

  


Sorted by: N/A

  


Filters: 

  * Filter Dates (drop-list of "Filter Dates" list items; defaults to "Period Year")
  * Period Year (blank = all) (visible if "Filter Dates" = "Period Year"; multi-select list of Period Years, with "Current Period Year" at the top)
  * From Date (blank = all) (visible if "Filter Dates" = "Date Range"; date; defaults to blank)
  * Thru (blank = all) (visible if "Filter Dates" = "Date Range"; date; defaults to the current date)
  * Calendar Year (blank = all) (visible if "Filter Dates" = "Calendar Year"; multi-select list of calendar years, with "Current Year" at the top)



  


Menu Visibility: Admin

  


Other Notes: 

  * N/A
