3.5.5. System Switches - "Collision" Terminology

  


Requirements

Custom System Switches pertaining to the terminology used for "Collision" coverage on Vehicles: 

  


  * Label: "Alternate text for Collision - Long label" 
    * Code: "Config_CollisionFullLabel" 
    * Field Type: Plain text (string) 
    * Description: Used to set Plan-specific wording for "Collision" coverage. 
      * Note that a word count limit could be considered in the future if length becomes an issue. 
    * Default Value: "Vehicle Loss Sharing" 
    * Set to: As described in the "Deployment - System Switches" section of the proposal.
    * Other Notes: 
      * Hard-coded 



  


  * Label: "Alternate text for Collision - Short label" 
    * Code: "Config_CollisionShortLabel" 
    * Field Type: Plain text (string)
    * Description: Used to the set the abbreviation for the Plan-specific wording for "Collision" coverage. 
    * Default Value: "VLS"
    * Set to: As described in the "Deployment - System Switches" section of the proposal.
    * Other Notes: 
      * Hard-coded 



  


  * Label: "Alternate text for Collision - Legal label" 
    * Code: "Config_CollisionLegalLabel" 
    * Field Type: Plain text (string) 
    * Description: Used to set the Plan-specific legal wording for "Collision" coverage. 
    * Default Value: "Collision" 
    * Set to: As described in the "Deployment - System Switches" section of the proposal.
    * Other Notes: 
      * Hard-coded
