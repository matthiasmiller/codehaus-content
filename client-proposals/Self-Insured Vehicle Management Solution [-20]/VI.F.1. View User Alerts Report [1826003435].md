6.6.1. View User Alerts Report

  


Requirements

*Done. 

  


Overview: This is a standard report of User Alert records.

  


Accessed from: 

  * Home | Alerts | View User Alerts (opens the prompt screen with the report filter)
  * Admin | User Alerts | View User Alerts (opens the prompt screen with the report filter) 



  


Title: User Alerts

  


Preface: N/A

  


Columns: 

  * Alert Message (no column heading; displays the "Message"; link to open the record) 
  * Created By
  * Date



  


Grouped by: "Title" (alphabetically)

  


Sorted by: Creation date

  


Filters: 

  * Show Alerts Since (date; defaults to 90 days before the current date)



  


Buttons: 

  * Send User Alert (opens a new blank User Alert record)



  


Menu Visibility: 

  * All



  


Development Specification

Tim Reitz 08/01/2025: Grouping by Title makes sense with the bold text display of the title, but it also would make sense to list them out by Creation Date.
