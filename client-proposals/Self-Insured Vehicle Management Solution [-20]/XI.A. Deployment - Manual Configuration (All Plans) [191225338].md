11.1. Deployment - Manual Configuration (All Plans)

  


Requirements

The following details are to be configured on a Plan-by-Plan basis as part of system setup at deployment. These are to be done as a joint effort between the CCI support team and the Plan.

  


  * Configure the manually-set System Switches 
  * Configure the Silverloom Settings: 
    * Contact Settings details
    * Coverage Settings details: 
    * New Member Card Printout details
    * Claim General Release Printout details
    * Agent Change Notice Printout details 
    * Authorized Signature 
    * Plan Logo 
    * Plan-Specific Settings details
    * Guidelines Compliance questions
  * Configure background processes 
  * Configure Handbook
  * Other as needed
