4.7. Vehicle Deactivation Reasons List

*Done. 

  


This is a custom user-editable simple list used to track Vehicle Deactivation Reasons. 

  


This list includes the following hard-coded items (which cannot be edited or deleted except via coding work):

  * Sold
  * Wrecked
  * Needed Added Coverage
  * Other (must specify)



  


This list is universal for all users, and all users can add to it via the Lists menu.
