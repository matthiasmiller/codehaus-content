6.2.9. Cancelations of Clients State Report

TODO_VA: standardize per current "reports" snippet

  


This is a report of inactive clients who have a reportable deactivation reason.

  


This is accessed from Admin | Cancellations | Cancellations of Clients State Report. 

  


Title: 

  * Cancellations of Clients State Report as of #End Date#
  * Cancellations of Clients State Report #Start Date# through #End Date#



  


Columns: 

  * Name
  * Address
  * City
  * State
  * Zip
  * Date Canceled
  * Client Deactivation Reason
  * Client Deactivation Reason (other)



  


Grouped by: N/A

  


Sorted by: Name

  


Filters: 

  * Start Date
  * End Date (defaults to the current date)



  


Menu Visibility: Admin
