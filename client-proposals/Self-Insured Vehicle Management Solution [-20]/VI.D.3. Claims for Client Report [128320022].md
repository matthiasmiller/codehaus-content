6.4.3. Claims for Client Report

*Done. 

  


Overview: This is the "All Claims" report, filtered to show only Claims for a single Client.

  


Accessed from:

  * Accessed from: the "View <#> Client Claim(s)" link on the Contact record (opens the report directly, bypassing the filter screen)



  


The following filter(s) are set differently than on the main report:

  * "Client or Vehicle Owner Name": set to the Contact record from which the report is opened; 
  * "End Date": set to blank = all
