6.1.3. Configure Client Deactivation Reasons Report

*Done. 

  


Overview: This is a custom report of Client Deactivation Reasons.

  


Accessed from: Advanced | Configuration | Configure Client Deactivation Reasons

  


Title: Client Deactivation Reasons

  


Columns: 

  * Reasons (displays the Reason, link to record)
  * Reportable? (Yes/No)
  * Requires Clarification? (Yes/No)



  


Grouped by: Active/Inactive (active at the top)

  


Sorted by: Reason

  


Filters: 

  *  Show inactive client deactivation reasons (checkbox; defaulting to not checked)



  


Buttons: 

  * New Reason (opens a new Client Deactivation Reason record) 



  


Menu Visibility: 

  * Visible to all users, but accessible for Admins only (grayed-out for non-Admins)
