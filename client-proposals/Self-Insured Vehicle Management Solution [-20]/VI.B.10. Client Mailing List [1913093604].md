6.2.10. Client Mailing List

*Done. 

  


Overview: This is a special custom report that runs on Contact records and is used to generate a list of Client addresses, which can be exported as a PDF or Excel file for generating address labels or for other purposes. 

  


Accessed from" Admin | Renewals and Printing | Client Mailing List (opens a prompt screen with the below filter) 

  


Title: "Client Mailing List"

  


Preface: N/A

  


Columns: 

  * Display Name (link to open the Contact record) 
  * Street Address Line 1
  * Street Address Line 2
  * City
  * State
  * Zip



  


Row-Specific Buttons: N/A

  


Grouped by: N/A

  


Sorted by: Name (alphabetically) 

  


Filters: 

  * Contact Type(s) (multi-select list of active Contact Types; defaults to none selected = all)



  


Buttons: N/A

  


Menu Visibility: Admin

  


Other Notes: 

  * To download an Excel spreadsheet of this report (for generating address labels, etc.), the user opens the drop-down menu in the top right-hand corner of the screen and selects the "Excel" option.
