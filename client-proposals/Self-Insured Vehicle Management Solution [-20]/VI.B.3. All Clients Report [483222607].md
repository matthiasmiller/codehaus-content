6.2.3. All Clients Report

*Done.

  


Overview: This is the "All Contacts" report, with the subset displaying only Client-type Contacts, filtered to show only active Clients.

  


Accessed from:

  * Home | Contacts | All Clients (opens the report directly, bypassing the filter screen)



  


Title: All Clients

  


The following filter(s) are set differently than on the main report:

  * Status = "Active"
