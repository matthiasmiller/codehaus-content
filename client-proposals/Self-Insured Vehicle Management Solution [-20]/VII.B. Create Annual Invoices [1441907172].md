7.2. Create Annual Invoices

  


Requirements

Overview: This is a special "import" report as part of the annual renewal procedure, to create Invoice records for all active clients who have fees or credits through the end of the current period.

  


Accessed from: Admin | Renewals and Printing | Create Annual Invoices

  


Opens a prompt screen with the following: 

  * Create Invoices for Period Start Date: July 01, 2025 (on-screen message)
