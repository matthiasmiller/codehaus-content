6.2.2. My Clients Report

Overview: This is the "All Contacts" report, with the subset displaying only Client-type Contacts, filtered to show only active Clients of the current user.

  


Accessed from:

  * Home | Contacts | My Clients (opens the report directly, bypassing the filter screen)



  


Title: My Clients

  


The following filter(s) are set differently than on the main report:

  * Status = "Active"
  * Agent = Set to the Contact for the current user
