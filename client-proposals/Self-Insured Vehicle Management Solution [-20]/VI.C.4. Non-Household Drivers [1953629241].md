6.3.4. Non-Household Drivers

*Done. 

  


Overview: This is a custom report of all active Vehicles with "Vehicle Primary Driver" set to "(Other/Non-household Driver)". This gives Admins visibility into which Vehicles are set up with a non-household driver as their primary driver.

  


Accessed from: Admin | Exception Reports | Non-Household Drivers (opens the filter prompt screen)

  


Title: Non-Household Drivers

  


Preface: N/A

  


Columns: 

  * Client Name (displays the current "Owner" for the Vehicle; link to the Owner's Contact record)
  * Client ID (displays the Owner's "Contact ID"; total at the top of each grouping displays the sum, in the following format: "<#> Client(s)") 
  * Vehicle Display Name (displays the vehicle information in the following format: "<Year> <Make> <Model> <VIN>"; link to the Vehicle record)
  * Driver Name (displays the name of the non-household driver in the following format: "<Last Name>, <Other Driver First Name> <Middle Name>)
  * Address (displays the street address from the Owner's Contact record, in the following format: "<Address>, <Address 2>") 
  * City (displays the "City" from the Owner's Contact record) 
  * State (displays the "State" from the Owner's Contact record) 
  * Zip (displays the "Zip" from the Owner's Contact record) 
  * Phone Number (displays the primary (top) Phone number from the Owner's Contact record) 



  


Grouped by: Agent (grouping label displays in the following format: "<Agent's Contact Display Name> \- <Agent Code>") 

  


Sorted by: Client Name (alphabetical) 

  


Filters: 

  * Agent (Blank = All) (drop list of active "In-State Agent"-type Contacts with a fund)



  


Buttons: N/A

  


Menu Visibility: Admin
