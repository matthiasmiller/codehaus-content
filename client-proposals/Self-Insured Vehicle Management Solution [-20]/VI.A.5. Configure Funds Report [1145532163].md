6.1.5. Configure Funds Report

*Done.

  


Overview: This is a custom report of Funds.

  


Accessed from: Advanced | Configuration | Configure Funds

  


Title: Funds

  


Columns: 

  * Fund Name (displays the Fund Name, link to record)
  * Checkbook Holder (displays the Checkbook Holder)



  


Grouped by: Active/Inactive (active at the top)

  


Sorted by: Fund Name

  


Filters: 

  * Show inactive Funds (checkbox; defaulting to not checked)



  


Buttons: 

  * New Fund (opens a new Fund record)



  


Menu Visibility: 

  * Visible to all users, but accessible for Admins only (grayed-out for non-Admins)
