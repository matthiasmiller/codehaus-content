6.1.4. Configure Congregations Report

*Done.

  


Overview: This is a custom report of Congregations.

  


Accessed from: Advanced | Configuration | Configure Congregations

  


Title: Congregations

  


Columns: 

  * Congregation Name (displays the name, link to record)



  


Grouped by: Active/Inactive (active at the top)

  


Sorted by: Congregation Name

  


Filters: 

  * Show inactive congregations (checkbox; defaulting to not checked)



  


Buttons: 

  * New Congregation (opens a new Congregation record)



  


Menu Visibility: 

  * Visible to all users, but accessible for Admins only (grayed-out for non-Admins)
