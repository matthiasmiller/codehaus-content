4.3. Vehicle Models List

  


Requirements

*Done. 

  


This is a custom user-editable simple list used to track Vehicle Models. 

  


Note that each Model is specific to one Vehicle Make.

  


Example list items may include: 

  * Explorer (specific to the Ford Make)
  * Corolla (specific to the Toyota Make) 
  * Jetta (specific to the Volkswagen Make)
  * etc. 



  


This list is universal for all users, and all users can add to it via the "+" option on the drop list on the Vehicle record.
