6.5.4. Aging Reports

*Done. 

  


Overview: This is a custom report of Invoice records with "Status" = "Invoiced".

  


Accessed from: Home | Invoices | Aging Reports (opens a prompt screen with the filter(s) for the report)

  


Title: Aging Report

  


Preface: N/A

  


Columns: 

  * Invoice Number (link to open the Invoice record)
  * Invoice Date
  * Invoice Amount
  * Name (displays the Invoice Contact)
  * Address
  * City
  * State
  * Zip
  * Phone



  


Grouped by: Agent & Agent Code, in the following format: "<Agent Name> \- <Agent Code>"

  


Sorted by:

  * First by: Invoice Date
  * Second by: Invoice Contact 
  * Third by: Invoice ID



  


Filters: 

  * Current Agent (Blank = All) (visible only for Admin users; drop list of "In-State Agent"-type Contacts; defaults to blank for Admins; hidden and defaulted to the current user for non-admins)



  


Buttons:

  * N/A



  


Menu Visibility: All users

  


Other Notes:

  * N/A
