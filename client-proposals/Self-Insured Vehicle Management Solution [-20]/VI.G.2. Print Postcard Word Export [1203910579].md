6.7.2. Print Postcard Word Export

*Done. 

  


Overview: This is a special custom report of that generates a Word file export of the Postcard Printout (see Postcard Printout spec). 

  


Accessed from: "Print Postcard" link on the Postcard record (opens the filter screen). 

  


Title: "Print Postcard"

  


Preface: N/A

  


Columns: 

  * Report / columns not displayed to the user (used for generating the printout) 



  


Row-Specific Buttons: N/A 

  


Grouped by: Report / columns not displayed to the user (used for generating the printout) 

  


Sorted by: Report / columns not displayed to the user (used for generating the printout) 

  


Filters: 

  * Title/Description (drop list of "Name/Description" items for all Postcards in the Solution; defaults to the record from which this screen was opened) 
  * Send to Contact Type(s) (multi-select list of active Contact Types; at least one item selection is required; defaults to none selected) 
  * Continue (button; clicking this button generates the Word document with the Postcard printout, based on the selected filters -- see Postcard Printout spec)

  * N/A



  


Menu Visibility: 

  * N/A (not included on any menus)
