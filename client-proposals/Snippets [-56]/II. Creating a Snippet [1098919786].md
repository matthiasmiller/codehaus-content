2\. Creating a Snippet

To create a snippet, open "Advanced | Snippets | Snippets". Click Create New Snippet.

  


  * Enter a name
  * Select the type
  * Enter the visibility if it will be conditionally visible
  * Enter the contents



  


Common visibility expressions include:

  * ContactType = "Customer"
  * IncidentType = "Sales Opportunity" AND IncidentWorkflow = "Primary Pipeline"
  * etc.



  


The contents can include expressions, such as:

My name is #ContactName#.

The incident title is #IncidentTitle#.

etc.
