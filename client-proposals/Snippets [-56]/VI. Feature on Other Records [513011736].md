6\. Feature on Other Records

  


Requirements

For records / memos that include the Snippet feature: 

  


  * Snippet (drop list of Snippets that include the current memo/record in the "Type(s)" selection; includes an ellipsis button to create a new Snippet (if none is selected) or open the selected Snippet) 
  * Append (visible if "Snippet" is not blank; button; appends the contents of the selected Snippet to the bottom of the corresponding memo)
