8.4. Export Special Placements

Clicking on the Export Special Placements menu option should open a page that prompts for the following:

  * Publication (required, list of publications, default to blank; for Publication Admin and Standard Users, show only their assigned publication, for Full Admin show all)
  * Publication Date (the date should default to the current Publication Date)
  * File Type (choice of PDF or Excel; defaulting to blank)



  


Clicking Continue exports an file of the Special Placement Report for the selected Publication Date and of the selected file type for the selected Publication.

  


Any user can run this export.
