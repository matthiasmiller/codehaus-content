17.1.1. Configure Contract Scripts Report

Overview: This is a custom report of Contract Scripts.

  


Accessed from: Advanced | Configuration | Configure Contract Scripts (opens the report directly, bypassing the filter screen)

  


Title: Contract Scripts

  


Preface: None

  


Columns: 

  * Name
  * Type



  


Grouped by: Type

  


Sorted by: Name (alphabetical)

  


Filters: 

  * Type (drop-list of Contract Scripts types)



  


Buttons: 

  * New Contract Script (opens blank new record)



  


Menu Visibility: 

  * All users



  


Live Updating: No
