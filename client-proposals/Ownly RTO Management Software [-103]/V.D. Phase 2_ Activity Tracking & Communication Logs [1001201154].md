5.4. Phase 2: Activity Tracking & Communication Logs

Goal: Give collections teams deeper visibility into customer behavior and engagement patterns.

  


Key Features:

  


  * Unified Communication System:
    * Centralized email per Rental Company ([customerservice@onlyrto.com](mailto:customerservice@onlyrto.com))
    * Single SMS/phone number for all communications
    * All interactions auto-formatted as memos on customer records
    * Built-in tel/sms link support for easy customer responses
  * Communication Logging: email, SMS, and call records stored as memos
  * Portal Activity Tracking:
    * Logins, timestamps, responsiveness patterns
    * Time-of-day activity analysis for optimal contact scheduling
  * Engagement Dashboard for collections teams
  * Engagement Alerts when past-due customers interact with the system
  * Enhanced Link Tracking:
    * Track tel/sms link clicks in communications
    * Measure customer responsiveness to different message types
