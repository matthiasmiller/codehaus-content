13.5.7. Search Notes Report

Overview: This is a custom report of all Notes, which are gathered from Contact, Contract, Contract Definition, Payment, Product, and User records throughout the system.

  


Accessed from:

Home | Notes | Search Notes

  


Title: Search Notes

  


Preface: NA

  


Columns: 

  * Linked Record
  * Created By/On
  * Notes
  * Needs Followup
  * Assignees
  * Due Date
  * Completed By/On



  


Row-Specific Buttons: NA

  


Grouped by:

  * Contact
  * Contract
  * Contract Definition
  * Payment
  * Product
  * User 



  


Sorted by: Note ID

  


Filters: 

  * Search Text
  * Originators (multi-select list of active users; match any)
  * Assignees (multi-select list of active users; match any)
  * Needs Followup (yes/no; blank = all)
  * Overdue (yes / no; blank = all)
  * Created (date range)



  


Buttons: NA

  


Menu Visibility: All Users
