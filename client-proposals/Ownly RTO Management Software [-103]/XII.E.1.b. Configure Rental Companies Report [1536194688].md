12.5.1.2. Configure Rental Companies Report

Overview: This is a custom report of Rental Companies.

  


Accessed from: Advanced | Configuration | Configure Rental Companies (opens the report directly, bypassing the filter screen)

  


Title: Rental Companies

  


Preface: None

  


Columns: 

  * Name



  


Grouped by: Active / Inactive (the grouping is only included when inactive Rental companies are included via the filter)

  


Sorted by: Name (alphabetical)

  


Filters: 

  * Include Inactive (checkbox)



  


Buttons: 

  * New Rental Company (opens blank new record)



  


Menu Visibility: 

  * All users



  


Live Updating: No

  


Other Notes:
