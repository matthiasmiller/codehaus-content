13.5.1.3. Configure Contract Definitions Report

Overview: This is a custom report of Contract Definitions.

  


Accessed from: Advanced | Configuration | Configure Contract Definitions (opens the report directly, bypassing the filter screen)

  


Title: Contract Scripts

  


Preface: None

  


Columns: 

  * Rental Company
  * State
  * Rental Contract Type
  * View (link to the record)



  


Grouped by: Rental Company (if the ask prompt is blank)

  


Sorted by: N/A

  


Filters: 

  * Rental Company (drop-list of Rental Companies)



  


Buttons: 

  * New Contract Definition (opens a new Contract Definition record) 



  


Menu Visibility: 

  * All users



  


Live Updating: No
