12.5.1.5. Configure Note Templates Report

Overview: This is a custom report of Note Templates.

  


Accessed from: Advanced | Configuration | Configure Note Templates (opens the report directly, bypassing the filter screen)

  


Title: Configure Note Templates

  


Preface: None

  


Columns: 

  * Name
  * Needs Followup



  


Grouped by: Rental Company (if the Rental Company ask prompt is blank)

  


Sorted by: Name (alphabetical)

  


Filters: 

  * Rental Company (drop-list of Rental Companies; defaults if there is only one rental company)
  * Type (drop-list of Note types)



  


Buttons: 

  * New Note Template (opens blank new record; default type and rental company fields based on ask prompts)



  


Menu Visibility: 

  * All users



  


Live Updating: No
