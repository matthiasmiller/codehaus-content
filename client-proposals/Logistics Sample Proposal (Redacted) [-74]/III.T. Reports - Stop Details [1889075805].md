3.20. Reports - Stop Details

Create a report that shows how many days each route has been run for a given period of time. The use case is to determine how many days the second truck is run each month.

  


Prompt:

  * Date Range
    * By Month (prompt for Month and Year)
    * By Year (prompt for Year)
    * By Range (prompt for Start and End Date)
  * Group By
    * Month
    * Week
    * Weekday
    * Date
  * Stop Type (multiple choice)
    * Pickups
    * Deliveries
    * Break

Group By

  * Day, Week (i.e. start of week), or Month (see prompt)



Columns:

  * Route Name
  * # of Days (number of days with one or more stops)
  * # of Stops
  * Avg Length of Stop
  * Avg Review



  


Show totals by month, as well as grand totals for the entire year at the bottom of the report.
