4.5. View Address Book

  


Requirements

This report would include all active households except Prior Marriages.

  


It will have no filters or groups.

  


It will sort by Household #.

  


It will show the following columns

  * Household #
  * Head of Household Last Name
  * Head of Household First Name
  * Head of Household Middle Name
  * Spouse Maiden Name
  * Spouse First Name
  * Spouse Middle Name
  * Address
  * City
  * State
  * Zip
  * Phone Number
