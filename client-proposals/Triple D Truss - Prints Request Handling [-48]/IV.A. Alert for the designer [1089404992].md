4.1. Alert for the designer

Add a system notification with the following details:

  * Title: "New Truss Print Request for Order #<xxxx>" 
  * Text (can be include expressions): None
  * Trigger: When the Generate Request checkbox on an Order is checked 
  * Action: Opens the Print Requests report, filtered to show only Incomplete Requests 
  * User(s) to notify: Designer
    * Note that this would be determined by the "Subscribe to New Truss Print Request Notifications" checkbox on the user's contact record. 
  * Other Notes: 
    * Note that this is a real-time alert.
