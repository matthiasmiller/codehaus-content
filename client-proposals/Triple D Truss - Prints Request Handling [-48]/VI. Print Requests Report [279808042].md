6\. Print Requests Report

This is a report of Print Requests with various filters.

  


This would be accessed from various menu options. See the "Menu Options" section. 

  


Title: "Print Requests" when showing all print requests. Otherwise, "Completed Print Requests" or "Incomplete Print Requests" 

  


Columns: 

  * Order # (link that opens the Truss Prints Child Screen for the order) 
  * Customer Name 
  * Span 
  * Pitch 
  * Description 



  


Grouped by: Incomplete/Complete first, then by Order # 

  * The Incomplete section at the top should be grouped by Order # with the earliest generated Print Requests at the top. (This will use the hidden Truss Print Request Date.)
  * The Complete section at the bottom should be grouped by Order # with the earliest completed Print Requests at the top. 
  * There would be a link to open the View/Edit Truss Prints child screen on the sales order on the heading row for each Order #. This would be a a full-page view of the child screen, in the same tab.



  


Sorted by: Generated Date or Completed Date (see above) 

  


Filters (match on all): 

  * Salesperson (list) 
  * Days (looks at generated/completed date; defaults to blank = all)
  * Span Feet (text field)
  * Span Inches (text field)
  * Pitch (text field)
  * Left O.H. (text field)
  * Right O.H. (text field)
  * Left Cant. (text field)
  * Right Cant. (text field)
  * Status (list of "Complete" or "Incomplete"; blank = all) 



  


Buttons: N/A 

  


Data Access: Available for all users 

  


Other Notes: 

  * Opening a Print Request from this report will show all truss prints for an order. The designer will need to cycle through trusses to get to the pending requests. This is detailed above in the "View/Edit Truss Prints" portion of the Truss Prints section.
