4.2. Alert for the salesperson

Add a system notification with the following details:

  * Title: "Truss Prints Uploaded for Order #<xxxx>"
  * Text: None 
  * Trigger: When the all the requests in an order are marked complete (via the checkboxes)
  * Action: Opens the Print Requests report, filtered to show only Completed Requests 
  * User(s) to notify: Salesperson assigned to the order
  * Other Notes:
  * Note that this is a real-time alert.
