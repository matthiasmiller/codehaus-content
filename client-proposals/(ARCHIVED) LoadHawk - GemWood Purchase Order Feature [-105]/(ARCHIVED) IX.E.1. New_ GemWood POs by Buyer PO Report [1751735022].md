9.5.1. New: GemWood POs by Buyer PO Report

Overview: This is the All Purchase Orders Report, filtered to display "GemWood"-Type POs items linked to a single "Buyer"-Type PO.

  


Accessed from:

  * Used as the right pane of the "Buyer Purchase Orders" report. 
  * Displayed in the bottom report on the "Buyer"-Type Purchase Order records

The following filter(s) are set differently from the main report:

  * "PO Type": Set to "GemWood"
  * "Buyer PO #": Set to to the record from which the report is opened or the the record passed through from another report
