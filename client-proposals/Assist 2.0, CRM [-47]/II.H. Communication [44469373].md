2.8. Communication

  


Requirements

The Communication record will be used to track individual points of contact with the customer.

  


It will track:

  * Opportunity
  * Date (required; default to today)
  * Time (required; default to current time)
  * Salesperson (required; defaults to current user)
  * Type
    * Phone Call
    * Text Message
    * Email
  * Notes (plain text multi-line)
