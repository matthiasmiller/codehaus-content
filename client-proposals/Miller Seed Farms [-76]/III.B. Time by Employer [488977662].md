3.2. Time by Employer

This is report of time cards, grouped by employer.

  


Columns: 

  * Date
  * Start
  * End
  * Hours
  * Notes



  


Grouped by: 

  * Employer (sub-grouped by Employee)



  


Sorted by: N/A

  


Filters: 

  * Start Date (blank = all)
  * End Date (blank = all)
  * Employers (blank = all)
