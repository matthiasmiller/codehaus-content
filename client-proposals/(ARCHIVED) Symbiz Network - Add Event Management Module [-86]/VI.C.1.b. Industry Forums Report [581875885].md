6.3.1.2. Industry Forums Report

*Documented 

  


Add the following new report: 

  


This is a filtered version of the the Master Events Report that displays only Industry Forums.

  


This is accessed from:

  * Admin | Event Management | Industry Forums (bypasses the filter screen to open the report directly)



  


The following filters are set to something other than their default:

  * Event Type = Industry Forum



  


Other Notes: 

  * See the spec for the Master Events Report for additional details (title format, visible columns & buttons, etc.)
