6.3.1.1. Annual Summits Report

*Documented 

  


Add the following new report: 

  


This is a filtered version of the the Master Events Report that displays only Annual Summits.

  


This is accessed from:

  * Admin | Event Management | Annual Summits (bypasses the filter screen to open the report directly)



  


The following filters are set to something other than their default:

  * Event Type = Annual Summit



  


Other Notes: 

  * See the spec for the Master Events Report for additional details (title format, visible columns & buttons, etc.)
