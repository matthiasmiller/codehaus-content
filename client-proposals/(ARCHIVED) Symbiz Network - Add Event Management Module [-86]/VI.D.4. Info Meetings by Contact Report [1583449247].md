6.4.4. Info Meetings by Contact Report

*Documented 

  


This is a filtered version of the the Event Registrations Report that displays Info Meeting Registrations for one Contact.

  


This is accessed from:

  * "<#> Info Meetings" link in the Lead Details section of the Contact record



  


The following filters are set to something other than their default:

  * Event Type = "Info Meeting" 
  * Registrant = corresponding Contact



  


Other Notes:

  * See the spec for the Event Registrations Report for additional details (title format, visible columns & buttons, etc.)
