6.3.1.4. Launch Meetings Report

*Documented 

  


Add the following new report: 

  


This is a filtered version of the the Events Report that displays only Launch Meetings.

  


This is accessed from:

  * Admin | Event Management | Launch Meetings (bypasses the filter screen to open the report directly)



  


The following filters are set to something other than their default:

  * Event Type = Launch Meeting



  


Other Notes: 

  * See the spec for the Master Events Report for additional details (title format, visible columns & buttons, etc.)
