3.2. Sales Curve vs Actual

Add a Sales Curve vs Actual report

  


Prompts:

  * Sales Rep (blank = all)
  * Dealer (blank = all)
  * District (blank = all)
  * Group by dealer (checkbox)
  * Filter by Year (required number)



  


Group by:

  * Dealer (if "Group by dealer")



  


Columns:

  * Week #
  * Weight (only visible when filtered by sales rep)
  * Goal $
  * Actual $
  * Actual Rolling Avg Weight



  


NOTE:

  * This would be a totals and subtotals only report. If it's easier to build and test, add an ask prompt that shows all the individual rows.
  * Simply sum the goal $ from all the included goals
  * Sum the actual sales amount from each week
