9.7. Export Bill of Lading Printout

  


Requirements

Purpose/Overview: This is a customized PDF printout that is given to the truck driver who hauls a load away from the sawmill. It includes basic information about an individual Export Tally and some details about the corresponding Booking. 

  


Printed From:

  * Export Tally Record
  * Yard App (option to open the PDF on the mobile device so it can be printed directly from the device; see corresponding section of the proposal) 



  


File Format/Name: "Bill of Lading - <Customer Name> \- <Export Tally ID> \- <current date in the MM-DD-YYYY format>"

  


Fields to be Filled: 

  * Export Tally ID: 
  * Booking #: 
  * Early Return Date: 
  * Cutoff Date: 
  * Description: 
  * Loaded Date: 
  * Container #: 
  * Heavy Weight: 
  * Light Weight: 
  * Seal #: 



  


Template: See mockup

  


Handling Page Breaks: N/A (this would always fit on one page, so no need for page breaks or page numbers.) 

  


Other Notes:

  * N/A
