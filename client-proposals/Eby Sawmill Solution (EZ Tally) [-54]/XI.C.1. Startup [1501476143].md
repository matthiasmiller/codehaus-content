11.3.1. Startup

Login: 

This screen displays the following label & link: "You are currently logged out. Log in". 

  


Clicking the "Log in" link opens the standard Google Account login page, where the user can select an account from a list or add a new account. Note that a user must have a Google account linked to the main Solution and use that same account to log into the Yard App. 

  


Logging in opens the Home screen. 

  


Home screen: 

This is the main menu page for the app. 

  


There will be prominent buttons for the following:

  * Yard Tally Scaling
  * Export Scaling
  * Export Loading
