6.1.3. Configure Log Conditions Report

  


Requirements

Overview: This is a report of all Log Condition records.

  


Accessed from: Advanced | Configuration | Configure Log Conditions 

  


Title: Configure Log Conditions

  


Columns: 

  * Description (link to record) 
  * Hotkey



  


Grouped by: Active (at the top), Inactive (at the bottom in gray)

  


Sorted by: Description

  


Filters: 

  * N/A



  


Buttons: 

  * New Log Condition (opens a blank Log Condition record)



  


Menu Visibility: Administrator only (menu option grayed-out for non-admins)

  


Other Notes: 

  * N/A
