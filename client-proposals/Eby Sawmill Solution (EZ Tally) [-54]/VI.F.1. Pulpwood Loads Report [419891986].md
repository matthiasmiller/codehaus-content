6.6.1. Pulpwood Loads Report

  


Requirements

Overview: This is a report of all Pulpwood Load records.

  


Accessed from: Main | Tracts | Pulpwood Loads (bypasses the filter screen to open the report directly). 

  


Title: Pulpwood Loads

  


Columns: 

  * Load ID (link to record)
  * Logger
  * Date Processed
  * Location
  * Tons
  * Amount
  * Logger Paid (Yes/No)
  * Landowner Paid (Yes/No)
  * Tract (link to record)



  


Grouped by: Status (Unpaid at the top)

  


Sorted by: Date Processed (furthest date at the top)

  


Filters: 

  * Logger (drop list of all Loggers)
  * Date Processed (date range)
  * Location (drop list of Locations)
  * Amount (number range)
  * Tract (drop list of Tract IDs)



  


Buttons: 

  * New Pulpwood Load (opens blank new record)



  


Menu Visibility: Users with the View Tracts, Yard Tallies, and Pulpwood Loads permission (menu option grayed-out for other users)
