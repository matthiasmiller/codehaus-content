6.1.2. Configure Species Report

  


Requirements

Overview: This is a report of Species records.

  


Accessed from: Advanced | Configuration | Configure Species

  


Title: Configure Species

  


Columns: 

  * Description (link to record)
  * Abbreviation
  * Hotkey



  


Grouped by: Active (at the top), Inactive (at the bottom in gray) 

  


Sorted by: Description 

  


Filters: N/A

  


Buttons: 

  * New Species (opens a blank new Species record) 



  


Menu Visibility: Administrator only (menu option grayed-out for non-admins)

  


Other Notes:

  * N/A
