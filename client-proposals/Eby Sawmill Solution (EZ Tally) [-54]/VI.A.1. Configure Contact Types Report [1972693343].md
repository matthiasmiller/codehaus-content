6.1.1. Configure Contact Types Report

  


Requirements

Overview: The Solution will include this standard report to view and manage Contact Types.

  


This is a report of the Contact Types in the Solution. By default it would only show active items.

  


Accessed from: Advanced | Configuration |Configure Contact Types

  


Title: Contact Types

  


Columns: 

  * Type (link to record)
  * Individual (Yes/No)
  * Organization (Yes/No)



  


Grouped by: Active/Inactive (active at the top)

  


Sorted by: Type name

  


Filters: 

  * Show inactive contact types (checkbox; defaulting to not filled)



  


Buttons: 

  * New Contact Type



  


Menu Visibility: Administrator only (menu option grayed-out for non-admins) (note that this is standard) 

  


Other Notes: 

  * N/A
