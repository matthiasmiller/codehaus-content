20.5. Export Tally Loading

  


Requirements

Separate process for loading:

  * Available list of bookings
    * Select the booking number they're working on
    * Select the load that they're working on (load # / tally #)
  * Entry
    * Any Logs Removed?
      * Yes -> Go into the Tally, and remove logs
      * No -> Continue
    * Container #
    * Heavy Weight
    * Light Weight
    * Net Weight (calculated)
    * Picture of Load (They use a picture for people overseas who've been cheated, and been accused of having stolen containers.)
    * Seal #
    * Complete (flags Date Loaded)



  


Tag # can autocount across all export tallies, but be editable.
