6.6.3. Pulpwood Summary Report

  


Requirements

Overview: This is a report that provides a summary of Pulpwood Tons and $ amount from a selected Tract (or all Tracts) on a per-Location basis, with one row for each Location.

  


Accessed from: Financial | Misc. Reports | Pulpwood Summary (bypasses the filter screen to open the report directly). 

  


Title: Pulpwood Summary 

  


Columns: 

  * Location 
  * Number of Loads (displays the number of Pulpwood Load records for the Location within the parameters of the other selected filters)
  * Tons (number to 2 decimals)
  * Amount (number to 2 decimals)
  * Freight Expense (number to 2 decimals; Tons * the Location's Freight/Ton rate)



  


Grouped by: N/A

  


Sorted by: Location

  


Filters: 

  * Tract (drop list of all Tracts; filters down as you type; defaults to blank = all)
  * Date Processed Start (looks at Date Processed on the Pulpwood Load records; defaults to the first date of the current month) 
  * Date Processed End (looks at Date Processed on the Pulpwood Load records; defaults to blank = all) 
  * Location (drop list of all Pulpwood Locations; defaults to blank = all)



  


Buttons: N/A

  


Menu Visibility: Users with the View and Edit Financials permission (by virtue of it being on the Financial Menu)

  


Other Notes:

  * N/A
