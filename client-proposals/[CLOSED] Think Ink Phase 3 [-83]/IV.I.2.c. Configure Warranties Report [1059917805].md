4.9.2.3. Configure Warranties Report

  


Requirements

This is a report of all Warranties in the Solution (all Sales Items with the Warranty category). While the Warranties can be managed from the main Manage Sales Items report, this is a separate menu option and custom report to make it easier for users.

  


This is accessed from Advanced | Configuration | Configure Device Warranties.

  


Title: Configure Warranties
