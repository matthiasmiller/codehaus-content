4.9.1.1. Configure Device Tags Report

  


Requirements

This is a report of all Device Tags.

  


This is accessed from Advanced | Configuration | Configure Device Tags. 

  


Title: Device Tags

  


Columns: 

  * Tag (link to record) 
  * Device Types (comma-separated list of linked Types; text wraps if necessary) 



  


Grouped by: Active / Inactive

  


Sorted by: Tag (alphabetical)

  


Filters: 

  * Show Inactive Tags (checkbox; defaults to not checked)



  


Buttons: 

  * New Device Tag (opens new blank record)



  


Menu Visibility: 

  * Grayed-out for non-admins



  


Other Notes:

  * N/A
