6.3.1. AppHosting.Zone Settings

  


Requirements

The Solution would include the following settings on the AppHosting.Zone Settings page: 

  


  * Managed Service Agreement section: 
    * Agreement Text (memo; supports expressions; used to generate the Managed Service Agreement printout) 



  


Data Access:

  * All users can view, but only admins can edit (this is the standard setting)
