4.9.3.5. Available Devices Report

  


Requirements

This is a report of all Devices that are available to go to a new customer (any Device that has a Device Status of Available). This is the All Devices report with the Status filter set to "Available".

  


This would be accessed from Home | Devices | Available Devices.
