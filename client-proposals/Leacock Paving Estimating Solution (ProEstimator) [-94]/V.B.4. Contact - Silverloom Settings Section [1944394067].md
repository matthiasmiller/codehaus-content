5.2.4. Contact - Silverloom Settings Section

  


Requirements

Silverloom Settings section (standard section; __

  


  * User ID (standard field; drop list of active User IDs that are not already linked to a Contact record; custom: visible if "Contact Type" includes "Employee"; custom warning on Save if visible and blank: "This employee is not linked to a User Profile.") 
  * Power App Contact ID (custom field; visible if not blank; optional; number; 0 decimals; set via import (data migration); used to identify imported Contacts and link them with imported Proposals, etc., so would be blank for manually-entered Contacts)
