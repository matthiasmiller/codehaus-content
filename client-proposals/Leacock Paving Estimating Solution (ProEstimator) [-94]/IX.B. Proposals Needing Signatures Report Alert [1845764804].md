9.2. Proposals Needing Signatures Report Alert

  


Requirements

Overview/Purpose: This is a report alert that serves as a reminder to Sales Reps about Awarded Proposals needing a documented signature.

  


Details: 

  * Title: "<#> Proposal(s) Needing Signatures" (with "#" = the number of corresponding records)
  * Displays: If there are any Proposals that meet the following criteria:
    * "Status" = "Awarded" or "Job Scheduled" and 
    * "Proposal Signed" = not checked / blank and 
    * the number of days since "Proposal Result Date" is greater than or equal to "7"
  * Action: Opens the "Proposals Needing Signatures" report (see corresponding spec)
  * User(s) to notify: All Sales Reps with one or more Proposals that meeting the criteria
  * Dismiss type: Auto-dismiss when all items are resolved
