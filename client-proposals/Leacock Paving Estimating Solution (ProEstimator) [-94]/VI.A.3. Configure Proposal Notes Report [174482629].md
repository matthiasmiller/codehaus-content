6.1.3. Configure Proposal Notes Report

  


Requirements

Overview: This is an editable custom report of all Proposal Note records in the database.

  


Accessed from: Advanced | Configuration | Configure Proposal Notes

  


Title: Proposal Notes

  


Columns: 

  * Sort Order (editable) 
  * Note Text (link to open the record)



  


Grouped by: Active/Inactive

  


Sorted by: 

  * First by: Sort Order 
  * Second by: Note Text (alphabetical)



  


Filters: 

  * N/A



  


Buttons: 

  * New Proposal Notes (opens blank new record)



  


Menu Visibility: All users

  


Save Type: Save Button

  


Live Updating: No

  


Other Notes:

  * N/A
