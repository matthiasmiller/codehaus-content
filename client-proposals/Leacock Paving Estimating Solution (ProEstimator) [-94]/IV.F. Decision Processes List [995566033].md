4.6. Decision Processes List

  


Requirements

This is a custom non-editable simple list used to track important factors for the Customer in the decision-making process. It includes the following items: 

  


  * Quality (existing item in Power App solution)
  * Availability (existing item in Power App solution)
  * Price (new item in the ProEstimator solution)
  * Other (new item in the ProEstimator solution)



  


This list is universal for all users.

  


Note that this sequence is static, so it should always be displayed in this sequence throughout the Solution.

  


This list will be set up by CodeCrafters as part of the main development. If changes need to be made to this list, it will require some coding work.
