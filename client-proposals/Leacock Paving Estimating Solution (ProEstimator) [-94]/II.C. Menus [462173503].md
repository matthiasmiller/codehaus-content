2.3. Menus

  


Requirements

Reports and features will be accessed via the Silverloom Business Suite menu.

  


ProEstimator includes the following menu tabs, with the listed menu options in addition to other options specced in their corresponding record / report specs (any menu options noted here are ones that are not specced out in other sections of this spec): 

  


  * Main Menu (custom menu; visible to all users; some menu options can be disabled or hidden based on Permissions, User Groups, etc.) 



  


  * Advanced (standard menu; visible to all users; some menu options can be disabled or hidden based on Permissions, User Groups, etc.) 
    * Advanced | Configuration | View / Update PA Oil Index (opens the "PA Oil Index Child Screen" from Silverloom Settings in its own tab, where it can be viewed or updated)
