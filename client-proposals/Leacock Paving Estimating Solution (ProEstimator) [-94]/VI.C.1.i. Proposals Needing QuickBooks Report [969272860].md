6.3.1.9. Proposals Needing QuickBooks Report

Overview: This is the All Proposals Report, filtered to display Proposals that need to be entered into QuickBooks.

  


Accessed from:

  * The "Reminder to Enter Proposal(s) into QuickBooks" email notification



  


The following filter(s) are set differently from the main report:

  * "Report View" = "Proposals Needing QuickBooks"
