6.3.1.5. My Proposals Report

  


Requirements

Overview: This is the "Master Proposals Report", filtered to display items for a single Sales Rep.

  


Accessed from:

  * Main Menu | Proposals | My Proposals (opens the report directly, bypassing the filter screen)



  


The following filter(s) are set differently than on the main report:

  * "Status": Set to the following items checked:
    * "Estimating in Progress"
    * "Ready to Send"
    * "Follow-Up Needed"
    * "Pending"
    * "Awarded"
  * "Sales Rep": Set to the Contact for the current user 



  


Other Notes:

  * This report (like all others) can be printed from the drop-down advanced menu in the top left-hand corner of the screen, by selecting "PDF" or "Excel".
