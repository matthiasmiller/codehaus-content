4.3. Units of Measure List

  


Requirements

This is a custom non-editable simple list used to track units of measure for Sales Items and Groups (both Automated Proposal Groups and Groups added to Proposals). 

  


It includes the following items:

  


  * CY (abbreviation for cubic yard) 
  * Day
  * HR (abbreviation for hour) 
  * IN (abbreviation for inch) 
  * LF (abbreviation for linear foot)
  * SF (abbreviation for square foot)
  * SY (abbreviation for square yard)
  * Ton
  * Unit



  


This list is universal for all users.

  


This list is displayed in alphabetical order throughout the Solution.

  


This list will be set up by CodeCrafters as part of the main development. If changes need to be made to this list, it will require some coding work.
