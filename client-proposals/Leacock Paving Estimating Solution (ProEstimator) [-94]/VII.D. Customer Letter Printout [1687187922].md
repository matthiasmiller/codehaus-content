7.4. Customer Letter Printout

  


Requirements

Purpose/Overview: This letter may be included with Proposals (print or email) if desired, and provides information to the Customer regarding care for their newly-applied asphalt. The text is configured in Silverloom Settings. 

  


Printed From: 

  * Main Menu | Printing | Print Customer Letter
  * A parallel version of this printout (coded separately) is conditionally included as the last page on the Proposal Printout, depending on how it is generated. Changes to this main printout should be reflected on that page. 



  


File Format/Name: "Customer Letter.pdf"

  


Fields to be Filled: 

  * Customer Letter Text: Displays the contents of the "Customer Letter Text" memo from Silverloom Settings



  


Handling Page Breaks: If multiple pages are needed, may break at any point between lines of text. 

  


Other Notes:

  * Includes the default "Powered by Silverloom Business Suite" footnote at the bottom of the page. 
  * Note that printout templates may be subject to limitations of the PDF templating library. CCI will communicate with the client in places where the template library requires significant changes from the original design.
