An optional dashboard can be provided to show a graphic interface of key metrics and calls to action. Preliminary / example design is included below for various features that could be included on this dashboard. If desired, this feature could be handled as a separate or future item. 

  


Possible Charts / Graphs:

  


  * Proposals by Timeframe:  
    * Vertical bar graph or line graph
    * Options for various timeframes (Month, Week, Previous Year, etc.) 
    * Options for various Proposal data (# created, # signed, $ value, etc.) 



  


  * Proposals by Sales Rep: 
    * Pie graph 
    * Shows a "slice" for each Sales Rep, based on # of Proposals or total $ value
    * Different color for each Sales Rep 
    * Could display results month-to-date, year-to-date, or for some other date range



  


  * Proposals by Job Type: 
    * Pie graph 
    * Shows a "slice" for each Job Type, based on # of Proposals or total $ value
    * Different color for each Job Type
    * Could display results month-to-date, year-to-date, or for some other date range
