3.5.2. System Switches - Custom

The Solution includes the following custom System Switches:

  


  * Label: "Minimum Proposal Numeric ID"   
    * Code: Config_MinimumProposalNumID
    * Field Type: Number (no decimals) 
    * Description: "Specifies the minimum "Numeric ID" value for Proposal #'s"
    * Value / set to: "10000" (to be set, probably to a number in the 15000's at deployment) 
    * Other Notes: 
      * Editable
      * Validation: Cannot be set to a negative number. 



  


  * Label: "Proposal eSignature URL"
    * Code: "ProposalESignatureURL"
    * Field Type: Plain text
    * Description: "Specifies the URL for the eSignature provider for Proposals."
    * Value / set to: [https://esignatures.com/](https://esignatures.com/)
    * Other Notes: 
      * Editable
