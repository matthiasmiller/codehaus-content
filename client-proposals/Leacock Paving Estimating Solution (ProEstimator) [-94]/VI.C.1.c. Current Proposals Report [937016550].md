6.3.1.3. Current Proposals Report

  


Requirements

Overview: This is the "Master Proposals Report", filtered to include only Proposals that are active (i.e. not leads or archived / scheduled / etc.).

  


Accessed from:

  * Main Menu | Proposals | Current Proposals (opens the report directly, bypassing the filter screen)



  


The following filter(s) are set differently than on the main report:

  * "Status": Set to the following items checked:
    * "Estimating in Progress"
    * "Ready to Send"
    * "Follow-Up Needed"
    * "Pending"
    * "Awarded" 



  


Other Notes:

  * This report (like all others) can be printed from the drop-down advanced menu in the top left-hand corner of the screen, by selecting "PDF" or "Excel".
