4.8. Default Sections List

  


Requirements

This is a custom editable simple list used to track default Section Names for Proposals. 

  


Example list items may include: 

  


  * Basketball Court Paving
  * Curb Patching
  * D-Zones
  * Driveway Paving
  * Fine Grading
  * Full Depth Paving
  * Heavy Duty Paving
  * High Jump
  * HOP Paving
  * Javelin
  * Joint and Curb Sealing
  * Light Duty Paving
  * Long Jump
  * Mill and Overlay
  * Parking Lot Paving
  * Patch Paving
  * Pickleball Court Paving
  * Pole Vault
  * Porous Paving
  * Prep
  * Tennis Court Paving
  * Township Paving
  * Track Oval
  * Trench Patching
  * Walkway Paving



  


This list is universal for all users, and can be editing via the "Lists" interface (accessed from Advanced | General | Lists).

  


This list is displayed in alphabetical order throughout the Solution. 

  


The initial items for this list are to be set up by Leacock Paving and/or CodeCrafters at deployment, based on the above items.
