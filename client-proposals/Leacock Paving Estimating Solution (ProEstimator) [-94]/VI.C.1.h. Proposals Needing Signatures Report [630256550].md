6.3.1.8. Proposals Needing Signatures Report

  


Requirements

Overview: This is the All Proposals Report, filtered to display Proposals that need a signature documented.

  


Accessed from:

  * Main Menu | Proposals | Proposals Needing Signatures (opens the filter screen, with the filters defaulted as noted below)
  * The "Proposals Needing Signatures" report alert



  


The following filter(s) are set differently from the main report:

  * "Status" = "Awarded" and "Job Selected"
  * "Report View" = "Proposals Needing Signatures"
