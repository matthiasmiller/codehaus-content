6.1.5. Configure Automated Proposal Groups Report

  


Requirements

Overview: This is a custom report of all Automated Proposal Group records in the database.

  


Accessed from: Advanced | Configuration | Configure Automated Proposal Groups

  


Title: Automated Proposal Groups

  


Columns: 

  * Name (link to open the record)
  * Unit of Measure 
  * Uses and Requires Thickness



  


Grouped by: Active/Inactive

  


Sorted by: Name (alphabetical)

  


Filters: 

  * N/A



  


Buttons: 

  * New Automated Proposal Group (opens blank new record)



  


Menu Visibility: All users
