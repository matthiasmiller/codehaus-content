2.2. Changes to Silverloom Settings

The following changes are included for Silverloom Settings:

  


New settings:

  * PA 1 Call Turnaround Window (this is the number of days required for buffer/lead time when requesting PA 1 Call services; currently 3 days)
  * PA 1 Call Expiration Window (this is the number of days for which the PA 1 Call certification is valid for a specific sit; currently 10 days)
