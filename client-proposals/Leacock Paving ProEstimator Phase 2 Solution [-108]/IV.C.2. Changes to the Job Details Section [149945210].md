4.3.2. Changes to the Job Details Section

  


Requirements

This captures whether prevailing wage applies to the project and, if so, the specific rates by labor type--Equipment Operator, Laborer, Truck Driver--which vary by county, state, and sometimes by project.

  


This includes the following new items: 

  


  * Prevailing Wage Applies (checkbox; this designates whether this project is subject to prevailing wage requirements)
  * Prevailing Wage Details (no label; visible if "Prevailing Wage Applies" checkbox is checked; embedded spreadsheet of the following:
    * Columns: 
      * Prevailing Wage Type (drop list of "Prevailing Wage Types" list items; required)
      * Prevailing Wage Rate (number; 2 decimals; required)
