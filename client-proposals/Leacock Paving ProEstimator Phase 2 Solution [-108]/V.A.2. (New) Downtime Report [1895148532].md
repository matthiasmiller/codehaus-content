5.1.2. (New) Downtime Report

  


Requirements

This is a report of downtime, based on Calendar Event records, with various filters.

  


  * Columns
    * Date
    * Crew
    * Reason
    * Customer Name
    * Job Name
    * Job #
    * Start Time
    * End Time
    * Duration (total row shows sum)



  


  


  * Filters:
    * Date Range
    * Reason (drop list)
    * Group by (drop list of:
      * Customer
      * Crew
      * Reason



  


Additional details to be determined in the In-Depth Design.
