This is a report of all Calendar Events, with various filters. 

  


  * Columns:
    * Event Type
    * Start Date
    * End Date
    * Employee (visible if "Event Type" = "Time Off")
    * Job # (visible if "Event Type" = "Job")
    * Crew
    * Title (visible if "Event Type" = "Other"



  


  * Filters:
    * Event Type (drop list)
    * Start Date
    * End Date
    * Job
    * Crew



  


Additional details to be determined in the In-Depth Design.
