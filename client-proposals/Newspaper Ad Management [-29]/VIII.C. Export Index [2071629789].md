8.3. Export Index

Clicking on the Export Index menu option should open a page with the following prompts:

  * Publication (required, list of publications, default to blank; for Publication Admin and Standard Users, show only their assigned publication, for Full Admin show all)
  * Publication Date (the date should default to the current Publication Date)



  


Clicking Continue exports an Excel file of the Index Report for the selected Publication Date of the selected Publication.

  


Any user can run this export.
