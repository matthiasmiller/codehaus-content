13.1. Importing Data

CodeCrafters/CodeHaus will be importing contact information for customers from the existing system. This will be done based on an Excel file or files provided by PNP. The following details should be imported and should be included in the file(s) provided by PNP: 

  * Company / Customer Name
  * Primary Contact First Name 
  * Primary Contact Last Name 
  * Address Line 1 
  * Address Line 2
  * City 
  * State
  * Zip
  * Primary Phone 
  * Secondary Phone 
  * Email 
  * Fax 



  


  


No ads data will be imported. PNP will manually transfer over ads information as needed.
