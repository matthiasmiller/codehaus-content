3.3.1.1. Contact Types Report

  


Requirements

This is a report of the Contact Types in the Solution. By default it would only show active items.

  


This would be accessed from a "Configure Contact Types" menu option in the Configuration section of the Advanced menu.

  


Title: Contact Types

  


Columns: 

  * Type (link to record)
  * Individual (Yes/No)
  * Organization (Yes/No)



  


Grouped by: Active/Inactive (active at the top)

  


Sorted by: Type name

  


Filters: 

  * Show inactive contact types (checkbox; defaulting to not filled)



  


Buttons: 

  * New Contact Type



  


Menu Visibility: Admin only (menu option is grayed out for other users) 

  


Other Notes: N/A
