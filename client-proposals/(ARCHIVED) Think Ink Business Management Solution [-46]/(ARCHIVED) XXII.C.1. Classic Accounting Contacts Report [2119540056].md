22.3.1. Classic Accounting Contacts Report

  


Requirements

This is a report of Classic Accounting Contact records.

  


This would be accessed from a "Classic Accounting Contacts" menu option in the "Classic Accounting" section on the Advanced menu.

  


Title: Classic Accounting Contacts

  


Columns: 

  * ID (link to record)
  * Org Name
  * First Name
  * Middle Name
  * Last Name
  * Email
  * Phone 1
  * Phone 2



  


Grouped by: 

  


Sorted by: 

  * ID



  


Filters: 

  * ID (blank = all)



  


Menu Visibility: Admin Only

  


Other Notes: N/A
