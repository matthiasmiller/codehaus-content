7.2. Manage Regions Report

  


Requirements

This is a two-pane report of Region records (left pane) and the Customers included in a given Region (right pane), with various filtering options.

  


This is accessed from Home | Regions | Manage Regions. 

  


Title: Regions

  


Left Pane (displays Regions, based on the selected filter criteria)

Columns: 

  * Region (link to record)
  * Region Notes
  * Sales Rep
  * Check-In Schedule (display the text such as "Every January, April, July, and October"; text wraps if needs)
  * Next Check-In (display the month of the next scheduled Check-In) 



  


Grouped by: Active / Inactive

  


Sorted by: Region (alphabetical)

  


Right Pane (displays the Customers by Region report, filtered to the selected Region in the left pane)

  


Filters (all apply directly to the left pane): 

  * Region (drop list of Regions; defaults to blank = all)
  * Sales Rep (drop list of Employees; defaults to blank = all)
  * Next Check-In Start (drop list of Months, in the format of "<Month> <Year>", starting with the current month and going 12 more months into the future; default to blank = all)
  * Next Check-In End (drop list of Months, in the format of "<Month> <Year>", starting with the current month and going 12 more months into the future; default to blank = all); if a range is selected, the report displays only the Regions whose Next Check-In Month falls inside the range)



  


Buttons: 

  * New Region (opens blank new Region record)



  


Menu Visibility: All users

  


Other Notes:

  * N/A



  


BID 1 day
