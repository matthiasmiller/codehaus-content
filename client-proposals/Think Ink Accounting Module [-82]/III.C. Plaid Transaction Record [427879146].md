3.3. Plaid Transaction Record

The Solution have a Plaid Transaction record. These would only be editable via imports. They would have:

  * ID (autonumber)
  * Plaid ID (37 alphanumeric, case-sensitive ID)
  * Date
  * Description
  * Category (user customizable list of Plaid Categories)
  * Account (user customizable list of Plaid Accounts)
