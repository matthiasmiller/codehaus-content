2.2. Changes for Authorized Signature & Logo

  


Requirements

*Documentation N/A

  


Authorized Signature:

  * A Plan-specific authorized signature image file (.jpg or .png) is stored in the "Authorized Signature" memo in Silverloom Settings, to be automatically added to printouts as needed. This may be uploaded by the Plan or by CCI.
  * The existing hard-coding of signature images on printouts is being replaced by displaying the image in "Authorized Signature". 



  


Logo:

  * A Plan-specific logo image file (.jpg or .png)  is stored in the "Plan Logo" memo in Silverloom Settings, to be automatically added to printouts as needed. This may be uploaded by the Plan or by CCI.
  * The existing hard-coding of logos on printouts is being replaced by displaying displaying the image in "Plan Logo".
