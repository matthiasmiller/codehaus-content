4.3. View Households

  


Requirements

The View Households should filter by:

  * Household #
  * Head of Household Last Name
  * Head of Household First Name
  * Head of Household Middle Name
  * Spouse Last/Middle Name
  * Spouse First Name
  * Spouse Middle Name
  * Child First Name
  * Child Middle Name
  * Church #
  * Book #
  * City
  * State (drop list)
  * Zip
  * Households needing followup (checkbox)



  


The Child prompts will only match children within the household. The Child columns will be hidden unless searching for a Child.

  


It should show columns for each of the prompts.

  


It should group by Active Households, then by Inactive Households.
