4.3.5. NG Location Record

Overview: This record is a read-only copy of the data from the NexGen Location record.

  


This is a custom list-tied record that's keyed off the NexGen key (externalRef).

  


Read-Only NG Fields:

  * externalRef (UID)
  * Name
  * Description
  * Directions
  * Address 1
  * Address 2
  * City
  * State/Province
  * Country
  * Customers (RG of externalRefs)
