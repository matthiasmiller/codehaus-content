4.4.4. Commissions

Overview: Commissions are based off of weight sold. Salespeople set quarterly commission goals. This report allows each salesperson to see their pounds sold, and commission $. This might also show the salesperson to see their goals for the quarter, broken down by week, etc.Â 

  


Prompt for:

  * Start Date
  * End Date
  * Salesperson (blank = all; only current user unless permission to view all salespeople commissions)
  * Show order details



  


Group by:

  * Salesperson
  * Commission Class



  


Columns:

  * Customer Name
  * Load Date
  * Pounds (totaled)
  * Commission $ (totaled)



  


Sort by

  * Load Date



  


Filter by:

  * Load Date



  


NOTE:

  * Salespeople will only be able to view their own commissions.
  * "Show order details" will show full detail. Otherwise, it will only show totals grouped by commission class.
