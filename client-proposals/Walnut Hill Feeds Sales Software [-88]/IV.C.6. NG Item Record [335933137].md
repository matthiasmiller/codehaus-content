4.3.6. NG Item Record

Overview: This record is a copy of the data from the NexGen Item record. This is mostly read-only, but it includes an editable Silverloom-specific section.

  


This record tracks both Ingredients and Formulas (note that Formulas might be stored separately from Ingredients).

  


This is a custom list-tied record that's keyed off the NexGen key (externalRef).

  


Read-Only NG Fields:

  * externalRef (UID)
  * Type (list of types)
  * Name
  * Description



  


Editable Silverloom Fields:

  * Custom Commission ($ / ton)
