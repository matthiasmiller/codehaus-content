1.2. Workflow Overview

Data Synchronization Workflow:

  * Data from QuickBooks and NexGen will be synchronized nightly. The data will be uploaded to Silverloom using CodeCrafters-provided automation software running locally on Walnut Hill's server.



  


Salesperson Workflow:

  * Salespeople will be able to view customer information including the following:
    * Current Balance and Invoices
    * Customer-specific notes
    * Past orders (including formula information)
  * Salespeople will be able to view their own commission details. This includes quarterly sales goals and progress as well as individual sales.



  


Route Management:

There are two options for Route Management:

  * Basic Option:
    * A report that allows for route optimization and planning, using simple as-the-crow-flies distance calculation.
  * Advanced Option:
    * A map interface that uses colored pins and map filters to provide a more intuitive and detailed route planning system.
