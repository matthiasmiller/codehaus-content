5.8. Truss Date Report

  


Requirements

Overview: This is an editable report of Production Lines. This report is used to track the available Board Feet per day for each Production Line. The report will automatically refresh as Users enter data to ensure that it is always showing the most recent information.

  


Accessed from: 

  * Quotes & Orders | Truss Date



  


Title: Truss Date

  


Columns: 

  * Context Menu (displays as a blank column; contains the context menu with options to Enter Board Feet Sold and Discard Changes to Record)
  * Production Line (link to record)
  * Capacity 
  * Truss Date 
  * Month (displays the name of the month from the Truss Date for the corresponding Production Line)
  * Available Board Feet
  * Notes (wrapped text with formatting)
  * Last Modified (date and time)



  


Grouped by: N/A

  


Sorted by: Production Line

  


Filters: 

  * N/A



  


Buttons: 

  * New Production Line (opens a blank new "Production Line" record)



  


Menu Visibility: 

  * All users



  


Save Type: Automatic

  


Live Updating: Yes

  


Other Notes:

  * Note that this report automatically refreshes as Users enter data. 
  * To avoid editing collisions, Board Feet adjustments are entered in a child screen.
