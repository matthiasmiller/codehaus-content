4.3.2. Customer section

Customer section:

  * Customer Name (optional; drop-list of Customer type contacts; when the screen is in non-edit mode, links to the record)
  * Customer Phone (read-only; displays the phone number in the top row of the Phone RG in the Customer's Contact record)
  * Job Name (optional; plain text field)
  * Job Phone (optional; plain text field; if a proper 10 digit phone number is entered, the number is auto-formatted)
  * Travel Time (optional; number field allowing 2 decimal places)
    * Located to the right of Job Phone 
  * Delivery Address (optional; plain text field)
  * Delivery City (optional; plain text field)
  * Delivery State (optional; drop-list of states; warning on Save if blank; warning on Save if the selected State has no tax rate specified)
  * Delivery Zip (optional; plain text field)
  * Delivery / Pickup Notes (optional; memo)
  * Delivery / Pickup Date (date; required if Status is set to Order)
  * Delivery / Pickup Time (optional; plain text field)
  * Lumber Delivery / Pickup Date (required; date field; visible and editable if Lumber Order and Truss Order or Metal Order are checked; becomes read-only if it is filled in and in the past)
  * Truss Delivery (label; this label and the checkboxes below it are grayed out if Truss Order is not checked)
    * Set (optional; checkbox; editable if Truss Order is checked)
    * Drop (optional; checkbox; editable if Truss Order is checked)
    * Roll Off (optional; checkbox; editable if Truss Order is checked)
    * Small Truck (optional; checkbox; editable if Truss Order is checked)
  * Lumber Delivery (label; this label and the checkboxes below it are grayed out if Lumber Order is not checked)
    * Moffett (optional; checkbox; editable if Lumber Order is checked)
    * Customer Unload (optional; checkbox; editable if Lumber Order is checked)
    * Multiple Loads (optional; checkbox; editable if Lumber Order is checked)
    * Roll Off (optional; checkbox; editable if Lumber Order is checked)
  * Metal Delivery (label; this label and the checkboxes below it are grayed out if Metal Order is not checked)
    * Moffett (optional; checkbox; editable if Metal Order is checked)
    * Customer Unload (optional; checkbox; editable if Metal Order is checked)
    * Sent with Trusses (optional; checkbox; editable if Metal Order is checked)
    * Straight Ship (optional; checkbox; editable if Metal Order is checked)
    * Straight Ship with Moffett (optional; checkbox; editable if Metal Order is checked)
    * Roll Off (optional; checkbox; editable if Metal Order is checked)
