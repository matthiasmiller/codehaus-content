8.3.2. Lumber Packing List Page

Lumber Packing List Page (visible if the "Lumber Order" checkbox is checked on the corresponding record):

  * Title: 
    * If printout is generated from a "Standard Packing List" button: "Lumber Packing List";
    * If printout is generated from a "Yard Packing List" button: "Yard Guys - Lumber Packing List" (displayed with a gray background)
  * Delivery / Pickup
  * Date (displays the Delivery / Pickup Date)
  * Time (displays the Delivery / Pickup Time)
  * Lumber Delivery Pickup
  * Date (displays the Lumber Delivery / Pickup Date)
  * Spacing
  * Res / Comm (checkbox; checked if Residential / Commercial checkbox is checked)
  * Ag (checkbox; checked if Agricultural checkbox is checked)
  * Loading GSL (displays the Loading GSL field from the Order Summary section of the record)
  * TCLL (displays the TCLL field from the Order Summary section of the record) 
  * TCDL (displays the TCDL field from the Order Summary section of the record) 
  * BCDL (displays the BCDL field from the Order Summary section of the record) 
  * Attic Loading (displays the Attic Loading record from the Order Summary section of the record)
  * Unlabeled table of the following (show 6 rows by default, even if there are blank rows; if printout is generated from a "Yard Packing List" button, this table is sorted numerically by Yard Sort Order, with items that have no Yard Sort Order at the bottom):
    * Quantity (displays the Qty column from the embedded spreadsheet in the Lumber section of the record)
    * Description (displays "<Feet>',<Inches>", - <Description>" from the embedded spreadsheet in the Lumber section of the record)
    * Unit Weight (displays the Unit Weight column from the embedded spreadsheet in the Lumber section of the record)
