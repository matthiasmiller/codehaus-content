5.3.2. Truss Sales

TODO_AP: Ben Reitz 11/14/2025: Standardize

  


This is a report of truss orders within the specified date range. It contains one row for each row in the Truss embedded spreadsheet on the order.

  


This is accessed from Triple D Truss | Sales Items | Truss Sales. 

  


Title: Truss Sales #Start Date# through #End Date#

  


Columns: 

  * Order Date
  * Span Ft
  * Span In
  * Pitch
  * Left O.H.
  * Right O.H.
  * Left Cant.
  * Right Cant.
  * Description
  * Unit Price
  * Quantity (sum total)
  * Price (sum total)



  


Grouped by: Order ID

  


Sorted by: Order ID

  


Filters: 

  * Start Date
  * End Date
  * Span Feet
  * Span Inches
  * Totals Only (checkbox)



  


Other Notes:

  * Checking Totals Only displays one row for each invoice in the time period, with quantity and price totaled for all rows in the spreadsheet on the order record.
