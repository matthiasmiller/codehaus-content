5.5.3. Edit Inventory

This is a report of all SKU lots.

  


This is accessed from Triple D Truss | Inventory | Edit Inventory.

  


Title: Edit Inventory

  


Columns: 

  * SKU
  * Category
  * Description
  * Unit Cost (editable column)
  * Profit Margin %
  * Sales Price
  * UOM
  * Unit Weight



  


Grouped by: Active/Inactive

  


Sorted by: SKU

  


Buttons: 

  * Save Changes



  


Save Type (for editable reports): Save Button
