Purpose/Overview: This is a PDF export of non-Starlight deliveries for the specified week that have the "Scheduled" or the "Will Call" checkbox checked.

  


Printed From: Triple D Truss | Deliveries | Main Production Schedule Printout

  


Filters: 

  * Week (blank = Will Call)  (date)
  * Minimum Daily Entries



  


The PDF displays a table with columns for Day, Customer, Item, Board Feet, Job, Cut, and Made. There are groupings for Monday - Friday, with the number of rows entered in Minimum Daily Entries.
