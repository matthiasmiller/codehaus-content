5.4.1. Quotes and Orders

TODO_AP: Ben Reitz 11/14/2025: Standardize

  


This is a report of quotes and orders.

  


This is accessed from Triple D Truss | Quotes & Orders | Search Quotes & Orders. 

  


Title: 

If Filter Date is Recent and Recent Days is blank: Search Quotes & Orders

If Filter Date is Recent and Recent Days is set: Quotes & Orders (#Recent Days# Days)

If Filter Date is Date Range and the dates are blank: Quotes & Orders

If Filter Date is Date Range and Start Date is set but End Date is blank: Quotes & Orders from #Start Date#

If Filter Date is Date Range and End Date is set but Start Date is blank: Quotes & Orders as of #End Date#

If Filter Date is Date Range and dates are set: Quotes & Orders #Start Date# through #End Date#

  


  


Columns: 

  * ID (link to record)
  * Bid Date
  * Delivery / Pickup Date
  * Order Total
  * Salesperson
  * Truss Order ("Yes" or blank)
  * Lumber Order ("Yes" or blank)
  * Metal Order ("Yes" or blank)
  * Customer Name
  * Job Name
  * Delivery City



  


Grouped by: Canceled + recency of order date and order bid date

Sub-grouped by: Order/Quote

  


Sorted by: Order ID

  


Filters: 

  * Order #
  * Filter Dates (drop-list of Recent, Date Range)
  * Recent Days (blank = all)  (displayed if Filter Dates is set to Recent)
  * Start Date (displayed if Filter Dates is set to Date Range)
  * End Date (displayed if Filter Dates is set to Date Range)
  * Customer (multi-select list of customers)
  * Status (blank = all)  (drop list of Quote, Order)
  * Include canceled (checkbox)



  


Buttons: 

  * New Quote / Order
