5.6.3.3. Print Yard Packing List Report

  


Requirements

Overview: This is the "Total Production/Delivery Costs" report, filtered to show Deliveries for Order and the details for generating the "Yard Packing List" printout. 

  


Accessed from: 

  * "Print Yard Packing List" link on the Quote / Order record



  


The following filter(s) are set differently than on the main report:

  * "Quote / Order #" = Set to the Order from which the link was clicked
  * "Show Packing List Columns" = Set to checked
