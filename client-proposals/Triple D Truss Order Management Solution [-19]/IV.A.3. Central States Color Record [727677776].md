4.1.3. Central States Color Record

Overview: This is a custom record to configure colors for the Central States-type metal items. 

  


Accessed via: Configure Central Colors Report

  


Sections and Fields: 

  * "This central states color is inactive." (visible if "Active" checkbox is not checked; on-screen message in gray font) 
  * Central States Color section: 
    * Name (required; plain text field; defaults to blank; validate against duplicates) 
    * Active (checkbox; defaults to checked; validate against deactivation if referenced by another record) 



  


Data Access: All users can view and edit
