5.3.3. Lumber Sales

TODO_AP: Ben Reitz 11/14/2025: Standardize

  


This is a report of lumber orders within the specified date range. It contains one row for each row in the Lumber embedded spreadsheet on the order.

  


This is accessed from Triple D Truss | Sales Items | Lumber Sales. 

  


Title: Lumber Sales #Start Date# through #End Date# 

  


Columns: 

  * Order Date
  * Description
  * Unit Weight
  * Unit Price
  * Span
  * Quantity
  * Price



  


Grouped by: Order ID

  


Sorted by: Order ID

  


Filters: 

  * Start Date
  * End Date
  * Lumber Items (list of lumber category SKU descriptions)
  * Totals Only (checkbox)



  


Other Notes: Checking Totals Only displays one row for each invoice in the time period, with quantity and price totaled for all rows in the spreadsheet on the order record.
