5.1.1.2. Industry Forum Event Type

*Documented 

  


The fields for the Industry Forum are set as follows:

  


  * Active: Checkbox checked
  * Event Type Name: "Industry Forum"
  * Default Event Visibility: "All Members"
  * Requires Food Notes if Scheduled: Checkbox not checked 
  * Always Requires Registration Deadline: Checkbox not checked
  * Displays Region & Rep: Checkbox not checked
