5.1.1.3. Info Meeting Event Type

*Documented 

  


The fields for the Info Meeting are set as follows:

  


  * Active: Checkbox checked
  * Event Type Name: "Info Meeting"
  * Default Event Visibility: "All Members"
  * Requires Food Notes if Scheduled: Checkbox checked
  * Always Requires Registration Deadline: Checkbox not checked 
  * Displays Region & Rep: Checkbox checked
