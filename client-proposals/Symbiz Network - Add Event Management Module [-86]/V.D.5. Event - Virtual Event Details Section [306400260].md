5.4.5. Event - Virtual Event Details Section

  


Requirements

*Documented 

  


"Virtual Event Details" section (visible if Event Format Virtual or Hybrid; visibility for users based on the "Event Visibility" field selection): 

  


  * Meeting Link (optional; field for pasting in a meeting link; for users who can edit the record, when out of Edit mode this becomes a clickable link to open the meeting; for users who cannot edit the record, this displays as a clickable link) 
  * Dial-In Number (optional; phone number field; defaults to blank)
  * Dial-In Meeting ID (optional; plain text field; defaults to blank)
  * Dial-In Passcode (optional; plain text field; defaults to blank) 



  


Other Notes: 

  * Warning on Save if Event Status = Scheduled and there is no Meeting Link or Dial-In Number: "This virtual/hybrid event does not have a meeting link or dial-in number set." (note that this warning should only apply if the Virtual Details section is visible).
