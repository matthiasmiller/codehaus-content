11.0.1. Meet & Greet Registrations Report

Add the following new report: 

  


This is a filtered version of the the Event Registrations Report that displays Pre-Summit Meet & Greet Registrations.

  


This is accessed from:

  * "M & G Registrants" link in the Pre-Summit Meet & Greet section of the Event record



  


The following filters are set to something other than their default:

  * Event Type = "Annual Summit" 
  * Event = the Event from which the report was opened



  


Other Notes:

  * See the spec for the Event Registrations Report for additional details (title format, visible columns & buttons, etc.)
