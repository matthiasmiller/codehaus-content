6.3.1.3. Info Meetings Report

*Documented 

  


Add the following new report: 

  


This is a filtered version of the the Master Events Report that displays only Info Meetings.

  


This is accessed from:

  * Admin | Event Management | Info Meetings (bypasses the filter screen to open the report directly)



  


The following filters are set to something other than their default:

  * Event Type = Info Meeting



  


Other Notes: 

  * See the spec for the Master Events Report for additional details (title format, visible columns & buttons, etc.)
