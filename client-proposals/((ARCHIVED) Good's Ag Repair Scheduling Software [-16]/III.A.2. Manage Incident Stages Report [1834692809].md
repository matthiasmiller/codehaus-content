3.1.2. Manage Incident Stages Report

This is a report of all Incident Stages.

  


This is accessed from: Advanced | Configuration | Manage Incident Stages 

  


Title: Manage Incident Stages

  


Columns: 

  * Order
  * Name (link to record)



  


Grouped by: Workflow

  


Sorted by: Order (numerically)

  


Filters: 

  * N/A



  


Buttons: 

  * New Workflow (opens blank new Workflow record)
  * New Stage (opens blank new Stage record)



  


Menu Visibility: All Users

  


Other Notes: 

  * N/A
