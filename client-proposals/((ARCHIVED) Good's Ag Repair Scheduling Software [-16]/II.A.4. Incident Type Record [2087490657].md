2.1.4. Incident Type Record

Overview: Incident Types will be configured on Incident Type records and viewed on the Configure Incident Types report.

  


Accessed via: 

  * Advanced | Configuration | Configure Incident Types



  


Sections and Fields: 

  * Type Name (plain text field)
  * Default Workflow (drop list of ___
  * Billable by default (checkbox; defaults to unchecked)
  * Require closing reason (checkbox; defaults to unchecked)



  


Data Access: All Users

  


Special Considerations: 

  * Copy Record: 
  * Delete Record: 
  * Merge Record: 



  


Other Notes:

  * Note that this record should have a link to access Modification History (standard).
  * The following Incident Type(s) are included in the Solution as "CodeCrafters-defined" items (with non-editable names): 
    * Work Order (used for all mechanic-work incidents)
