3.1.4. Configure Contact Types Report

This is a report of all Contact Types.

  


This is accessed from Advance | Configuration | Configure Contact Types

  


Title: Configure Contact Types

  


Columns: 

  * Type (link to record)



  


Grouped by: Active/Inactive (active at the top)

  


Sorted by: N/A

  


Filters: 

  * Show inactive contact types (checkbox; defaults to unchecked)



  


Buttons: 

  * New Contact Type (opens blank new Contact Type record)



  


Menu Visibility: All Users

  


Other Notes:

  * N/A
