2.1.5. Contact Type Record

Overview: Contact Types will be configured on a Contact Type record and viewed on a Contact Types report.

  


Accessed via: 

  * Advanced | Configuration | Configure Contact Types



  


Sections and Fields: 

  * Type (plain text field)
  * Active (checkbox; defaults to checked)



  


Data Access: All Users

  


Special Considerations: 

  * Copy Record: 
  * Delete Record: 
  * Merge Record: 



  


Other Notes:

  * Note that this record should have a link to access Modification History (standard).
